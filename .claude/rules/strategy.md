- 핵심 전략: **Minervini SEPA 기반 성장주 모멘텀 + 주도 섹터/주도주 모델**
- 방향성:
  - 신고가 근처의 강한 종목 + 강한 섹터에만 집중
  - 한 번에 많이 벌기보다 **큰 손실을 막는 것**을 우선
- Market Wizards 3부작의 철학은 `docs/10_philosophy/*.md`를 Source of Truth로 삼고,
  이 문서는 그 철학에서 나온 **비즈니스 규칙만 요약**한다.

## 2. 유니버스 규칙

- 기본 유니버스:
  - KOSPI, KOSDAQ 상장 일반 주식
- 제외:
  - ETF, ETN, SPAC, 우선주, 리츠
  - 관리종목, 거래정지, 상폐 예정, 투자주의 환기종목 (가능한 한)
  - 최근 20일 평균 거래대금 < 5억
  - 시가총액 < 300억

## 3. Alpha (Trend Template) 규칙

- Minervini Trend Template 8조건 (개념):

  1. SMA50 > SMA150 > SMA200
  2. 오늘 종가 > SMA50
  3. SMA150 > SMA200
  4. SMA200 상승 기울기
  5. 종가 >= 52주 고점의 75%
  6. 종가 >= 52주 저점의 130%
  7. RS가 상위 그룹 (예: 상위 30%)
  8. 종가 > SMA200

- Alpha 통과 기준:

  - **8개 중 최소 5개 이상** True
  - RS(120일) 백분위 >= 70
  - `close > SMA50`

- 코드 수준 규칙:
  - 조건 개수/임계값을 완화하여 "통과 종목 수 늘리기" 금지
  - 완화가 필요하면 **문서부터 수정** 후 코드 반영

## 4. Beta (VCP / 수축) 규칙

- 목표: Alpha 통과 종목 중 **매수 타이밍이 무르익은** 종목을 찾는 것
- 핵심 개념:
  - 변동성 수축: 최근 파동의 고점/저점 폭이 점점 줄어든다.
  - 거래량 건조: 수축 구간에서 거래량이 줄어든다.
  - 상방 돌파 준비: 52주 고점 근처에서 박스권 축소

- 규칙:
  - 최소 3파 이상의 파동이 존재
  - 각 파동의 고점/저점 폭이 점진적으로 감소
  - 박스 상단이 52주 고점 * 0.85 이상
  - 수축 구간 평균 거래량 < 50일 평균 거래량 * 0.6
  - 당일 캔들의 폭(고가-저가)/종가 < 2%

- Beta는 **타이밍 필터**이지, 새로운 전략이 아니다:
  - Alpha를 통과하지 않은 종목을 Beta에서 새로 픽 업하는 것 금지

## 5. Gamma (실적/매크로) 규칙

- 핵심:
  - EPS/매출 성장, ROE, 영업이익률 개선
  - 매크로(특히 화학: 유가/스프레드/재고)와 정합성 확인

- 기본 실적 필터 (가능할 때):

  - EPS YoY >= +25%
  - EPS QoQ >= 0 또는 플러스에 근접 (흑자 전환 등)
  - 매출 YoY >= +15%
  - ROE >= 15%

- 데이터 없을 때:
  - "추정"으로 만드는 것 금지
  - 대신 거래대금/외인/기관 누적 수급/업황 지표를 사용하되,
    `earnings_proxy`로 따로 표시하고 **실제 EPS와 혼동하지 않기**

## 6. Delta (리스크/포지션) 규칙

- 한 종목당 최대 손실: **계좌 기준 1%~2%** (기본 1%)
- 기본 손절: 진입가 대비 **-7% ~ -8%**
- 목표:
  - RR(Risk/Reward) >= 1.5 (기본: 2.0 목표)
  - 계좌 전체 동시 보유 종목 수: 3~10개 범위

- 필수:
  - 모든 포지션에 **진입가/손절/목표가**가 존재해야 함
  - "일단 사고 나중에 생각" 금지

---

## 7. Omega (최종 선택) 규칙

- 최종 추천 종목 수:
  - 기본 1~3개
- 고려 요소:
  - LeaderScore 상위
  - 섹터 집중도 (한 섹터에 몰리지 않게)
  - 동일 스토리 중복 최소화

- Omega는 "AI가 마음대로 선택하는 블랙박스"가 아니다:
  - 모든 선택에는 **사유(reason)**가 문장 형태로 붙어야 함
  - 예: "반도체 섹터 Top1, TT 8/8, VCP 3단계, EPS YoY +30%"
