# 섹터별 지표 매핑 (Sector Indicator Map)

> WICS 중분류 34개 섹터 × 각 10개 지표 = **340개 모니터링 셀**
> 원칙: 실물·매크로 선행지표 60% 이상, 가격(후행) 지표 최소화.
> 출처는 `docs/DATA_SOURCES_CATALOG.md`의 어댑터 상태와 1:1 매핑됩니다.

---

## 표기 규칙

- **출처 컬럼**:
  - `**EIA**` 같이 굵게 = 기존 어댑터로 즉시 가능
  - `KIS✓` / `ECOS✓` / `FRED✓` / `KOSIS✓` / `DART✓` / `EIA✓` / `EXIM✓` / `Naver✓` = 구현 완료
  - `TRASS☐` / `OECD☐` / `WB☐` / `IMF☐` / `pytrends☐` / `CFTC☐` / `Comtrade☐` = 어댑터 추가 필요
  - `Crawl⚠` = 안정 API가 없어 크롤링/대체 소스 필요
- **선/동/후 컬럼**: 선행 / 동행 / 후행
- **반복 출처**: 모든 섹터에 공통으로 들어가는 항목 (KIS 업종지수, 원/달러 환율, KOSPI200 등)은 각 섹터 마지막 슬롯에 배치합니다.
