## 1. Market Wizards (1989)

### 1-1. Michael Marcus
| 항목 | 내용 |
|------|------|
| **배경** | 원래 심리학 전공. Ed Seykota에게 배움. Commodities Corporation에서 $30,000 → $80,000,000 |
| **핵심 관점** | 펀더멘털 + 기술적 분석 동시 활용. "최고의 거래는 두 가지가 동시에 맞을 때" |
| **매매 원칙** | ① 추세 방향 거래 ② 시장이 불편하면 포지션 축소 ③ 분할 진입 |
| **키움 조건식 아이디어** | 20일선 > 60일선 > 120일선 (정배열) AND 거래량 20일평균 대비 150% 이상 |

### 1-2. Bruce Kovner
| 항목 | 내용 |
|------|------|
| **배경** | 하버드 정치학. 택시 운전하며 $3,000로 대두 선물 시작. Caxton Associates 설립 |
| **핵심 관점** | 거시경제 분석 기반의 글로벌 매크로 트레이딩. 기술적 분석은 진입 타이밍용 |
| **매매 원칙** | ① 리스크 먼저 정의 (손절점 사전 결정) ② 차트 패턴으로 진입 ③ 시장이 틀리면 즉시 퇴장 |
| **키움 조건식 아이디어** | 직접 조건식 적용 어려움 (매크로). 간접: 환율/금리 연동 섹터 + 추세 돌파 조건 결합 |

### 1-3. Richard Dennis (터틀 트레이딩 창시자)
| 항목 | 내용 |
|------|------|
| **배경** | $400 → $200,000,000. "터틀 실험"으로 유명. 추세추종의 아이콘 |
| **핵심 관점** | 순수 추세추종. 시스템 트레이딩. "트레이딩은 가르칠 수 있다" |
| **매매 원칙** | ① 20일 고가 돌파 매수 (System 1) ② 55일 고가 돌파 매수 (System 2) ③ 10일/20일 저가 이탈 매도 ④ ATR 기반 포지션 사이징 |
| **키움 조건식** | `현재가 >= 20일 최고가` AND `거래량 > 20일 평균거래량` |
```
[터틀 System 1]
조건1: 현재가 >= MAX(고가, 20)     ← 20일 신고가 돌파
조건2: 거래량 >= 이동평균(거래량,20) × 1.0
조건3: 현재가 > 이동평균(종가,50)   ← 중기 추세 확인

[터틀 System 2]
조건1: 현재가 >= MAX(고가, 55)     ← 55일 신고가 돌파
조건2: 현재가 > 이동평균(종가,120)  ← 장기 추세 확인
```

### 1-4. Paul Tudor Jones
| 항목 | 내용 |
|------|------|
| **배경** | 코튼 트레이더 출신. Tudor Investment Corp. 1987년 블랙먼데이에서 200% 수익 |
| **핵심 관점** | 역추세 + 추세추종 혼합. 엘리엇 파동, 시장 심리 중시. "먼저 방어, 그다음 공격" |
| **매매 원칙** | ① 200일 이동평균선이 핵심 필터 ② 손실 통제 최우선 ③ 가격이 200MA 아래면 매수 금지 |
| **키움 조건식** | `현재가 > 200일 이동평균` AND `200일 이동평균 상승 중` |
```
조건1: 현재가 > 이동평균(종가,200)
조건2: 이동평균(종가,200) > 이동평균(종가,200)[5일전]  ← 200MA 기울기 양
조건3: 등락률(5일) > -3%                              ← 급락 종목 제외
```

### 1-5. Ed Seykota
| 항목 | 내용 |
|------|------|
| **배경** | MIT 전기공학. 최초의 컴퓨터 기반 트레이딩 시스템 개발자 중 한 명. 12년간 250,000% 수익 |
| **핵심 관점** | 순수 시스템 추세추종. "추세는 친구다 — 끝날 때까지" (The trend is your friend until the end) |
| **매매 원칙** | ① 손절을 걸어라 ② 승자를 타라 ③ 작게 배팅하라 ④ 규칙을 따르라 ⑤ 언제 규칙을 깰지 알아라 |
| **키움 조건식** | 장기 이동평균 + EMA 크로스 |
```
조건1: EMA(12) > EMA(26)                    ← 단기 모멘텀 양
조건2: 현재가 > 이동평균(종가,200)            ← 장기 추세 확인
조건3: 이동평균(종가,50) > 이동평균(종가,200) ← 골든크로스 유지
```

### 1-6. Larry Hite
| 항목 | 내용 |
|------|------|
| **배경** | 배우/각본가에서 전향. Mint Investment Management. 시스템 트레이딩의 선구자 |
| **핵심 관점** | 리스크 관리가 전부. "1 거래에 자산의 1% 이상 리스크 금지". 수익은 리스크 통제의 부산물 |
| **매매 원칙** | ① 한 거래 리스크 ≤ 자산의 1% ② 시스템 테스트 후 실행 ③ 분산 투자 ④ 추세추종 |
| **키움 조건식** | Dennis와 유사한 추세 돌파 + ATR 기반 필터 |
```
조건1: 현재가 >= MAX(고가, 20)
조건2: 현재가 > 이동평균(종가,100)
조건3: 거래대금 >= 5억    ← 유동성 확보 (소형주 제외)
```

### 1-7. William O'Neil (CAN SLIM)
| 항목 | 내용 |
|------|------|
| **배경** | Investor's Business Daily 창립자. CAN SLIM 방법론 개발. 최고의 성장주 투자자 중 한 명 |
| **핵심 관점** | 펀더멘털(실적 가속) + 기술적(차트 패턴) 결합. "시장의 최고 종목은 공통된 특성을 가진다" |
| **CAN SLIM** | **C**urrent EPS ↑25%+ / **A**nnual EPS ↑25%+ / **N**ew(신제품/신고가) / **S**upply(유통주식) / **L**eader(업종 선도) / **I**nstitutional(기관 매수) / **M**arket(시장 방향) |
| **키움 조건식** | SEPA 시스템의 핵심 기반 |
```
[CAN SLIM 조건식]
조건1: 현재가 >= 52주최고가 × 0.75        ← 고가 근접 (N)
조건2: 현재가 > 이동평균(종가,50)          ← 50MA 위
조건3: 이동평균(종가,50) > 이동평균(종가,200) ← 정배열
조건4: 거래량 > 이동평균(거래량,50) × 1.5  ← 수급 확인 (S)
조건5: 등락률(1일) > 0                    ← 양봉
조건6: 시가총액 >= 1000억                  ← 기관 관심 가능 (I)

[펀더멘털 — API/수동 확인]
- EPS QoQ >= 25%  (C)
- EPS YoY >= 25%  (A)
- 매출 성장률 >= 20%
- ROE >= 15%
```

### 1-8. David Ryan (O'Neil 제자)
| 항목 | 내용 |
|------|------|
| **배경** | William O'Neil의 제자. US Investing Championship 3연패. 성장주 스페셜리스트 |
| **핵심 관점** | O'Neil CAN SLIM의 실전 적용. 컵앤핸들/VCP 패턴 전문. "차트 패턴은 수급의 발자국" |
| **매매 원칙** | ① EPS 가속 필수 ② 업종 내 1~2등 종목만 ③ 적절한 베이스 패턴에서 매수 ④ 7~8% 손절 |
| **키움 조건식** | O'Neil 조건 + VCP 수축 패턴 |
```
조건1: 현재가 >= 52주최고가 × 0.85      ← 고가 근접
조건2: 현재가 > 이동평균(종가,50)
조건3: (고가-저가)/종가 < 0.03           ← 변동성 수축 (VCP 힌트)
조건4: 거래량 < 이동평균(거래량,50) × 0.7 ← 거래량 건조 (수축 확인)
조건5: 이동평균(종가,50) > 이동평균(종가,150) > 이동평균(종가,200)
```

### 1-9. Marty Schwartz

```
조건1: 현재가 > EMA(10)

조건4: MACD > Signal
```

## 2. New Market Wizards (1992)

### 2-1. Stan Druckenmiller
| 항목 | 내용 |
|------|------|
| **배경** | 소로스의 수석 포트폴리오 매니저. 파운드 공매도($10B)의 실질적 설계자. 30년 무손실 기록 |
| **핵심 관점** | "확신이 있을 때 크게 배팅". 유동성/중앙은행 정책이 시장을 움직인다 |
| **매매 원칙** | ① 확신 높으면 레버리지 ② 틀리면 빠르게 인정 ③ 유동성 방향을 따라라 |
| **키움 조건식** | 유동성 장세 수혜주 |
```
조건1: 현재가 > 이동평균(종가,50) > 이동평균(종가,200)
조건2: 거래대금 >= 50억                  ← 기관/외인 유동성 유입 가능
조건3: 외국인순매수(5일) > 0             ← 외인 자금 유입
조건4: 등락률(20일) > 이동평균등락률(코스피,20) ← 시장 대비 강세
```

### 2-2. William Eckhardt (터틀 공동 창시자)
| 항목 | 내용 |

| **배경** | 터키 출신. 가치투자 전문. 연평균 40%+ |
| **핵심 관점** | 깊은 가치 + 촉매. "50센트짜리 1달러를 사라 — 단, 1달러로 가게 할 촉매가 있어야" |
| **매매 원칙** | ① PER/PBR 극단적 저평가 ② 내부자 매수 ③ 부채비율 낮은 ④ 실적 턴어라운드 |
| **키움 조건식** | 깊은 가치 + 턴어라운드 |
```
조건1: PBR < 0.7
조건2: PER < 8
조건3: 부채비율 < 100%
조건4: 현재가 < 52주최고가 × 0.50        ← 고점 대비 50% 이상 하락
조건5: 등락률(5일) > 0                   ← 바닥 반등 시작
조건6: 거래량 > 이동평균(거래량,20) × 1.5 ← 수급 유입
```

### 3-6. Mark Minervini 이외 추가 트레이더

## 4. 키움 조건식 종합 매핑표

주식 시장에 직접 적용 가능한 트레이더들의 전략을 유형별로 그룹핑:

### A. 추세추종 (Trend Following)
> Dennis, Seykota, Hite, Bielfeldt, Seidler, Weiss, Jones

```
[추세추종 마스터 조건식]
조건1: 이동평균(종가,50) > 이동평균(종가,150) > 이동평균(종가,200)  ← 3선 정배열
조건2: 현재가 > 이동평균(종가,50)
조건3: 이동평균(종가,200) > 이동평균(종가,200)[22일전]              ← 200MA 상승
조건4: 현재가 >= MAX(고가, 20)                                    ← 20일 신고가
조건5: 거래량 > 이동평균(거래량,20)                                ← 돌파 수급
```

### B. 성장주 모멘텀 (Growth Momentum)
> O'Neil, Ryan, Driehaus, Minervini, Walton, Cohen

```
[성장 모멘텀 마스터 조건식]
조건1: 현재가 > 이동평균(종가,150) AND 현재가 > 이동평균(종가,200) ← TT
조건2: 이동평균(종가,50) > 이동평균(종가,150)
조건3: 현재가 >= 52주최저가 × 1.25                                ← 바닥 탈출
조건4: 현재가 >= 52주최고가 × 0.75                                ← 고가 근접
조건5: 거래량 > 이동평균(거래량,50) × 1.5                         ← 매집 수급
조건6: 시가총액 >= 500억                                          ← 최소 유동성

[펀더멘털 필수 체크 — 키움 API / 수동]
- EPS QoQ >= 25%
- EPS YoY >= 25%
- 매출 성장 >= 15%
- ROE >= 15%
```

### C. 단기 스윙 (Short-term Swing)
> Schwartz, Raschke, Trout

```
[스윙 트레이딩 마스터 조건식]
조건1: 현재가 > EMA(10)                                          ← Schwartz 핵심
조건2: RSI(14) > 40 AND RSI(14) < 65                             ← 과열 전 진입
조건3: ADX(14) > 20                                              ← 추세 존재
조건4: (고가-저가)/종가 < 0.025                                   ← 좁은 레인지
조건5: 현재가 > 이동평균(종가,50)                                  ← 기본 추세
```

### D. 역발상/가치 (Contrarian/Value)
> Rogers, Steinhardt, Okumus

```
[역발상 가치 조건식]
조건1: PBR < 0.8
조건2: PER < 업종평균PER × 0.6
조건3: 부채비율 < 120%
조건4: 현재가 < 52주최고가 × 0.55                                 ← 깊은 하락
조건5: RSI(14) < 35                                              ← 과매도 진입
조건6: 거래량 > 이동평균(거래량,20) × 1.3                         ← 반등 수급
```

### E. 변동성 수축 돌파 (Volatility Contraction Breakout)
> Minervini VCP, Ryan, Weinstein

```
[VCP/스퀴즈 조건식]
조건1: 이동평균(종가,50) > 이동평균(종가,150) > 이동평균(종가,200) ← 정배열
조건2: 현재가 > 이동평균(종가,50)
조건3: 볼린저밴드폭(20) < 15%                                    ← 변동성 수축
조건4: 거래량 < 이동평균(거래량,50) × 0.6                         ← 거래량 건조
조건5: 현재가 >= 52주최고가 × 0.85                                ← 고가 근접
조건6: (고가-저가)/종가 < 0.02                                    ← 일봉 변동성 축소
```

## 5. SEPA 시스템과의 연결

현재 프로젝트의 에이전트 구조와 시장의 마법사 트레이더들의 매핑:

| SEPA Agent | 역할 | 반영된 트레이더 철학 |
|------------|------|---------------------|
| **Alpha** | Trend Template + RS | **Minervini** (TT 8조건), **O'Neil** (CAN SLIM의 L), **Jones** (200MA 필터) |
| **Beta** | VCP 수축/돌파 | **Minervini** (VCP), **Ryan** (컵앤핸들), **Weinstein** (저변동성 셋업), **Raschke** (NR7) |
| **Gamma** | EPS/매출 가속 + 매크로 | **O'Neil** (C, A of CANSLIM), **Driehaus** (실적 서프라이즈), **Druckenmiller** (매크로 유동성) |
| **Delta** | 진입/손절/목표/수량 | **Hite** (1% 룰), **Lipschutz** (R:R 3:1), **Basso** (변동성 사이징), **Dennis** (ATR 사이징) |
| **Omega** | 최종 추천 | 전체 종합. **Seykota** (규칙 따르기), **Eckhardt** (불편한 결정 수용) |

### 미반영 / 향후 고려 가능한 전략

| 전략 | 관련 트레이더 | 적용 방법 |
|------|-------------|-----------|
| 역발상/가치 스크리닝 | Rogers, Steinhardt, Okumus | 별도 "Value Agent" 또는 Gamma 확장 |
| 공매도 후보 | Galante | 별도 "Short Agent" (대차잔고/공매도 데이터 필요) |
| 섹터 로테이션 | Blake, Cohen | Leaders Agent에 섹터 상대강도 반영 (현재 일부 구현) |
| 단기 스윙/평균회귀 | Schwartz, Trout, Raschke | 별도 "Swing Agent" (분/시간봉 데이터 필요) |
| 시장 타이밍 | Jones (200MA), Schwartz (10EMA) | Market Index Agent에 시장 체제 판단 추가 |

## 부록: 키움 HTS 조건검색식 입력 참고

키움 영웅문 HTS에서 조건검색식 작성 시 주요 함수:

| 함수 | 설명 | 예시 |
|------|------|------|
| `이동평균(종가,N)` | N일 단순이동평균 | 이동평균(종가,50) |
| `EMA(N)` | N일 지수이동평균 | EMA(10) |
| `MAX(고가,N)` | N일 중 최고 고가 | MAX(고가,20) |
| `MIN(저가,N)` | N일 중 최저 저가 | MIN(저가,20) |
| `RSI(N)` | N일 RSI | RSI(14) |
| `MACD(단기,장기,시그널)` | MACD | MACD(12,26,9) |
| `ADX(N)` | N일 ADX | ADX(14) |
| `볼린저밴드(N,K)` | 볼린저밴드 | 볼린저밴드(20,2) |
| `ATR(N)` | N일 ATR | ATR(14) |
| `등락률(N일)` | N일간 등락률 | 등락률(20일) |
| `거래대금` | 당일 거래대금 | 거래대금 >= 10억 |
| `시가총액` | 시가총액 | 시가총액 >= 1000억 |
| `PER` / `PBR` | 밸류에이션 | PER < 15 |
| `52주최고가` / `52주최저가` | 연중 고저 | 현재가 >= 52주최고가 × 0.9 |

> **참고**: 키움 Open API를 통해 조건검색식을 코드로 호출할 때는
> `GetConditionLoad()` → `GetConditionNameList()` → `SendCondition()` 순서로 사용합니다.
> 현재 프로젝트의 `sepa/data/kiwoom.py`는 OHLCV 데이터 조회용이며,
> 조건검색 호출은 별도 구현이 필요합니다.
