## 한 줄 요약

> **"고전적 차트 패턴에 기반한 매매에서 리스크를 최우선으로 관리하면 수십 년간 일관된 수익이 가능하다."**

## 핵심 투자 철학 (5 Principles)

1. **고전적 차트 패턴의 힘 (Classical Chart Patterns Work)**
   - 헤드앤숄더, 삼각형, 직사각형, 웨지, 컵앤핸들 등 고전 패턴은 100년+ 검증된 도구.
   - Richard Schabacker와 Edwards & Magee의 원전에 기반한 정통 차트 분석.

2. **리스크가 수익보다 먼저다 (Risk First, Always)**
   - 모든 거래에서 잠재 손실을 먼저 계산한다. 수익은 그다음.
   - "나는 리스크를 트레이드한다. 수익은 리스크 관리의 부산물이다."

3. **패턴 실패도 정보다 (Failed Patterns Are Informative)**
   - 패턴이 실패하면 종종 반대 방향의 강력한 움직임이 뒤따른다.
   - 패턴 실패를 역이용하는 것도 전략의 일부.

4. **장기적 규율 (Decades of Discipline)**
   - 40년+ 경력의 비결은 기법이 아니라 규율의 일관성.
   - "좋은 트레이더와 위대한 트레이더의 차이는 수십 년간의 규율이다."

5. **투명성과 기록 (Transparency and Record-Keeping)**
   - 모든 매매를 상세히 기록하고 공개. 블로그와 SNS에서 실시간 매매 공유.
   - "자신의 매매를 공개하지 못한다면 그 매매에 자신이 없다는 뜻이다."

## 대표 전략 유형

### Classical Chart Pattern Trading

Brandt의 전략은 고전적 차트 패턴 인식과 리스크 우선 접근의 결합이다:

| 구성 요소 | 역할 |
|-----------|------|
| **패턴 인식** | H&S, 삼각형, 직사각형, 컵앤핸들, 플래그 등 식별 |
| **돌파 확인** | 패턴 경계 돌파 시 거래량 확인으로 진위 판단 |
| **리스크 계산** | 진입 전 손절선과 목표가를 먼저 설정 |
| **패턴 실패 활용** | 실패한 패턴의 반대 방향 진입 |
| **다중 시장** | 주식, 원자재, 통화, 암호화폐까지 차트 패턴 적용 |

### 주요 차트 패턴

```
헤드앤숄더(H&S):          컵앤핸들:
    H                          ┌──┐ Handle
   / \                    ┌───┘  └──→ 돌파
  /   \   S              /           
 / L   \ / \            /             
/   \   V   \          │              
     Neckline──→ 돌파   └──────┘ Cup   
```

## 매매 기법

### 진입 (Entry)

- **패턴 완성 후 돌파**: 차트 패턴이 완성되고 경계선을 돌파할 때 진입
- **거래량 확인**: 돌파 시 평균 이상의 거래량이 동반되어야 유효
- **풀백 진입**: 돌파 후 패턴 경계선으로의 풀백(throwback)에서 추가 진입
- **패턴 실패 역이용**: 상방 돌파 실패 시 숏, 하방 이탈 실패 시 롱
- **주간 차트 우선**: 일봉보다 주봉 차트의 패턴에 더 높은 신뢰도 부여

### 청산 (Exit)

- **패턴 측정 목표**: 패턴의 높이를 돌파 지점에 더한 측정 목표(measured move) 활용
- **트레일링 스톱**: 수익이 나면 이동평균 또는 ATR 기반 트레일링 스톱
- **반대 패턴**: 반대 방향의 차트 패턴이 형성되면 청산
- **부분 청산**: 측정 목표의 50%에서 절반, 100%에서 나머지 청산

### 손절 (Stop-Loss)

- **패턴 무효화 지점**: 패턴의 구조가 무효화되는 가격 수준에 손절선 설정
- **예시**: H&S 숏 → 오른쪽 어깨 위에 손절, 삼각형 돌파 롱 → 삼각형 하단 아래 손절
- **즉각 실행**: 손절선 도달 시 예외 없이 실행. "희망은 전략이 아니다"
- **사전 결정**: 진입 시점에 이미 손절선 확정. 사후 수정 절대 금지

### 포지션 사이징 (Position Sizing)

- **리스크 기반**: 진입가와 손절가의 차이 × 수량 = 총 자본의 0.5-1%
- **보수적 사이징**: 많은 트레이더보다 작은 사이즈. "살아남는 것이 이기는 것보다 중요"
- **패턴 품질**: 교과서적 패턴에 표준 사이즈, 불완전한 패턴은 반으로 축소
- **최대 노출**: 전체 포트폴리오 리스크 5% 이내

### 리스크 관리 (Risk Management)

- **40년 생존의 비결**: "큰 손실만 피하면 복리가 모든 것을 해결한다"
- **연속 손절 규칙**: 연속 손절 시 사이즈 축소 및 시장 재평가
- **시장 환경**: 추세가 불명확한 횡보장에서는 사이즈 축소 또는 관망
- **매매 일지**: 모든 거래를 상세히 기록, 주기적으로 복기하여 패턴 개선

## 심리와 의사결정

### 핵심 멘탈 모델

- **장인 정신**: 40년간 같은 방법론을 깊이 파고든 장인. "깊이가 넓이를 이긴다"
- **겸손**: 시장 예측이 아닌 패턴 대응에 집중. "나는 예측하지 않는다. 반응할 뿐이다"
- **투명성**: 매매를 공개함으로써 자기 규율을 강화
- **회복력**: 수백 번의 손절과 드로다운을 겪으면서도 시스템을 유지하는 정신적 강인함

### Brandt의 심리 규칙

1. 차트가 말하는 것을 들어라. 차트에게 말하려 하지 마라
2. 패턴 실패에 분노하지 마라. 실패한 패턴은 반대 방향의 기회다
3. 수십 년간의 규율이 모든 기법보다 중요하다
4. 자신의 매매를 공개하라. 투명성이 규율을 강화한다
5. 리스크를 먼저 생각하라. 수익은 리스크 관리의 자연스러운 결과다

## 대표 사례

### 성공 사례: 40년+ 일관된 트랙레코드

- **배경**: Brandt는 1976년부터 40년 이상 차트 패턴 기반 트레이딩으로 연평균 58% 수익
- **전략**: Edwards & Magee 스타일의 고전적 차트 패턴을 일관되게 적용
- **핵심**: 기법의 변화가 아닌 규율의 일관성. 같은 패턴을 40년간 반복
- **교훈**: "위대한 트레이딩은 비밀 기법이 아니라 수십 년간의 일관된 실행에서 나온다"

### 성공 사례: 비트코인 차트 분석

- **배경**: Brandt는 2010년대 후반 비트코인 시장에서도 고전적 차트 패턴을 적용
- **분석**: 비트코인의 주요 상승/하락 구간에서 H&S, 삼각형 등을 정확히 식별
- **결과**: 새로운 자산군에서도 동일한 차트 패턴이 유효함을 증명
- **교훈**: 고전적 차트 패턴은 자산군에 관계없이 보편적으로 작동한다

### 실패 사례: 패턴의 한계 인정

- **배경**: 차트 패턴의 승률은 50-60% 수준. 즉, 40-50%는 실패
- **대응**: 실패를 받아들이고 리스크 관리로 보완. 승률이 아닌 R:R로 수익 확보
- **기록**: *Diary of a Professional Commodity Trader*에서 실패 사례를 투명하게 공개
- **교훈**: "100% 정확한 패턴은 없다. 중요한 것은 맞을 때 얼마를 벌고 틀릴 때 얼마를 잃느냐이다"

## 내 시스템과의 연결

### SEPA 공통점

Brandt의 차트 패턴 분석과 리스크 우선 접근은 OmX의 Alpha+Beta+Delta 에이전트와 연결된다:

| Brandt 원칙 | OmX 구현 |
|-------------|----------|
| 차트 패턴 인식 | **Alpha 에이전트** — 컵앤핸들, 이중 바닥 등 패턴 자동 감지 |
| 돌파 확인 | **Beta 에이전트** — VCP Pivot 돌파 + 거래량 확인 |
| 리스크 우선 | **Delta 에이전트** — 진입 전 R:R 자동 계산, 최소 2:1 필터 |
| 패턴 실패 감지 | **Beta 에이전트** — 돌파 후 재진입 실패 감지 |
| 매매 기록 | **전체 시스템** — 자동 매매 일지 및 성과 추적 |

### SEPA 차이점

| 항목 | Brandt 방식 | OmX 시스템 |
|------|------------|-----------|
| 패턴 범위 | 모든 고전 패턴 (H&S, 삼각형, 웨지 등) | VCP, 컵앤핸들 중심 |
| 방향 | 롱/숏 양방향 | 주로 롱 (Stage 2) |
| 시장 | 원자재, 통화, 주식, 암호화폐 | 한국/미국 주식 |
| 추세 필터 | 패턴 자체가 방향 지시 | Trend Template 8조건 선행 |
| 타임프레임 | 주봉 위주 (수주~수개월) | 일봉+주봉 (수주~수개월) |

### 전략군 (Strategy Family)

- **Family A (Trend Following)**: 추세 방향의 차트 패턴 돌파
- **Family E (Volatility Pattern)**: 변동성 수축 후 돌파 패턴 (VCP, 삼각형 등)

### 에이전트 매핑

| 에이전트 | Brandt 요소 | 구현 내용 |
|----------|------------|----------|
| **Alpha** | 패턴 인식 | 컵앤핸들, 이중 바닥, 플래그 등 자동 감지 |
| **Beta** | 돌파 확인 | 패턴 경계 돌파 + 거래량 급증 확인 |
| **Beta** | 패턴 실패 감지 | 돌파 후 재진입 실패 시 역방향 신호 |
| **Delta** | 리스크 우선 | 패턴 무효화 지점 기반 자동 손절선, R:R 계산 |
| **Omega** | 패턴 품질 평가 | 패턴 대칭성, 거래량 프로파일, 추세 컨텍스트 종합 |

### 구현 아이디어

1. **차트 패턴 자동 인식 엔진**: 고전적 차트 패턴(H&S, 삼각형, 컵앤핸들 등)을 알고리즘으로 자동 감지
2. **패턴 측정 목표 자동 계산**: 패턴 높이를 기반으로 목표가(measured move) 자동 산출
3. **패턴 실패 역이용 알림**: 패턴 돌파 후 재진입 실패 감지 시 반대 방향 기회 알림
4. **패턴 카탈로그 통계**: 각 패턴 유형의 역사적 승률, 평균 이동폭, 실패율 데이터베이스
5. **매매 일지 자동 생성**: 진입/청산/손절 시 자동으로 매매 일지 항목 생성, 스크린샷 포함
