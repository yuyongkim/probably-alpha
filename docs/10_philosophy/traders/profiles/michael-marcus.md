# Michael Marcus

## 기본 정보

| 항목 | 내용 |
|------|------|
| 본명 | Michael Marcus |
| 출생 | 1940년대, 미국 |
| 활동 기간 | 1970년대 ~ 현재 |
| 주요 저서 | 없음 (극도로 사적인 인물) |
| 인터뷰 수록 | *Market Wizards* (Jack Schwager, 1989) — 첫 번째 인터뷰이 |
| 주요 실적 | Commodities Corporation에서 $30,000을 $80,000,000으로 증식 (약 2,500배), 10년간 연평균 수백% 수익 |

## 한 줄 요약

> **"추세를 따르되 거시적 큰 그림을 놓치지 마라. 그리고 확신이 있을 때는 크게 베팅하라."**

## 핵심 투자 철학 (5 Principles)

1. **추세 추종 + 매크로 혼합 (Trend Following + Macro Blend)**
   - 순수 기술적 추세 추종에 거시경제 분석을 결합한 하이브리드 접근법.
   - 기술적 신호가 매크로 방향과 일치할 때 확신도와 포지션 크기를 극대화한다.

2. **큰 그림 우선 (Big Picture First)**
   - 개별 시장에 진입하기 전 반드시 글로벌 매크로 환경을 점검한다.
   - 금리, 달러, 원자재의 큰 추세가 개별 포지션의 방향과 일치해야 한다.

3. **확신에 비례한 베팅 (Bet Size = Conviction)**
   - 모든 트레이드가 동일하지 않다. 최고의 기회에 가장 큰 베팅.
   - Ed Seykota로부터 배운 교훈: "좋은 트레이드를 식별하는 것보다 큰 트레이드를 크게 하는 것이 더 중요하다."

4. **멘토와 학습 (Learn from the Best)**
   - Ed Seykota가 Marcus의 멘토. "Ed는 나에게 트레이딩을 가르쳤다."
   - Marcus는 이후 Bruce Kovner를 가르치는 등, 지식의 전수를 중시.

5. **자본 보호의 절대성 (Capital Preservation)**
   - 큰 손실은 절대 허용하지 않는다. 살아남아야 기회를 잡을 수 있다.
   - "첫 번째 규칙: 돈을 잃지 마라. 두 번째 규칙: 첫 번째 규칙을 잊지 마라."

## 대표 전략 유형

### Trend + Macro Hybrid

Marcus의 전략은 기술적 추세 추종과 거시경제 판단을 결합한다:

| 구성 요소 | 역할 |
|-----------|------|
| **추세 식별** | 이동평균, 브레이크아웃으로 추세 방향 확인 |
| **매크로 필터** | 금리, 통화, 정책 방향이 추세와 일치하는지 확인 |
| **진입 타이밍** | 추세 내 풀백(pullback)에서 진입, 돌파 시 추가 |
| **집중 베팅** | 추세 + 매크로 일치 시 포지션 대폭 확대 |
| **다중 시장** | 주식, 선물, 통화 등 다양한 시장에서 기회 추구 |

## 매매 기법

### 진입 (Entry)

- **추세 확인**: 이동평균 정배열, 고점/저점 상승 패턴 확인
- **매크로 일치**: 거시 환경이 해당 시장의 방향을 지지하는지 확인
- **풀백 진입**: 상승 추세 중 단기 조정 시 진입. 추격 매수 지양
- **돌파 매수**: 주요 저항선 돌파 시 거래량 확인 후 진입
- **다중 확인**: 최소 2개 이상의 독립적 이유가 있을 때만 진입

### 청산 (Exit)

- **추세 이탈**: 이동평균 하향 돌파, 주요 지지선 붕괴 시 청산
- **매크로 변화**: 거시 환경이 반대 방향으로 전환되면 포지션 축소
- **목표 달성**: 사전 설정한 기술적 목표치 도달 시 부분 청산
- **트레일링 스톱**: 수익이 나는 포지션은 트레일링 스톱으로 보호

### 손절 (Stop-Loss)

- **기술적 손절**: 진입 근거가 된 지지선/패턴이 무효화되면 즉시 손절
- **금액 기반**: 한 거래에서 자본의 5% 이상 잃지 않도록 관리
- **시간 손절**: 예상한 방향으로 움직이지 않으면 정리
- **빠른 실행**: 손절 결정에 주저하지 않는다. "잘못된 포지션에서 하루도 더 머무르지 마라"

### 포지션 사이징 (Position Sizing)

- **확신도 비례**: 최고 확신 트레이드에 포트폴리오의 큰 비중 배분
- **피라미딩**: 수익이 나고 추세가 확인되면 점진적으로 추가 매수
- **초기 탐색**: 불확실한 트레이드는 소규모로 시작
- **집중 vs 분산**: 동시에 5~10개 포지션 유지, 상위 2~3개에 자본 집중

### 리스크 관리 (Risk Management)

- **생존 우선**: "살아남는 것이 가장 중요하다. 살아 있으면 기회는 반드시 온다"
- **연속 손실 대응**: 연속 손실 시 사이즈를 대폭 줄이고 자신감 회복 후 재확대
- **상관관계 인식**: 같은 방향의 포지션이 몰리지 않도록 관리
- **감정 모니터링**: 흥분하거나 두려울 때는 사이즈를 줄인다

## 심리와 의사결정

### 핵심 멘탈 모델

- **겸손**: "시장은 항상 나보다 크다." 오만함이 가장 큰 적
- **인내**: 최적의 기회가 올 때까지 기다린다. 강제로 트레이드를 만들지 않는다
- **독립성**: 다른 트레이더의 의견에 휘둘리지 않는다. 자신의 분석만 신뢰
- **유연성**: 시장이 예상과 다르게 움직이면 즉시 관점을 수정

### Marcus의 심리 규칙

1. 최고의 트레이드는 쉽게 느껴진다. 힘들게 유지해야 하는 포지션은 잘못된 것이다
2. 다른 사람의 팁으로 매매하지 마라. 자신의 분석이 없으면 손절도 할 수 없다
3. 연속 손실 후에는 사이즈를 줄여라. 자신감을 회복한 후 다시 키워라
4. 시장이 비합리적이라고 화내지 마라. 시장이 항상 옳다
5. 트레이딩은 마라톤이다. 한 번의 거래가 아닌 수천 번의 거래로 승부한다

## 대표 사례

### 성공 사례: $30,000에서 $80,000,000으로

- **배경**: Marcus는 Commodities Corporation에서 약 $30,000으로 시작하여 10년간 약 $80,000,000까지 자산을 증식
- **전략**: 원자재 선물 시장에서 추세 추종 + 매크로 분석 혼합. 특히 1970년대 인플레이션 환경에서 원자재 상승 추세에 대규모 베팅
- **핵심 요인**: 매크로 환경(인플레이션, 약달러)과 기술적 추세가 일치하는 대형 기회에 집중 베팅
- **교훈**: 매크로와 기술적 분석의 일치가 최고의 신호이며, 이때 크게 베팅해야 한다

### 성공 사례: 독일 마르크 트레이드

- **배경**: 베를린 장벽 붕괴 후 독일 통일 과정에서 마르크의 방향성을 예측
- **분석**: 매크로 분석(독일 통일 비용, 분데스방크 정책)과 기술적 추세의 일치를 확인
- **결과**: 대규모 수익 실현
- **교훈**: 역사적 이벤트가 매크로 방향을 제공하고, 기술적 분석이 타이밍을 제공

### 실패 사례: 초기 경력 — 과도한 레버리지

- **배경**: Marcus는 초기에 과도한 레버리지로 여러 차례 계좌를 거의 날림
- **원인**: 확신은 있었지만 리스크 관리가 부족. 포지션 사이즈가 계좌 대비 과도
- **전환점**: Ed Seykota를 만나 체계적 리스크 관리와 추세 추종의 규율을 배움
- **교훈**: "아무리 좋은 트레이드라도 사이즈 관리가 없으면 파산한다"

## 내 시스템과의 연결

### SEPA 공통점

Marcus의 추세+매크로 혼합 접근법은 OmX 시스템의 Alpha+Gamma 에이전트 결합과 정확히 일치한다:

| Marcus 원칙 | OmX 구현 |
|-------------|----------|
| 추세 확인 | **Alpha 에이전트** — Trend Template 8조건으로 추세 필터링 |
| 매크로 일치 확인 | **Gamma 에이전트** — 금리, 유동성, 경기 사이클 분석 |
| 추세 + 매크로 결합 | **Omega 에이전트** — Alpha + Gamma 신호 통합 스코어링 |
| 집중 베팅 | **Delta 에이전트** — 확신도 연동 포지션 사이징 |
| 손절 규율 | **Delta 에이전트** — 자동 손절 트리거 |

### SEPA 차이점

| 항목 | Marcus 방식 | OmX 시스템 |
|------|------------|-----------|
| 시장 | 원자재 선물, 통화, 주식 전방위 | 한국/미국 주식 집중 |
| 분석 | 주관적 판단 + 차트 리딩 | 정량적 알고리즘 기반 |
| 매크로 | 직관적 거시 판단 | FRED/ECOS 데이터 기반 정량 분석 |
| 레버리지 | 선물 레버리지 적극 활용 | 무레버리지 현물 기반 |
| 멘토링 | 인적 네트워크 기반 학습 | 시스템 로그 기반 자동 학습 |

### 전략군 (Strategy Family)

- **Family A (Trend Following)**: Marcus의 기술적 추세 추종 요소
- **Family F (Macro/Fundamental)**: Marcus의 거시경제 분석 요소

### 에이전트 매핑

| 에이전트 | Marcus 요소 | 구현 내용 |
|----------|------------|----------|
| **Alpha** | 추세 식별 | 이동평균 정배열, 52주 신고가 근접도 자동 계산 |
| **Gamma** | 매크로 필터 | 금리, 유동성, 경기 선행 지표 자동 수집/분석 |
| **Delta** | 확신도 베팅 | Alpha+Gamma 복합 점수 기반 포지션 사이징 |
| **Delta** | 손절 관리 | 기술적 지지선 이탈 시 자동 손절 |
| **Omega** | 복합 판단 | 추세 + 매크로 일치도를 종합 점수로 산출 |

### 구현 아이디어

1. **추세-매크로 일치도 스코어**: Alpha(추세)와 Gamma(매크로)의 방향 일치도를 0~100으로 정량화
2. **다중 확인 필터**: 최소 2개 이상의 독립 신호(추세+매크로+거래량)가 일치해야 진입 허용
3. **확신도 기반 동적 사이징**: 추세-매크로 일치 점수가 높을수록 포지션 사이즈 자동 확대
4. **연속 손실 자동 감속**: 연속 3회 손절 시 Delta가 자동으로 사이즈 50% 축소
5. **매크로 환경 변화 알림**: Gamma가 금리/유동성 환경 변화 감지 시 즉시 포트폴리오 재평가 트리거
