## 한 줄 요약

> **"전장에서든 시장에서든 규율 있는 실행이 생존과 승리를 결정한다 — 계획을 세우고, 계획대로 실행하고, 실행을 복기하라."**

## 핵심 투자 철학 (5 Principles)

1. **군사적 규율의 트레이딩 적용 (Military Discipline in Trading)**
   - 트레이딩은 전투와 같다. 사전 계획, 규율 있는 실행, 사후 복기의 사이클이 핵심이다.
   - 감정이 아닌 절차(SOP: Standard Operating Procedure)에 따라 행동한다.
   - "계획 없이 시장에 들어가는 것은 무장 없이 전장에 나서는 것과 같다."

2. **패턴 인식 (Pattern Recognition)**
   - 시장은 인간의 심리를 반영하며, 심리는 반복적 패턴을 만든다.
   - 차트 패턴, 가격 행동, 거래량 패턴을 통해 반복되는 기회를 포착한다.
   - 패턴의 맥락(market context)이 패턴 자체만큼 중요하다.

3. **리스크 관리가 수익보다 선행한다 (Risk Management Before Returns)**
   - 생존이 최우선. 자본을 보전해야 다음 기회에 참여할 수 있다.
   - 매 거래에서 잃을 수 있는 최대 금액을 사전에 정의하고 반드시 준수한다.
   - "좋은 트레이더는 돈을 잘 버는 사람이 아니라 돈을 잘 지키는 사람이다."

4. **체계적 실행 (Systematic Execution)**
   - 진입, 청산, 손절, 포지션 사이징의 모든 과정을 사전에 규정한다.
   - 실시간으로 판단하는 것이 아니라, 사전에 판단한 것을 실시간으로 실행한다.
   - 실행의 일관성이 전략의 우수성보다 더 중요할 수 있다.

5. **지속적 개선 (Continuous After-Action Review)**
   - 군대의 AAR(After-Action Review) 프로세스를 트레이딩에 적용한다.
   - 모든 거래를 복기하여 무엇이 잘 되었고, 무엇이 잘못되었고, 무엇을 개선할지 분석한다.
   - "같은 실수를 반복하는 것은 학습하지 않았다는 증거다."

## 대표 전략 유형

### 패턴 기반 규율 트레이딩 (Pattern-Based Disciplined Trading)

Netto의 전략은 패턴 인식을 군사적 규율로 실행하는 구조다:

| 구성 요소 | 역할 |
|-----------|------|
| **패턴 스캔** | 차트 패턴, 가격 행동, 거래량 이상 신호 탐색 |
| **컨텍스트 평가** | 시장 환경, 섹터 흐름, 매크로 상황과의 정합성 확인 |
| **작전 계획** | 진입가, 손절가, 목표가, 포지션 크기를 사전 확정 |
| **규율 실행** | 계획대로 실행, 실시간 감정적 수정 금지 |
| **사후 복기** | AAR을 통한 실행 품질 평가 및 개선점 도출 |

### 핵심 패턴 유형

1. **돌파 패턴**: 저항 돌파, 컵핸들, 베이스 탈출 등 모멘텀 진입
2. **풀백 패턴**: 상승 추세 중 지지 영역으로의 조정 후 반등 포착
3. **반전 패턴**: 과매도 상태에서의 바닥 패턴 인식
4. **거래량 패턴**: 비정상적 거래량 증가를 동반한 가격 움직임
5. **시간대 패턴**: 장 초반/중반/후반의 특성적 가격 행동

## 매매 기법

### 진입 (Entry)

- **패턴 확인 + 컨텍스트 정합**: 기술적 패턴이 시장 환경과 부합할 때만 진입
- **사전 주문**: 시장 개장 전에 조건부 주문을 설정하여 감정적 진입 방지
- **거래량 확인**: 패턴 돌파 시 거래량 증가를 필수 확인 조건으로 포함
- **다중 시간대**: 일봉, 주봉, 분봉 등 다중 시간대에서 패턴 일치 확인

### 청산 (Exit)

- **사전 목표가**: 진입 시 설정한 목표가에서 기계적으로 이익 실현
- **트레일링 스톱**: 추세가 지속될 때 이동평균선 또는 ATR 기반 트레일링
- **부분 청산**: 1차 목표에서 50% 실현, 나머지는 트레일링으로 관리
- **시간 제한**: 일정 기간 내 목표 방향 전개가 없으면 재평가

### 손절 (Stop-Loss)

- **절대 규칙**: 진입 전 설정한 손절 수준을 절대 변경하지 않음
- **기술적 기반**: 패턴의 무효화 지점(이전 저점 하회, 지지선 이탈)에 손절 설정
- **자본 기반**: 단일 거래 최대 손실 = 총 자본의 1-2%
- **연속 손절 프로토콜**: 3회 연속 손절 시 거래 중단, AAR 실행

### 포지션 사이징 (Position Sizing)

- **리스크 고정**: 거래당 리스크를 총 자본의 1-2%로 고정, 손절폭에 따라 주식 수 역산
- **컨빅션 가중**: 높은 확신 패턴에 약간의 추가 가중 (최대 2.5%)
- **드로우다운 연동**: 드로우다운 5% 이상 시 포지션 크기 50% 축소
- **점진적 복구**: 드로우다운 회복 시 단계적으로 포지션 크기 정상화

### 리스크 관리 (Risk Management)

- **일일 손실 한도**: 일일 최대 손실 도달 시 당일 거래 종료
- **주간 손실 한도**: 주간 최대 손실 시 주 마감까지 거래 중단
- **월간 손실 한도**: 월간 최대 손실 시 월말까지 거래 중단 + 전면 AAR
- **상관관계 리스크**: 동일 섹터/테마 포지션의 총 리스크 한도 관리

## 심리와 의사결정

### 핵심 멘탈 모델

- **전투 마인드셋**: 시장을 감정적으로 대하지 않고, 임무 수행의 관점에서 접근한다.
- **프로세스 신뢰**: 개별 결과가 아닌 프로세스의 장기적 우위를 신뢰한다.
- **스트레스 관리**: 군 경험에서 익힌 고압 상황에서의 침착한 의사결정 능력을 활용한다.
- **팀워크 사고**: 여러 포지션을 "분대원"처럼 관리하며, 각자의 역할과 전체 임무의 조화를 추구한다.

### Netto의 심리 규칙

1. 작전 계획 없이 시장에 진입하지 마라
2. 감정은 인지하되 행동에 반영하지 마라 — 절차에 따라 실행하라
3. 모든 거래 후 AAR을 실행하라 — 복기하지 않은 거래에서는 배울 수 없다
4. 연속 손실은 규율의 시험이다 — 규율을 유지한 자가 회복한다
5. 최고의 전투는 피할 수 있는 전투다 — 기회가 아니면 참여하지 마라

## 대표 사례

### 성공 사례: 패턴 인식 + 규율 실행

- **배경**: 시장 조정 후 특정 섹터에서 반복적으로 나타나는 돌파 패턴을 인식
- **전략**: 사전 작전 계획 수립 후, 패턴 완성 시 기계적으로 진입, 사전 설정한 손절과 목표가로 관리
- **핵심 요인**:
  - 시장 환경(Omega)과 패턴(Alpha)의 정합성을 사전 확인
  - 진입 후 계획 수정 없이 규율 있게 실행
  - AAR을 통해 성공 요인을 문서화하고 향후 유사 패턴에 적용
- **교훈**: 좋은 패턴 + 좋은 환경 + 규율 있는 실행 = 반복 가능한 수익

### 실패/교훈 사례: 작전 계획 이탈

- **배경**: 빠르게 움직이는 시장에서 사전 계획 없이 충동적으로 진입
- **원인**: FOMO에 의한 계획 이탈, 군사적 규율의 일시적 붕괴
- **대응**: 손절 후 즉시 AAR 실행, 충동 진입의 트리거 분석
- **교훈**: "시장이 빠르게 움직일 때가 가장 계획이 필요한 순간이다. 급할수록 돌아가라."

### 전략군 (Strategy Family)

- **Family B+C (Momentum + Swing)**: Netto의 핵심 전략군. 패턴 기반 모멘텀 진입과 스윙 트레이딩
- **Alpha+Beta**: 기술적 패턴(Alpha) 인식과 추세 추종(Beta)의 결합

### 에이전트 매핑

| 에이전트 | Netto 요소 | 구현 내용 |
|----------|-----------|----------|
| **Alpha** | 패턴 인식 | VCP, 돌파, 풀백 등 핵심 패턴 자동 감지 및 신호 생성 |
| **Beta** | 추세 추종 | 패턴 돌파 후 추세 지속 구간에서의 포지션 관리 |
| **Delta** | 리스크 관리 / 사이징 | 거래당 1-2% 리스크 고정, 드로우다운 연동 축소 로직 |
| **Omega** | 시장 컨텍스트 | 시장 레짐에 따른 패턴 유효성 필터 |

### 구현 아이디어

1. **자동 패턴 스캐너**: 돌파, 풀백, 컵핸들 등 핵심 패턴을 실시간으로 스캔하고 알림
2. **다중 시간대 정합성 필터**: 일봉/주봉/분봉에서 동시에 신호가 일치하는 경우만 통과
3. **AAR 자동화**: 모든 거래 종료 후 자동으로 진입 근거, 실행 품질, 결과 분석 리포트 생성
4. **드로우다운 자동 대응**: 연속 손실/드로우다운 확대 시 자동 포지션 축소 및 거래 제한
5. **패턴-컨텍스트 상관 분석**: 과거 데이터에서 특정 패턴이 특정 시장 환경에서 어떤 성과를 보였는지 통계 분석
