# David Shaw

| 학력 | Stanford University 컴퓨터 과학 박사 |

| 주요 회사 | D.E. Shaw & Co. (1988년 설립) |

## 한 줄 요약

> **"컴퓨터 과학과 수학을 금융에 적용하여, 머신러닝과 통계적 차익거래로 시장의 미세한 비효율을 대규모로 포착하는 정량적 컴퓨팅 선구자."**

## 핵심 투자 철학 (5 Principles)

1. **컴퓨팅 파워가 엣지다 (Computing Power is the Edge)**
   - 인간의 분석 능력을 초월하는 대규모 데이터 처리가 핵심 경쟁력이다.
   - 수천 개의 변수를 동시에 분석하여 미세한 패턴을 포착한다.
   - 기술적 인프라(하드웨어, 소프트웨어, 네트워크)에 대한 투자가 수익의 원천이다.

2. **통계적 엣지의 대규모 반복 (Statistical Edge at Scale)**
   - 개별 거래의 엣지는 미세하지만, 수천~수만 건의 거래를 통해 복리화된다.
   - "한 번의 거래에서 큰 수익"이 아닌 "작은 엣지 × 대규모 반복"이 전략이다.
   - 대수의 법칙이 작동하도록 충분한 거래 빈도와 분산을 유지한다.

3. **시장에 패턴은 존재한다 — 발견이 어려울 뿐 (Patterns Exist, Discovery is Hard)**
   - 효율적 시장 가설은 불완전하다. 시장에는 통계적으로 유의한 패턴이 존재한다.
   - 이 패턴은 인간의 눈으로는 보이지 않을 정도로 미세하고 복잡하다.
   - 컴퓨터 과학과 통계학의 결합만이 이 패턴을 발견할 수 있다.

4. **인재가 최고의 자산이다 (Talent is the Greatest Asset)**
   - D.E. Shaw는 PhD급 과학자, 수학자, 프로그래머를 적극 채용한다.
   - 금융 배경이 아닌 과학/기술 배경의 인재가 새로운 시각을 제공한다.
   - Jeff Bezos가 D.E. Shaw 출신인 것은 이 인재 철학의 증거이다.

5. **기술은 빠르게 진화하므로 끊임없이 혁신하라 (Constant Innovation)**
   - 오늘의 엣지는 내일 사라질 수 있다. 경쟁자도 기술을 발전시키기 때문이다.
   - 지속적인 리서치와 모델 개선이 장기적 생존의 열쇠이다.
   - 현재 전략에 안주하지 않고 항상 다음 세대의 전략을 준비한다.

## 대표 전략 유형

### 통계적 차익거래 (Statistical Arbitrage)

대규모 데이터 분석과 컴퓨팅으로 시장의 미세한 가격 비효율을 포착:

| 구성 요소 | 설명 |
|-----------|------|
| **다중 팩터 모델** | 수백 개의 팩터를 동시에 분석하여 종목 순위화 |
| **페어/바스켓 트레이딩** | 관련 종목 간 가격 괴리를 포착하여 시장 중립 포지션 구축 |
| **고빈도 실행** | 미세한 엣지를 대규모로 반복 실행하여 수익 누적 |
| **머신러닝 적용** | 패턴 인식, 예측 모델, 비정형 데이터 분석에 ML 활용 |

### 머신러닝 기반 전략

- **지도 학습**: 과거 데이터에서 수익률 예측 모델 훈련
- **비지도 학습**: 종목 클러스터링, 시장 레짐 분류
- **자연어 처리**: 뉴스, 공시, 소셜미디어 데이터에서 센티먼트 추출
- **강화 학습**: 동적 포트폴리오 최적화, 실행 알고리즘 개선
- **비정형 데이터**: 위성 이미지, 신용카드 데이터 등 대안 데이터 분석

## 매매 기법

### 진입 (Entry)

- **모델 신호**: 다중 팩터/ML 모델이 통계적으로 유의한 신호를 생성할 때 자동 진입
- **시장 중립 구성**: 롱 포지션과 숏 포지션을 동시에 구축하여 시장 리스크 제거
- **실행 최적화**: 시장 충격을 최소화하는 알고리즘 실행 (VWAP, TWAP 등)
- **다수 소규모 포지션**: 수백~수천 개의 소규모 포지션으로 극대 분산
- **실시간 처리**: 시장 데이터를 실시간으로 처리하여 즉각적 의사결정

### 청산 (Exit)

- **모델 역전**: 동일 모델이 반대 신호를 생성하면 포지션 청산
- **알파 소멸**: 엣지가 통계적으로 유의하지 않게 되면 포지션 축소
- **리밸런싱**: 정기적/비정기적 포트폴리오 리밸런싱을 통한 포지션 조정
- **리스크 한도**: 포트폴리오 레벨의 리스크 한도 초과 시 자동 포지션 축소

### 손절 (Stop-Loss)

- **포트폴리오 레벨**: 개별 종목이 아닌 전체 포트폴리오 단위로 리스크 관리
- **팩터 노출 한도**: 특정 팩터에 대한 노출이 한도를 초과하면 자동 조절
- **드로다운 한도**: 전략별/포트폴리오별 최대 드로다운 한도 설정
- **모델 성능 모니터링**: 모델의 실시간 성능이 기대치에서 벗어나면 자동 축소

### 포지션 사이징 (Position Sizing)

- **최적화 기반**: 포트폴리오 최적화 알고리즘으로 각 포지션의 최적 비중 결정
- **리스크 버짓**: 전체 리스크 버짓을 팩터별, 전략별로 배분
- **동적 조정**: 시장 변동성, 상관관계 변화에 따라 실시간으로 비중 조정
- **유동성 제약**: 시장 유동성을 고려한 최대 포지션 크기 제한

### 리스크 관리 (Risk Management)

- **다층적 리스크 관리**: 포지션 → 전략 → 포트폴리오 → 회사 레벨의 다층 관리
- **팩터 리스크 분해**: 시장, 섹터, 스타일 등 팩터별 리스크 기여도 실시간 분석
- **스트레스 테스트**: 2008 금융위기 등 극단적 시나리오 시뮬레이션
- **모델 리스크 관리**: 다수의 독립적 모델을 운용하여 단일 모델 실패 위험 분산
- **기술 인프라 리스크**: 시스템 장애, 네트워크 문제 등 기술 리스크에 대한 이중화

## 심리와 의사결정

### 핵심 멘탈 모델

- **과학적 방법론**: 가설 → 검증 → 반복의 과학적 프로세스를 투자에 적용한다.
- **데이터 주도**: 직관이나 경험이 아닌 데이터와 통계에 기반한 의사결정을 한다.
- **지적 겸손**: 시장은 복잡계이며, 완벽한 예측은 불가능함을 인정한다.
- **팀 기반 의사결정**: 개인의 천재성보다 팀의 집단 지성을 신뢰한다.

### Shaw의 조직 문화 원칙

1. 최고의 인재를 채용하고, 자유로운 연구 환경을 제공한다
2. 실패를 허용하되, 같은 실패를 반복하지 않는다
3. 학문적 엄밀함과 실용적 성과를 모두 추구한다
4. 비밀 유지 — 전략의 세부 사항은 외부에 공개하지 않는다
5. 기술과 금융의 경계를 허물고 융합한다

## 대표 사례

### 성공 사례: D.E. Shaw의 성장

- **배경**: 1988년 2,800만 달러로 시작하여 세계 최대 퀀트 헤지펀드 중 하나로 성장
- **전략**: 통계적 차익거래, ML 기반 예측 모델, 다양한 자산 클래스 확장
- **핵심 요인**:
  - 컴퓨터 과학자의 관점에서 금융을 재해석
  - PhD급 인재의 지속적 확보
  - 기술 인프라에 대한 과감한 투자
  - 다수의 비상관 전략 동시 운용
- **교훈**: 금융은 궁극적으로 정보 처리 문제이며, 컴퓨팅 역량이 핵심 경쟁력이다

### 선구적 사례: 머신러닝의 조기 적용

- **배경**: 1990년대 초, 머신러닝이 학술 영역에 머물러 있을 때 금융에 적용을 시작
- **혁신**: 신경망, 의사결정 트리 등 ML 기법으로 시장 패턴 인식
- **결과**: "머신러닝 + 금융"의 선구자로 업계 전체의 패러다임에 영향
- **교훈**: 기술 혁신의 조기 채택이 장기적 경쟁 우위를 만든다

### 도전 사례: 2007-2008 퀀트 위기

- **배경**: 2007년 8월 퀀트 헤지펀드들이 동시에 손실을 겪는 "퀀트 지진" 발생
- **원인**: 유사한 전략을 사용하는 펀드들의 동시 청산(crowded trade unwind)
- **대응**: 리스크 관리 시스템의 강화, 전략 다양성 확대, 유동성 관리 개선
- **교훈**: 퀀트 전략의 군집(crowding) 리스크는 가장 위험한 시스템 리스크 중 하나

### 전략군 (Strategy Family)

- **Family Quant**: Shaw는 퀀트 투자의 궁극적 형태를 대표
- **기술 비전 참고**: OmX 시스템의 장기 기술 로드맵에 핵심 영감

### 에이전트 매핑

| 에이전트 | Shaw 요소 | 활용 방안 |
|----------|----------|----------|
| **Alpha** | 다중 팩터 모델 | 팩터 수 확장 및 ML 기반 종목 스코어링 (장기 목표) |
| **Beta** | 패턴 인식 ML | VCP 감지에 CNN/시계열 모델 적용 (장기 목표) |
| **Omega** | 포트폴리오 최적화 | 다중 에이전트 출력의 ML 기반 통합 |
| **시스템 전체** | 기술 인프라 | 데이터 파이프라인, 실시간 처리 아키텍처 참고 |

### 구현 아이디어

1. **ML 기반 VCP 감지**: 컨볼루션 신경망(CNN)으로 VCP 패턴을 학습시켜 자동 감지 정확도 향상
2. **다중 팩터 스코어링 고도화**: Trend Template 외 추가 팩터를 ML 모델로 통합하여 종목 순위 개선
3. **대안 데이터 통합**: 뉴스 센티먼트, 소셜미디어 데이터를 보조 입력으로 활용
4. **실시간 데이터 파이프라인**: Shaw의 기술 인프라 개념을 참고한 실시간 데이터 처리 아키텍처
5. **전략 군집 탐지**: 유사 전략 운용자의 포지션 집중도를 모니터링하여 crowding 리스크 경고
