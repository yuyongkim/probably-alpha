# Claudio Guazzoni

## 기본 정보

| 항목 | 내용 |
|------|------|
| 본명 | Claudio Guazzoni |
| 활동 기간 | 1990년대 ~ 2000년대 |
| 인터뷰 수록 | *Stock Market Wizards* (Jack Schwager, 2001) |
| 운용 스타일 | Short Bias — 과대평가/악화 종목 공매도 |
| 핵심 전략 | 펀더멘탈 악화 + 과대평가 종목의 체계적 공매도 |
| 핵심 특징 | 리스크 집착적 포지션 관리, 신중한 사이징 |
| 주요 실적 | 강세장에서도 공매도로 수익 달성 |
| 배경 | 리스크 관리 중심의 보수적 숏 셀러 |

## 한 줄 요약

> **"과대평가되거나 펀더멘탈이 악화되는 종목을 정밀하게 식별하여 공매도하되, 극도로 신중한 포지션 사이징과 리스크 관리로 강세장에서도 생존하는 숏 바이어스 트레이더."**

## 핵심 투자 철학 (5 Principles)

1. **과대평가는 결국 수정된다 (Overvaluation Always Corrects)**
   - 시장은 장기적으로 펀더멘탈에 수렴한다.
   - 과도한 낙관, 투기적 열풍, 회계 조작에 의한 고평가는 반드시 붕괴한다.
   - "중력은 주식에도 적용된다 — 다만 시간이 걸릴 뿐이다."

2. **리스크가 최우선이다 (Risk Comes First)**
   - 공매도는 이론적으로 무한 손실이 가능하므로 리스크 관리가 생명이다.
   - 개별 포지션의 최대 손실을 항상 사전에 한정한다.
   - "얼마를 벌 수 있는지보다 얼마를 잃을 수 있는지를 먼저 생각한다."

3. **펀더멘탈 악화가 최고의 숏 근거다 (Deteriorating Fundamentals = Best Short)**
   - 단순히 "비싸다"는 이유만으로 숏하지 않는다.
   - 실적 악화, 마진 축소, 부채 증가, 경쟁력 상실 등 구체적 악화 징후를 확인한다.
   - 펀더멘탈 악화 + 과대평가의 조합이 이상적인 숏 후보이다.

4. **타이밍보다 생존이 중요하다 (Survival Over Timing)**
   - 숏 포지션이 궁극적으로 옳더라도 그 전에 스퀴즈로 파산하면 의미 없다.
   - 작은 사이즈와 넓은 손절로 생존을 보장한 후 시간이 일하도록 한다.
   - "맞을 때까지 살아남아야 한다."

5. **군중의 광기에 베팅하라 (Bet Against the Crowd's Madness)**
   - 시장의 과열, 버블, 투기적 광기는 반복적으로 발생한다.
   - 이러한 극단적 상황이 숏 셀러에게 최고의 기회를 제공한다.
   - 단, 타이밍은 예측 불가능하므로 포지션 관리가 핵심이다.

## 대표 전략 유형

### 펀더멘탈 숏 셀링 (Fundamental Short Selling)

펀더멘탈이 악화되고 있으나 시장이 아직 인식하지 못한 종목을 공매도:

| 구성 요소 | 설명 |
|-----------|------|
| **실적 악화 탐지** | 매출 둔화, 마진 축소, 실적 하향 조정 추세 |
| **회계 분석** | 공격적 회계 처리, 숨겨진 부채, 현금흐름 괴리 |
| **과대평가 측정** | PER, PBR, EV/EBITDA 등이 역사적/동종업계 대비 극단적 수준 |
| **촉매 식별** | 실적 발표, 감사 보고서, 경쟁 심화 등 하락 촉매 |

### 포지션 관리 원칙

- **소규모 진입**: 개별 숏 포지션을 매우 작게 시작
- **확인 시 추가**: 펀더멘탈 악화가 진행될수록 점진적 추가
- **스퀴즈 대비**: 숏 스퀴즈를 견딜 수 있는 여유 자금 항상 확보
- **다수 종목 분산**: 개별 종목 리스크를 분산하기 위해 다수 숏 포지션 유지

## 매매 기법

### 진입 (Entry)

- **펀더멘탈 악화 확인**: 최소 2개 이상의 악화 지표가 확인된 종목만 숏 후보
- **과대평가 확인**: 밸류에이션이 역사적/업종 평균 대비 상위 분위에 위치
- **기술적 보조**: 상승 추세가 약화되거나 저항선에 근접한 타이밍 선호
- **촉매 근접**: 실적 발표, 규제 심사 등 하락 촉매가 임박한 시점에 진입
- **소규모 시작**: 전체 계획 포지션의 25-33%로 초기 진입

### 청산 (Exit)

- **목표가 달성**: 적정 가치 수준까지 하락하면 수익 실현
- **펀더멘탈 개선**: 악화 추세가 반전되거나 개선 징후가 나타나면 청산
- **리스크 한도**: 사전 설정 손실 한도 도달 시 무조건 청산
- **분할 청산**: 목표가 접근 시 단계적으로 포지션 축소

### 손절 (Stop-Loss)

- **절대 한도**: 개별 포지션 손실이 사전 설정 한도(예: 투입 자본의 15-20%) 도달 시 청산
- **기술적 손절**: 신고가 돌파 시 숏 테시스 재검토, 필요 시 손절
- **포트폴리오 한도**: 전체 포트폴리오 드로다운이 한도 초과 시 전체 포지션 축소
- **스퀴즈 대응**: 급격한 상승 시 패닉이 아닌 규칙에 따라 사이즈 축소

### 포지션 사이징 (Position Sizing)

- **매우 보수적**: 개별 숏 포지션은 전체 자본의 2-5% 수준으로 제한
- **무한 손실 인식**: 주가 상승에 이론적 한도가 없으므로 사이징이 곧 리스크 관리
- **변동성 반영**: 변동성이 높은 종목일수록 사이즈를 더 축소
- **다수 분산**: 10-20개 이상의 숏 포지션으로 개별 종목 리스크 분산
- **총 숏 노출 관리**: 전체 숏 익스포저가 자본의 일정 비율을 초과하지 않도록 관리

### 리스크 관리 (Risk Management)

- **숏 스퀴즈 대비**: 차입 비용, 대차 가능성, 유동성을 사전에 확인
- **섹터 분산**: 특정 섹터에 숏 포지션이 집중되지 않도록 분산
- **시장 환경 인식**: 강세장에서는 숏 사이즈를 더 줄이고, 약세장에서 확대
- **현금 버퍼**: 마진 콜과 스퀴즈 상황에 대비한 충분한 현금 여유
- **상관관계 모니터링**: 숏 포지션 간 상관관계가 높아지지 않는지 지속 점검

## 심리와 의사결정

### 핵심 멘탈 모델

- **역발상 사고**: 시장의 낙관에 의문을 제기하고, 문제를 찾는 데 특화된 사고방식.
- **인내심과 고독**: 숏 셀러는 대부분의 시간 동안 "틀린" 것처럼 보인다. 이를 견디는 인내가 필수.
- **확률적 생존**: "결국 맞다"는 것보다 "맞을 때까지 살아남는 것"이 더 중요하다.
- **보수적 공격성**: 기회를 포착할 때는 공격적이되, 항상 보수적 사이징으로 생존을 보장한다.

### Guazzoni의 심리 규칙

1. 주가가 "비싸다"는 이유만으로 숏하지 않는다 — 비싼 것은 더 비싸질 수 있다
2. 시장이 내 포지션에 불리하게 움직여도 화내지 않는다 — 시장은 내 의견에 관심 없다
3. 숏 스퀴즈에 버틸 수 있는 크기만 베팅한다
4. 펀더멘탈이 옳다면 시간이 해결해줄 것이다 — 단, 생존해야 한다
5. 강세장에서 영웅이 되려 하지 않는다 — 약세장을 기다린다

## 대표 사례

### 성공 사례: 과대평가 종목의 펀더멘탈 붕괴

- **배경**: 시장에서 인기를 끌던 종목의 펀더멘탈 악화 징후를 조기에 포착
- **전략**: 회계 분석을 통해 공격적 매출 인식, 마진 축소 추세를 확인하고 소규모 숏 진입
- **핵심 요인**:
  - 시장이 여전히 낙관적일 때 독립적 분석으로 문제점 식별
  - 소규모 포지션으로 시작하여 악화 확인 시 점진적 추가
  - 실적 발표에서 문제가 표면화되며 주가 급락, 유의미한 수익 실현
- **교훈**: 독립적 펀더멘탈 분석 + 인내심 + 보수적 사이징의 삼위일체

### 도전 사례: 강세장에서의 숏 스퀴즈

- **배경**: 분석적으로 올바른 숏 포지션이 강세장 모멘텀에 의해 역방향으로 크게 이동
- **결과**: 단기적으로 유의미한 미실현 손실 발생
- **대응**: 보수적 사이징 덕분에 손절 한도에 도달하지 않고 버틸 수 있었음
- **후속**: 결국 펀더멘탈 악화가 표면화되며 주가 하락, 포지션 수익으로 전환
- **교훈**: 숏 셀링에서 사이징은 단순한 리스크 관리가 아닌 전략의 핵심 구성 요소

### 전략군 (Strategy Family)

- **Family G (Short Bias)**: Guazzoni는 체계적 숏 셀링의 대표 사례
- **Agent Short (미래 확장)**: OmX 시스템에 숏 에이전트를 추가할 때 핵심 참고 인물

### 에이전트 매핑

| 에이전트 | Guazzoni 요소 | 활용 방안 |
|----------|--------------|----------|
| **Alpha (역방향)** | Stage 3/4 종목 필터 | Trend Template 조건의 역전 — 하락 추세 종목 식별 |
| **Gamma (역방향)** | 실적 악화 감지 | 실적 감속, 마진 축소, 하향 조정 종목 자동 식별 |
| **Delta** | 숏 리스크 관리 | 숏 포지션 특유의 리스크(무한 손실, 스퀴즈) 반영 사이징 |
| **Short Agent (미래)** | 전체 숏 전략 | Guazzoni의 방법론을 자동화한 전용 숏 에이전트 |

### 구현 아이디어

1. **역 Trend Template**: Stage 3/4로 전환 중인 종목을 자동 식별하는 하락 추세 필터
2. **펀더멘탈 악화 스캐너**: 매출 둔화율, 마진 변화, 부채 비율 악화를 자동 감지하는 모듈
3. **숏 스퀴즈 리스크 지표**: 공매도 비율, 대차 비용, 유동성을 종합한 스퀴즈 리스크 스코어
4. **회계 품질 필터**: 공격적 회계 처리 징후를 자동 탐지 (매출 인식 패턴, 현금흐름 괴리 등)
5. **숏 포지션 모니터링 대시보드**: 숏 포지션의 실시간 손익, 대차 비용, 스퀴즈 리스크를 통합 표시
