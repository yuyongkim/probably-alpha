## 핵심 투자 철학 (5 Principles)

1. **엣지가 없으면 매매하지 마라 (No Edge, No Trade)**
   - 통계적/수학적으로 증명 가능한 우위(edge)가 있을 때만 매매한다.
   - "느낌", "직관", "경험"은 엣지가 아니다. 정량적으로 측정 가능해야 한다.

2. **Kelly Criterion이 최적 사이즈를 결정한다 (Kelly Criterion Determines Optimal Size)**
   - 엣지의 크기에 비례하여 포지션 사이즈를 결정한다.
   - 과소 베팅은 기회비용, 과대 베팅은 파산 리스크. Kelly가 균형점을 찾아준다.

3. **장기적으로 확률은 수렴한다 (In the Long Run, Probability Converges)**
   - 단일 거래의 결과는 무의미하다. 대수의 법칙이 장기 성과를 결정한다.
   - 충분한 반복 횟수가 확보되면 엣지가 확실하게 수익으로 전환된다.

4. **리스크는 정량적으로 관리한다 (Risk Is Managed Quantitatively)**
   - 감정이 아닌 수학으로 리스크를 측정하고 관리한다.
   - 최대 드로다운, 표준편차, VAR 등 정량 지표 기반 리스크 관리.

5. **시장의 비효율성은 존재한다 (Market Inefficiencies Exist)**
   - 효율적 시장 가설은 이론적으로 우아하지만 현실에서는 불완전하다.
   - 워런트 미스프라이싱, 전환사채 차익거래, 통계적 차익거래 등 비효율이 수익의 원천.

## 매매 기법

### 진입 (Entry)

- **엣지 확인**: 수학적 모델로 현재 시장 가격의 미스프라이싱을 정량적으로 확인
- **헤지 동시 구축**: 진입 시 방향성 리스크를 제거하는 반대 포지션을 동시에 구축
- **Kelly 사이즈**: 엣지의 크기에 따라 Kelly Criterion으로 포지션 크기 결정
- **유동성 확인**: 진입과 청산이 원활한 유동성이 확보된 경우에만 매매
- **자동 실행**: 사전 프로그래밍된 규칙에 따라 자동으로 진입

### 청산 (Exit)

- **미스프라이싱 해소**: 가격이 이론가에 수렴하면 양쪽 포지션을 동시 청산
- **시간 기반**: 미스프라이싱이 일정 기간 내 해소되지 않으면 기회비용 고려 청산
- **모델 업데이트**: 새로운 데이터로 모델이 업데이트되어 엣지가 소멸하면 청산
- **리스크 한도**: 포지션의 리스크가 사전 설정 한도를 초과하면 축소/청산

### 손절 (Stop-Loss)

- **모델 기반**: 가격 수준이 아닌 모델 기반 신호로 손절 결정
- **헤지 보호**: 헤지된 포지션은 방향성 리스크가 제한되어 대규모 손절 빈도 낮음
- **최대 드로다운**: 전체 포트폴리오 드로다운 한도 엄격 설정
- **시스템 정지**: 모델 성과가 기대치를 유의미하게 하회하면 전략 자체를 일시 정지

### 포지션 사이징 (Position Sizing)

- **Kelly Criterion**: f* = (bp - q) / b (b=배당률, p=승률, q=패율)
- **분수 Kelly**: 실전에서는 Full Kelly의 50-75% 사용 (Half Kelly 선호)
- **근거**: Full Kelly는 이론적 최적이지만 변동성이 극심. Half Kelly는 성장의 75%를 유지하면서 변동성을 크게 줄임
- **다중 포지션**: 여러 독립적 엣지에 자본을 분산하여 전체 Kelly 최적화

### 리스크 관리 (Risk Management)

- **분산**: 상관관계가 낮은 다수의 포지션으로 개별 리스크 상쇄
- **헤지 리스크**: 헤지의 불완전성(basis risk)을 지속적으로 모니터링
- **모델 리스크**: 모델이 현실을 완벽히 반영하지 못할 가능성을 항상 인식
- **유동성 리스크**: 시장 스트레스 시 유동성 증발에 대한 시나리오 분석
- **운영 리스크**: 시스템 오류, 데이터 오류 등 비시장 리스크도 체계적 관리

## 심리와 의사결정

### 핵심 멘탈 모델

- **수학적 확신**: 감정이 아닌 수학에 기반한 확신. 엣지가 증명되면 단기 손실에 동요하지 않는다.
- **대수의 법칙 신뢰**: 충분한 반복이 보장되면 확률이 수렴한다는 믿음. 개별 결과에 초연.
- **겸손한 과학자**: 자신의 모델이 틀릴 수 있음을 항상 인정. 시장에서 살아남는 것이 옳은 것보다 중요하다.
- **지적 호기심**: 시장을 수익 추구가 아닌 지적 퍼즐로 접근. 이 태도가 감정적 의사결정을 방지한다.

### Thorp의 심리 규칙

1. 엣지를 정량적으로 증명할 수 없으면 매매하지 마라
2. 감정으로 사이즈를 결정하지 마라. Kelly가 결정한다
3. 단기 결과에 일희일비하지 마라. 장기 기대값에 집중하라
4. 자신의 모델을 맹신하지 마라. 현실 검증을 지속하라
5. 시장에서 살아남는 것이 돈을 버는 것보다 중요하다

## 대표 사례

### 성공 사례: Princeton Newport Partners (1969-1988)

- **배경**: Thorp가 공동 설립한 헤지펀드, 19년간 운용
- **성과**: 연평균 19.8% 수익률, 227개월 중 손실 월 3회, 극히 높은 Sharpe Ratio
- **전략**: 전환사채 차익거래, 워런트 차익거래, 통계적 차익거래
- **핵심**: 수학적 엣지 + Kelly 사이징 + 체계적 헤지 = 거의 무위험 수익
- **교훈**: 수학적으로 증명된 엣지를 규율 있게 실행하면 장기적 일관성이 보장된다

### 성공 사례: Beat the Dealer — 블랙잭 카드 카운팅

- **배경**: 1962년 출간한 *Beat the Dealer*에서 블랙잭 카드 카운팅으로 카지노를 이길 수 있음을 수학적으로 증명
- **투자 연결**: 확률 우위 + 최적 베팅 사이즈(Kelly)의 원리를 금융 시장에 직접 적용
- **결과**: 카드 카운팅의 성공이 금융 시장에서의 정량적 접근에 대한 확신의 기반
- **교훈**: 엣지의 원리는 카지노든 시장이든 동일하다. 측정 → 사이징 → 반복 실행

### 실패 사례: LTCM과의 교훈적 대비

- **배경**: Thorp는 LTCM의 전략을 분석하고 과도한 레버리지와 유동성 리스크를 경고
- **LTCM**: 유사한 차익거래 전략이었지만 극단적 레버리지로 1998년 붕괴
- **Thorp의 차이**: Kelly Criterion에 따른 보수적 사이징, 유동성 리스크 관리
- **교훈**: 동일한 엣지라도 사이징이 잘못되면 파산한다. Kelly의 Half Kelly가 Full Kelly보다 현실적인 이유

### 전략군 (Strategy Family)

- **Family Quant (Quantitative/Statistical)**: Thorp의 핵심 전략군. 수학적 모델 기반 정량 매매

### 에이전트 매핑

| 에이전트 | Thorp 요소 | 구현 내용 |
|----------|-----------|----------|
| **Delta** | Kelly Criterion | 엣지 크기 × 승률 기반 최적 포지션 사이즈 자동 계산 |
| **Delta** | 리스크 정량화 | VAR, 최대 드로다운, Sharpe Ratio 실시간 모니터링 |
| **Omega** | 엣지 통합 | 다중 에이전트 스코어의 기대값 계산, 엣지 유무 판정 |
| **Alpha** | 통계적 필터 | Trend Template 조건의 통계적 유의성 검증 |
| **Beta** | 패턴 확률 | VCP 패턴의 과거 성공률 통계 기반 확률 부여 |

### 구현 아이디어

1. **Kelly Criterion 사이징 엔진**: 각 신호의 과거 승률과 R:R을 기반으로 Kelly 최적 사이즈 자동 계산 (Half Kelly 적용)
2. **엣지 대시보드**: 전체 시스템의 엣지(기대값)를 실시간 표시. 엣지가 소멸되면 매매 중단 신호
3. **Monte Carlo 시뮬레이션**: 현재 전략의 미래 성과 분포를 시뮬레이션하여 최악의 시나리오 대비
4. **Sharpe Ratio 모니터링**: 롤링 Sharpe Ratio 추적으로 전략 성과 저하 조기 감지
5. **모델 검증 파이프라인**: 정기적으로 백테스트 결과와 실제 결과를 비교, 통계적 유의성 검증
