## 한 줄 요약

> **"정확한 진입 시점(Specific Entry Point)에서 Stage 2 상승 추세 종목만을 매수하고, 엄격한 손절로 자본을 지키는 것이 복리 수익의 핵심이다."**

## 핵심 투자 철학 (5 Principles)

1. **추세 추종 (Trend Following)**
   - Stage 2 상승 추세에 있는 종목만 매수한다. Stage 1(바닥 다짐), Stage 3(천장), Stage 4(하락)는 절대 진입하지 않는다.
   - Stan Weinstein의 Stage Analysis를 기반으로 하되, 더 정밀한 정량적 조건(Trend Template)을 적용한다.

2. **변동성 수축이 기회다 (Volatility Contraction = Opportunity)**
   - 큰 상승 후 변동성이 줄어드는 구간(VCP)이 최적의 진입 시점이다.
   - 변동성 수축은 매도 압력 소진과 기관 매집을 의미한다.

3. **리스크 관리가 수익을 만든다 (Risk Management Creates Returns)**
   - 단일 거래 최대 손실 7-8%. 절대 예외 없음.
   - R:R(Risk-Reward) 최소 2:1 이상인 거래만 취한다.
   - "큰 손실을 피하면 큰 수익은 자연히 따라온다."

4. **시장이 옳다 (The Market Is Always Right)**
   - "주식은 당신이 그것을 보유하고 있는지 모른다(The stock doesn't know you own it)."
   - 자존심과 매매를 분리한다. 손절은 실패가 아니라 생존 전략이다.

5. **소수 정예 집중 (Concentrated Best Ideas)**
   - 확신 높은 소수 종목에 집중 투자한다.
   - 분산 투자는 무지에 대한 보험이며, 확신이 있다면 집중이 답이다.

## 대표 전략 유형

### SEPA (Specific Entry Point Analysis)

SEPA는 Minervini가 독자적으로 개발한 체계적 진입 분석 프레임워크로, 다음 요소를 통합한다:

| 구성 요소 | 역할 |
|-----------|------|
| **Trend Template** | 8가지 정량 조건으로 Stage 2 상승 추세 필터링 |
| **VCP (Volatility Contraction Pattern)** | 최적 진입 타이밍 포착 |
| **Fundamental Catalyst** | 실적 가속(earnings acceleration)을 통한 펀더멘탈 확인 |
| **Risk/Reward Calculus** | 진입 전 R:R 계산, 최소 2:1 확보 |
| **Market Direction** | 시장 전체 추세 확인 (M factor) |

### Trend Template 8 조건

Minervini의 핵심 필터인 Trend Template은 다음 8가지 정량 조건을 모두 만족해야 한다:

1. 현재가 > 150일 이동평균 (SMA150)
2. 현재가 > 200일 이동평균 (SMA200)
3. SMA150 > SMA200
4. SMA200가 최소 1개월 이상 상승 추세
5. SMA50 > SMA150 > SMA200
6. 현재가 > SMA50
7. 현재가 >= 52주 최저가의 130% 이상
8. 현재가 >= 52주 최고가의 75% 이내 (25% 이내 권장)

이 조건을 모두 통과하는 종목만이 Stage 2 상승 추세에 있다고 판단한다.

### VCP (Volatility Contraction Pattern)

VCP는 Minervini의 가장 대표적인 차트 패턴이다:

- **구조**: 2~4차례의 가격 수축(contraction) 파동으로 구성
- **각 수축 파동**: 이전 수축보다 폭이 좁아짐 (예: -25% → -15% → -8% → -4%)
- **거래량**: 수축이 진행될수록 거래량도 감소 (dry-up)
- **의미**: 매도 물량(overhead supply)이 소진되고 있음을 시각적으로 보여줌
- **진입**: 마지막 수축의 상단(pivot point)을 거래량 증가와 함께 돌파할 때 매수
- **실패**: 돌파 후 재차 pivot 아래로 하락하면 즉시 손절

```
예시: 3-wave VCP
$100 ──┐
       │  -20%     -12%      -6%
$80  ──┘──┐   $88──┐  $94──┐
          │       │       │
          └───────┘───────┘── Pivot $100 돌파 → 매수
          Volume ▼▼▼  ▼▼   ▼   (거래량 감소)
```

## 매매 기법

### 진입 (Entry)

- **조건**: Trend Template 8조건 통과 + VCP 형성 + Pivot Point 돌파
- **타이밍**: Pivot Point를 당일 거래량이 평균의 150% 이상일 때 돌파하는 순간
- **추격 매수 금지**: Pivot에서 5% 이상 벌어진 종목은 추격하지 않는다 (chasing)
- **분할 매수**: 1차 진입 후 추가 매수는 수익이 나고 있는 경우에만 (pyramiding up)

### 청산 (Exit)

- **목표가**: R:R 기반으로 설정. 최소 목표 수익은 진입 리스크의 2~3배
- **트레일링 스톱**: 큰 수익 발생 시 이동평균선(10일, 20일, 50일)을 트레일링 기준으로 활용
- **Climax Top 인식**: 급등 + 거래량 폭증 + 갭업 = 청산 신호
- **시간 손절**: 진입 후 2~3주 내 의미 있는 움직임이 없으면 청산 고려

### 손절 (Stop-Loss)

- **절대 규칙**: 매수가 대비 7-8% 하락 시 무조건 손절
- **조건부 타이트닝**: 시장 환경이 나쁘거나 확신이 낮으면 5-6%로 축소
- **Pivot 기준**: VCP Pivot 아래로 하락 시 즉시 손절 (7-8% 이전이라도)
- **절대 금지**: 손절선을 하향 조정하는 행위 (widening stops)

### 포지션 사이징 (Position Sizing)

- **단일 포지션**: 총 자본의 20-25% 이내
- **최대 노출**: 동시 보유 종목 4-5개 (집중 투자)
- **리스크 기반**: 각 포지션의 달러 리스크(entry - stop) × 수량 = 총 자본의 1-2%
- **확신도 반영**: 모든 셋업이 동일하지 않음. 최고 확신 셋업에 더 큰 사이즈

### 리스크 관리 (Risk Management)

- **연속 손절 규칙**: 3회 연속 손절 시 사이즈를 50% 축소하고 재평가
- **드로다운 한도**: 계좌 10% 드로다운 시 매매 중단, 시장 재평가
- **시장 노출**: 약세장에서는 현금 비중 80-100%까지 확대
- **복리 보호**: "큰 손실 1회 > 작은 이익 10회"를 항상 명심

## 심리와 의사결정

### 핵심 멘탈 모델

- **확률적 사고**: 개별 거래의 결과는 중요하지 않다. 100회, 1000회 거래의 기대값이 중요하다.
- **분리(Detachment)**: 포지션에 감정적으로 결합하지 않는다. 종목에 "사랑에 빠지지" 않는다.
- **과정 중심**: 결과(outcome)가 아닌 과정(process)에 집중. 올바른 과정은 장기적으로 올바른 결과를 만든다.
- **자기 인식**: 자신의 감정 상태를 지속적으로 모니터링. 흥분하거나 두려울 때 사이즈를 줄인다.

### Minervini의 심리 규칙

1. 손절은 비용이 아니라 보험료다
2. "맞다/틀리다"는 의미 없다. 돈을 벌었는가/잃었는가만 중요하다
3. 매일 아침 어제의 포지션을 처음 보는 것처럼 평가하라
4. 남의 의견은 무시하라. 차트와 가격만이 진실이다
5. 연승 중일 때가 가장 위험하다 — 자만은 최대의 적

## 대표 사례

### 성공 사례: 1997년 U.S. Investing Championship

- **배경**: Minervini는 1997년 U.S. Investing Championship에 참가하여 155%의 연간 수익률을 기록, 우승
- **전략**: SEPA 시스템을 체계적으로 적용. Stage 2 종목 중 VCP를 형성한 기술주에 집중
- **핵심 요인**:
  - 1997년 강세장에서 시장 방향(M factor)이 유리했음
  - 엄격한 손절로 드로다운을 5% 미만으로 유지
  - 수익이 나는 포지션에 추가 매수(pyramiding)하여 수익 극대화
- **교훈**: 올바른 시장 환경 + 체계적 시스템 + 엄격한 리스크 관리 = 경이적 수익

### 성공 사례: 초기 경력 — 5년 연평균 220%

- **배경**: Minervini는 약 $100,000에서 시작하여 5년간 연평균 220%의 수익률을 기록, 자산을 수백만 달러로 증식
- **전략**: 소형 성장주에 집중. VCP + 실적 가속 조합
- **핵심 요인**: 소형주의 높은 변동성을 VCP로 제어하며, 실적 서프라이즈를 촉매로 활용

### 실패 사례: 초기 학습기

- **배경**: Minervini는 거래 초기 약 6년간 지속적으로 돈을 잃었다
- **원인**:
  - 손절 규칙 없이 매매
  - 하락하는 종목을 물타기(averaging down)
  - 시장 방향을 무시하고 개별 종목에만 집중
  - 팁과 루머에 의존한 매매
- **전환점**: 체계적 연구를 시작하고, 과거 대형 상승주(superperformers)의 공통 특성을 분석
- **교훈**: "나는 6년간의 실패를 통해 무엇을 하지 말아야 하는지를 배웠다. 그것이 SEPA의 기초가 되었다."

## 내 시스템과의 연결

### SEPA 공통점

Minervini의 SEPA는 본 프로젝트(OmX)의 **핵심 근간**이다. 시스템 전체가 Minervini의 방법론을 디지털화/자동화한 것이다:

| Minervini 원본 | OmX 구현 |
|----------------|----------|
| Trend Template 8조건 | **Alpha 에이전트** — 8개 조건의 정량적 자동 스크리닝 |
| VCP 패턴 인식 | **Beta 에이전트** — 수축 파동 감지, 거래량 dry-up 확인 |
| R:R 계산 + 손절 | **Delta 에이전트** — 자동 리스크/리워드 계산, 포지션 사이징 |
| 최종 종합 판단 | **Omega 에이전트** — 모든 에이전트 출력을 통합하여 최종 신호 생성 |
| 실적 가속 확인 | **Gamma 에이전트** — EPS 가속 및 펀더멘탈 필터 |

### SEPA 차이점

| 항목 | Minervini 원본 | OmX 시스템 |
|------|----------------|-----------|
| 판단 방식 | 주관적 경험 + 정량 조건 혼합 | 완전 정량/알고리즘 기반 |
| VCP 인식 | 육안 차트 분석 | 알고리즘 기반 자동 감지 |
| 시장 방향 | 개인적 판단 | Market Regime 지표 기반 |
| 실행 속도 | 수동 주문 | 신호 발생 → 자동 알림/반자동 실행 |
| 유니버스 | 미국 주식 위주 | 한국/미국 확장 가능 |

### 전략군 (Strategy Family)

- **Family B (Growth Momentum)**: Minervini의 핵심 전략군. 실적 가속 + 추세 + 패턴의 삼위일체
- **Family E (VCP)**: VCP 패턴 전문 전략군. Minervini가 유일한 원조

### 에이전트 매핑

| 에이전트 | Minervini 요소 | 구현 내용 |
|----------|----------------|----------|
| **Alpha** | Trend Template | SMA50/150/200 관계, RS 순위, 52주 고저 비율 자동 계산 |
| **Beta** | VCP | 수축 파동 수, 수축률 감소 확인, 거래량 패턴 분석 |
| **Delta** | Risk Management | 7-8% 손절, R:R 계산, 포지션 사이징 자동화 |
| **Omega** | Final Decision | Alpha+Beta+Delta+Gamma 통합 스코어링 |

### 구현 아이디어

1. **VCP 자동 감지 알고리즘**: 최근 N일간의 고가-저가 범위(range)를 파동별로 측정, 감소 패턴 확인
2. **Pivot Point 자동 설정**: VCP 마지막 수축의 고점을 Pivot으로 자동 설정, 돌파 시 알림
3. **Trend Template 대시보드**: 8개 조건의 실시간 충족 현황을 시각화
4. **드로다운 기반 자동 사이즈 조절**: 연속 손절 또는 계좌 드로다운 시 자동으로 포지션 사이즈 축소
5. **Climax Top 감지**: 급등률 + 거래량 급증 + 갭 조합으로 매도 신호 자동 생성
