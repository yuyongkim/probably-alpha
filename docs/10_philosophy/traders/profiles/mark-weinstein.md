# Mark Weinstein

## 기본 정보

| 항목 | 내용 |
|------|------|
| 본명 | Mark Weinstein |
| 출생 | 1940년대, 미국 |
| 활동 기간 | 1970년대 ~ 현재 |
| 주요 저서 | 없음 (극도로 비공개적인 인물) |
| 인터뷰 수록 | *Market Wizards* (Jack Schwager, 1989) — 마지막 인터뷰이 |
| 주요 실적 | 극도로 높은 승률(90%+), S&P 500 트레이딩 대회에서 $100,000을 단기간에 수배 증식 |
| 운용 스타일 | Low-Volatility Squeeze / Psychology-First |
| 웹사이트 | 없음 (완전 비공개 생활) |

## 한 줄 요약

> **"변동성이 극도로 수축된 지점에서 진입하고, 심리적 규율로 자본을 절대적으로 보호한다."**

## 핵심 투자 철학 (5 Principles)

1. **저변동성 진입 (Low-Volatility Entry)**
   - 변동성이 극도로 줄어든 지점이 최적의 진입점이다.
   - 변동성 수축은 대형 움직임의 전조. 리스크가 최소인 지점에서 진입할 수 있다.

2. **자본 보호가 전부다 (Capital Preservation Is Everything)**
   - "나는 돈을 잃는 것을 견딜 수 없다." 극도의 손실 혐오가 높은 승률의 원동력.
   - 리스크가 확실히 제한된 상황에서만 진입. 조금이라도 불확실하면 관망.

3. **심리가 기술보다 중요하다 (Psychology Over Technique)**
   - 매매 기법은 10%, 심리가 90%이다.
   - 두려움, 탐욕, 자만심을 통제하는 것이 모든 기술적 분석보다 중요.

4. **인내와 선택 (Patience and Selectivity)**
   - 모든 기회에 참여할 필요 없다. 최고의 기회만 선택.
   - "나는 날마다 매매하지 않는다. 완벽한 셋업이 나타날 때까지 기다린다."

5. **준비가 성공을 결정한다 (Preparation Determines Success)**
   - 매매 전 철저한 준비. 시나리오별 행동 계획을 사전에 수립.
   - "전투는 시작되기 전에 이미 결정된다. 준비가 곧 승리다."

## 대표 전략 유형

### Low-Volatility Squeeze + Psychology-First

Weinstein의 전략은 극도의 변동성 수축 지점에서의 정밀 진입이다:

| 구성 요소 | 역할 |
|-----------|------|
| **변동성 수축 감지** | Bollinger Band 수축, ATR 최저치 등으로 스퀴즈 포착 |
| **다중 시장 스캔** | 주식, 원자재, 통화 등에서 동시에 스퀴즈 패턴 탐색 |
| **방향 판단** | 스퀴즈 방향을 거래량, 추세, 펀더멘탈로 예측 |
| **정밀 진입** | 스퀴즈 돌파 직전 또는 직후 진입 |
| **타이트 손절** | 스퀴즈 범위 아래에 극히 좁은 손절선 설정 |

### Bollinger Band Squeeze 예시

```
         ━━━━ Upper Band
        /    \
       /      ─── 수축 구간 (스퀴즈)
      /        \_________________________
     /          ──── Lower Band            \
    /                                       \
   수축 → 돌파 → 방향성 확장                  
                  ↑ 진입 지점                
```

## 매매 기법

### 진입 (Entry)

- **스퀴즈 확인**: Bollinger Band 폭이 N개월 최저, ATR이 역사적 저점 근처
- **방향 예측**: 장기 추세 방향, 거래량 패턴, 펀더멘탈로 돌파 방향 예측
- **정밀 타이밍**: 스퀴즈 경계 돌파 직후 진입. 너무 이른 진입은 방향 불확실
- **다중 시장**: 주식, 선물, 옵션 등 다양한 시장에서 스퀴즈 동시 탐색
- **극도의 선택성**: 10개 스퀴즈 중 1~2개만 진입. 완벽한 셋업만 매매

### 청산 (Exit)

- **목표 도달**: 스퀴즈 돌파 후 예상 이동 폭(밴드 폭의 1.5~2배) 도달 시 청산
- **모멘텀 약화**: 돌파 후 움직임의 속도가 둔화되면 부분 청산
- **반대 스퀴즈**: 반대 방향의 스퀴즈 패턴 출현 시 포지션 정리
- **시간 기반**: 보유 기간이 예상을 초과하면 재평가

### 손절 (Stop-Loss)

- **스퀴즈 범위 기준**: 스퀴즈 범위(밴드 하단 또는 최근 저점) 아래에 손절선
- **극히 타이트**: 변동성 수축 덕분에 손절 폭이 매우 작음 → 높은 R:R 가능
- **즉각 실행**: 손절선 도달 시 무조건 실행. "한 틱도 기다리지 않는다"
- **90% 승률의 비밀**: 스퀴즈 진입 + 타이트 손절 = 작은 손실, 큰 이익

### 포지션 사이징 (Position Sizing)

- **리스크 기반**: 타이트한 손절 폭 × 수량 = 자본의 1-2%
- **스퀴즈 품질**: 완벽한 스퀴즈에 더 큰 사이즈, 불완전한 스퀴즈는 최소 사이즈
- **동시 포지션 제한**: 동시에 3~5개 이내. 과도한 분산은 집중력 분산
- **확대 조건**: 스퀴즈 돌파 후 방향 확인 시 추가 매수 가능

### 리스크 관리 (Risk Management)

- **승률 기반**: 90%+ 승률로 드로다운 자체를 최소화
- **리스크/리워드 극대화**: 타이트한 손절로 R:R 3:1 이상 확보
- **시장 환경**: 전체 시장이 불안정할 때는 매매 자체를 중단
- **심리 관리**: 손실 후 즉시 복수 매매 금지. 냉정 회복 후 재진입

## 심리와 의사결정

### 핵심 멘탈 모델

- **두려움의 관리**: "두려움은 나쁜 것이 아니다. 두려움이 나를 조심스럽게 만들고, 그것이 나를 살린다"
- **완벽주의적 선택**: 조금이라도 의심이 들면 진입하지 않는다. 완벽한 셋업만 매매
- **명상과 집중**: 매매 전 정신적 준비를 철저히. 마음이 맑지 않으면 매매하지 않음
- **감사와 겸손**: 시장에 대한 경외심. "시장은 겸손하지 않은 자를 파괴한다"

### Weinstein의 심리 규칙

1. 매매는 80-90%가 심리다. 기법을 아는 것과 실행하는 것은 전혀 다르다
2. 손실을 견딜 수 없다면 그것은 포지션이 너무 크다는 뜻이다
3. 남의 의견을 듣지 마라. 특히 포지션을 보유한 후에는 절대로
4. 준비하지 않은 매매는 하지 마라. 모든 시나리오에 대한 계획이 있어야 한다
5. 시장을 존중하라. 시장은 항상 예상보다 멀리, 예상보다 오래 갈 수 있다

## 대표 사례

### 성공 사례: S&P 500 트레이딩 대회

- **배경**: Weinstein은 S&P 500 트레이딩 대회에서 $100,000를 단기간에 수배로 증식
- **전략**: 저변동성 스퀴즈 진입 + 극히 타이트한 손절 + 높은 R:R
- **핵심**: 90%+ 승률과 3:1+ R:R의 결합. 수학적으로 거의 잃을 수 없는 구조
- **교훈**: 최적의 진입 시점(스퀴즈)과 엄격한 리스크 관리의 결합이 경이적 성과를 만든다

### 성공 사례: 다중 시장 스퀴즈 트레이딩

- **배경**: 주식뿐 아니라 원자재, 통화 시장에서도 동일한 스퀴즈 전략 적용
- **전략**: 어떤 시장이든 변동성 수축 → 돌파 패턴은 동일하게 작동
- **결과**: 다중 시장에서의 기회 확보로 안정적 수익원 다각화
- **교훈**: 변동성 수축은 자산군에 관계없이 보편적으로 작동하는 패턴이다

### 실패 사례: 초기 경력 — 심리 관리의 중요성 학습

- **배경**: Weinstein은 초기에 기법적으로는 우수했으나 심리적으로 미숙하여 큰 손실 경험
- **원인**: 과도한 확신, 손절 지연, 복수 매매(revenge trading)
- **전환점**: 심리 관리를 최우선으로 두기 시작. "기법은 10%에 불과하다"는 깨달음
- **교훈**: "아무리 좋은 시스템도 심리가 뒷받침하지 않으면 무용지물이다"

## 내 시스템과의 연결

### SEPA 공통점

Weinstein의 저변동성 스퀴즈는 Minervini의 VCP와 본질적으로 동일한 원리이며, OmX의 핵심 기능이다:

| Weinstein 원칙 | OmX 구현 |
|----------------|----------|
| 변동성 수축 감지 | **Beta 에이전트** — VCP 수축 파동 감지 (Weinstein의 스퀴즈와 동일 원리) |
| 스퀴즈 돌파 진입 | **Beta 에이전트** — Pivot Point 돌파 신호 |
| 타이트한 손절 | **Delta 에이전트** — 스퀴즈 범위 기반 자동 손절선 |
| 높은 R:R | **Delta 에이전트** — R:R 자동 계산, 최소 2:1 필터 |
| 심리 관리 | **전체 시스템** — 자동화로 감정 개입 최소화 |

### SEPA 차이점

| 항목 | Weinstein 방식 | OmX 시스템 |
|------|---------------|-----------|
| 시장 | 주식, 원자재, 통화 다중 시장 | 한국/미국 주식 집중 |
| 방향 | 롱/숏 양방향 | 주로 롱 (Stage 2 상승) |
| 진입 근거 | 스퀴즈 단독으로 충분 | Trend Template + VCP 복합 조건 |
| 승률 목표 | 90%+ (극도의 선택성) | 50-60% (적절한 R:R과 조합) |
| 매매 빈도 | 매우 낮음 (완벽한 셋업만) | 중간 (Stage 2 + VCP 충족 시) |

### 전략군 (Strategy Family)

- **Family E (Volatility Pattern)**: Weinstein의 핵심 전략군. 변동성 수축/스퀴즈 기반 진입

### 에이전트 매핑

| 에이전트 | Weinstein 요소 | 구현 내용 |
|----------|----------------|----------|
| **Beta** | 스퀴즈 감지 | Bollinger Band 수축, ATR 최저치 자동 감지 |
| **Beta** | 돌파 방향 | 거래량 패턴, 추세 방향으로 돌파 방향 예측 |
| **Delta** | 타이트 손절 | 스퀴즈 범위 기반 자동 손절선 (매우 좁은 폭) |
| **Delta** | R:R 극대화 | 타이트 손절 + 넓은 목표가 → 3:1+ R:R 자동 계산 |
| **Omega** | 선택적 진입 | 스퀴즈 품질 점수가 임계값 이상일 때만 신호 발생 |

### 구현 아이디어

1. **스퀴즈 스크리너**: Bollinger Band 폭과 ATR이 N개월 최저인 종목 자동 추출
2. **스퀴즈 품질 점수**: 수축 기간, 수축 폭, 거래량 감소도를 종합하여 스퀴즈 품질 점수 산출
3. **방향 예측 모델**: 스퀴즈 돌파 방향을 장기 추세, 거래량 패턴, 섹터 모멘텀으로 예측
4. **VCP-스퀴즈 통합**: Minervini의 VCP와 Weinstein의 스퀴즈를 통합한 변동성 수축 종합 스코어
5. **심리 체크리스트**: 매매 전 심리 상태 자가 점검 프롬프트 (불안, 흥분, 복수심 등 확인)
