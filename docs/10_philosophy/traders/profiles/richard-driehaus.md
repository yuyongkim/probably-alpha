# Richard Driehaus

## 기본 정보

| 항목 | 내용 |
|------|------|
| 본명 | Richard H. Driehaus |
| 출생 | 1942년, 미국 시카고 (사망 2021년) |
| 활동 기간 | 1968년 ~ 2021년 |
| 주요 저서 | 직접 저서 없음 (인터뷰 및 강연 중심) |
| 인터뷰 수록 | *The New Market Wizards* (Jack Schwager, 1992) |
| 주요 실적 | Driehaus Capital Management 설립, 소형 성장주 펀드 연평균 30%+ 수익률 (1980~2000년대) |
| 운용 스타일 | Growth Momentum / Small-Cap Growth |
| 운용사 | Driehaus Capital Management (시카고) |

## 한 줄 요약

> **"비싸게 사서 더 비싸게 파는 것이 진정한 모멘텀 투자다 — 저가 매수의 함정에서 벗어나 이익 가속과 상대강도에 집중하라."**

## 핵심 투자 철학 (5 Principles)

1. **Buy High, Sell Higher (비싸게 사서 더 비싸게 팔아라)**
   - 대부분의 투자자는 "싸게 사서 비싸게 판다"는 원칙에 집착하지만, Driehaus는 이미 상승 추세에 진입한 종목을 매수하여 더 높은 가격에 매도하는 전략을 추구한다.
   - 신고가 부근의 종목이 더 높은 확률로 추가 상승한다는 것을 실증적으로 확인했다.
   - "바닥을 잡으려는 시도는 떨어지는 칼을 잡는 것과 같다."

2. **Earnings Surprise가 핵심 촉매다 (Earnings Surprise as Primary Catalyst)**
   - 실적 발표에서 컨센서스를 초과하는 어닝 서프라이즈가 주가 상승의 가장 강력한 촉매다.
   - 단순히 좋은 실적이 아니라, 시장의 기대를 "뛰어넘는" 실적이 중요하다.
   - 어닝 서프라이즈 이후 주가의 과소반응(underreaction) 현상을 적극 활용한다.

3. **상대강도 상위 종목에 집중하라 (Relative Strength Top Tier)**
   - 상대강도(Relative Strength) 순위 상위 종목만이 투자 대상이다.
   - RS 순위 70 이상, 이상적으로는 90 이상인 종목에 집중한다.
   - "강한 종목은 더 강해지고, 약한 종목은 더 약해진다" — 모멘텀의 지속성을 신뢰한다.

4. **시장 전체보다 개별 종목이 중요하다 (Bottom-Up Stock Selection)**
   - 시장 전체 방향 예측보다 개별 종목의 펀더멘탈과 모멘텀에 집중한다.
   - 약세장에서도 강한 개별 종목은 존재하며, 이를 찾아내는 것이 핵심 역량이다.
   - 다만 시장 환경이 극도로 악화되면 현금 비중을 확대한다.

5. **작은 손실, 큰 수익 (Small Losses, Big Gains)**
   - 손실을 작게 유지하고, 수익이 나는 포지션은 최대한 오래 보유한다.
   - 승률이 50% 미만이라도 평균 수익이 평균 손실을 크게 초과하면 장기적으로 큰 부를 축적할 수 있다.
   - "내 승률은 약 55-60%이지만, 수익 거래의 평균 이익이 손실 거래의 3배 이상이다."

## 대표 전략 유형

### Growth Momentum (성장 모멘텀) 전략

Driehaus의 전략은 펀더멘탈 성장과 기술적 모멘텀을 결합한 하이브리드 접근법이다:

| 구성 요소 | 역할 |
|-----------|------|
| **Earnings Acceleration** | 분기별 EPS 성장률이 가속되는 종목 필터링 |
| **Revenue Growth** | 매출 성장이 동반되는지 확인 (실적의 질 검증) |
| **Earnings Surprise** | 컨센서스 대비 초과 실적 발표 종목 포착 |
| **Relative Strength** | RS 순위 상위 종목으로 유니버스 제한 |
| **Price/Volume Action** | 신고가 돌파, 거래량 증가 등 기술적 확인 |

### 핵심 스크리닝 기준

1. 최근 분기 EPS 성장률 > 전분기 EPS 성장률 (가속 확인)
2. 매출 성장률 > 15% (실적의 질)
3. 어닝 서프라이즈 > 5% (컨센서스 초과)
4. RS 순위 > 70 (상대강도 상위)
5. 주가가 50일 이동평균 위에 위치 (상승 추세)
6. 소형주 또는 중형주 선호 (시가총액 $500M ~ $5B)

## 매매 기법

### 진입 (Entry)

- **어닝 서프라이즈 매수**: 실적 발표 후 컨센서스를 크게 초과한 종목을 즉시 매수. 시장의 과소반응을 활용한다.
- **신고가 돌파 매수**: 의미 있는 베이스 패턴을 형성한 후 신고가를 돌파하는 종목을 매수한다.
- **EPS 가속 확인 후 진입**: 2분기 이상 연속으로 EPS 성장률이 가속되는 시점에 진입한다.
- **분할 진입**: 초기 포지션을 설정한 후, 실적 확인이 추가로 이루어질 때 피라미딩한다.

### 청산 (Exit)

- **모멘텀 둔화 청산**: EPS 성장률이 감속하거나 어닝 서프라이즈가 사라지면 청산 검토.
- **상대강도 하락 청산**: RS 순위가 70 아래로 하락하면 포지션 축소 또는 청산.
- **기술적 이탈 청산**: 50일 이동평균을 하향 이탈하면 청산 고려. 200일 이동평균 이탈은 즉시 청산.
- **Climax Run 인식**: 급등 + 갭업 + 거래량 폭증이 동시에 나타나면 수익 실현.

### 손절 (Stop-Loss)

- **초기 손절선**: 매수가 대비 7-10% 하락 시 무조건 손절.
- **이동평균 기반 손절**: 주요 이동평균(20일, 50일) 아래로 마감 시 손절 실행.
- **시간 기반 손절**: 진입 후 3-4주 내 의미 있는 상승이 없으면 재평가.
- **뉴스 기반 손절**: 실적 하향 조정, 가이던스 하향 등 펀더멘탈 악화 시 즉시 청산.

### 포지션 사이징 (Position Sizing)

- **핵심 포지션**: 확신도 높은 종목에 포트폴리오의 5-10% 배분.
- **보유 종목 수**: 통상 20-30개 (소형주 분산). 대형 펀드 운용 시 더 많은 종목 보유.
- **리스크 기반 사이징**: 각 포지션의 달러 리스크를 총 자본의 0.5-1%로 제한.
- **확신도 조절**: 어닝 서프라이즈 + RS 상위 + 기술적 돌파가 모두 일치하면 사이즈 확대.

### 리스크 관리 (Risk Management)

- **포트폴리오 드로다운 한도**: 계좌 15% 드로다운 시 전반적 노출 축소.
- **섹터 집중 제한**: 단일 섹터 비중 30% 이내 유지 (IT 버블기 예외적 집중 후 교훈).
- **시장 환경 적응**: 약세장에서는 현금 비중 40-60%까지 확대 가능.
- **상관관계 관리**: 유사한 모멘텀 특성을 가진 종목 간 상관관계를 모니터링하여 집중 리스크 완화.

## 심리와 의사결정

### 핵심 멘탈 모델

- **역발상적 모멘텀**: "대부분의 투자자가 싸게 사려고 할 때, 나는 비싸게 산다. 이것이 역발상이다."
- **확률적 프레임**: 개별 거래의 승패에 집착하지 않는다. 장기 기대값이 양(+)이면 시스템을 신뢰한다.
- **유연성**: 시장이 변하면 전략도 변해야 한다. 고집은 트레이딩의 적이다.
- **독립적 사고**: 월가 컨센서스와 반대 포지션을 취하는 것을 두려워하지 않는다.

### Driehaus의 심리 규칙

1. 손실 포지션에 대한 애착을 버려라 — 주식은 당신에게 빚진 것이 없다
2. "나는 틀릴 수 있다"를 매일 아침 되뇌어라
3. 성공한 거래에서 배우는 것과 실패한 거래에서 배우는 것은 동일하게 중요하다
4. 시장의 메시지를 듣되, 시장의 소음은 무시하라
5. 자신만의 시스템에 대한 확신이 없으면 큰 돈을 벌 수 없다

## 대표 사례

### 성공 사례: 1980~1990년대 소형 성장주 랠리

- **배경**: Driehaus는 1980~1990년대 미국 소형 성장주의 대세 상승기에 연평균 30%+ 수익률을 기록하며 "성장주 투자의 아버지"라는 별명을 얻었다.
- **전략**: 기술/헬스케어/소비재 섹터의 소형 성장주에서 EPS 가속 + 어닝 서프라이즈 + RS 상위 조합을 체계적으로 적용.
- **핵심 요인**:
  - 소형주의 정보 비효율성(analyst coverage 부족)을 활용하여 어닝 서프라이즈의 과소반응을 포착
  - 강세장에서 모멘텀 전략의 순풍을 최대한 활용
  - 엄격한 손절로 간헐적 하락장에서의 드로다운 최소화
- **교훈**: 모멘텀 전략은 강세장에서 시장 대비 압도적 초과수익을 만들어낸다.

### 성공 사례: Post-Earnings Drift 활용

- **배경**: Driehaus는 어닝 서프라이즈 이후 주가의 추가 상승(Post-Earnings Announcement Drift, PEAD) 현상을 일찍이 인식하고 체계적으로 활용했다.
- **전략**: 실적 발표 직후 큰 서프라이즈를 기록한 종목을 즉시 매수하여 2-3개월 보유.
- **핵심 요인**: 학술 연구로도 확인된 PEAD 현상을 실전에서 체계적으로 구현한 초기 사례.

### 실패 사례: IT 버블 붕괴 (2000-2001)

- **배경**: 2000년 IT 버블 붕괴 시 성장주 중심 포트폴리오가 큰 타격을 받았다.
- **원인**:
  - 기술주에 대한 과도한 섹터 집중
  - 모멘텀 전략의 구조적 약점 — 급반전 시 대응 지연
  - 밸류에이션 무시에 따른 버블 종목 포함
- **대응**: 손절 규칙에 따라 하락 종목을 순차적으로 정리. 회복 후 전략을 보완하여 섹터 집중 한도를 도입.
- **교훈**: 모멘텀 전략은 시장 급반전에 취약하며, 섹터 분산과 시장 환경 인식이 필수적이다.

## 내 시스템과의 연결

### SEPA 공통점

Driehaus의 Growth Momentum 접근법은 SEPA/OmX 시스템의 핵심 구성 요소와 깊이 연결된다:

| Driehaus 원칙 | OmX/SEPA 대응 |
|----------------|----------------|
| Earnings Surprise 기반 매수 | **Gamma 에이전트** — EPS 서프라이즈 감지, 컨센서스 초과율 계산 |
| Relative Strength 상위 필터 | **Alpha 에이전트** — RS 순위 자동 계산, 상위 필터링 |
| 신고가 돌파 + 거래량 | **Beta 에이전트** — 돌파 패턴 인식, 거래량 분석 |
| Buy High Sell Higher | **SEPA Stage 2 진입** — 이미 상승 추세에 있는 종목만 매수 |
| 소형주 초과수익 | **유니버스 필터** — 시가총액 기반 소형/중형주 스크리닝 |

### SEPA 차이점

| 항목 | Driehaus | OmX 시스템 |
|------|----------|-----------|
| 유니버스 | 소형 성장주 특화 | 전체 시가총액 커버 |
| 진입 기준 | 어닝 서프라이즈 중심 | VCP/Trend Template 중심 |
| 보유 종목 수 | 20-30개 (분산) | 4-8개 (집중) |
| 보유 기간 | 수주 ~ 수개월 | 수주 ~ 수개월 (유사) |
| 밸류에이션 | 거의 무시 | 보조 지표로 활용 |
| 패턴 분석 | 단순 돌파 위주 | VCP 등 정밀 패턴 |

### 전략군 (Strategy Family)

- **Family B (Growth Momentum)**: Driehaus의 핵심 전략군. EPS 가속 + 모멘텀 + RS의 조합이 Family B의 원형(archetype) 중 하나.

### 에이전트 매핑

| 에이전트 | Driehaus 요소 | 구현 내용 |
|----------|---------------|----------|
| **Alpha** | RS 순위 필터 | RS 순위 70/90 이상 자동 필터링, Stage 2 확인 |
| **Gamma** | Earnings Surprise | 실적 발표 후 서프라이즈율 자동 계산, PEAD 시그널 생성 |
| **Delta** | Risk Management | 7-10% 손절 자동 설정, 포지션 사이징 |
| **Omega** | 종합 판단 | Alpha + Gamma + Delta 통합 스코어링 |

### 구현 아이디어

1. **어닝 서프라이즈 알림 시스템**: 실적 발표 직후 컨센서스 초과율 자동 계산, 임계값 초과 시 즉시 알림. Gamma 에이전트에 PEAD 트리거 로직 추가.
2. **EPS 가속 대시보드**: 최근 4분기 EPS 성장률을 시각화하여 가속/감속 패턴을 한눈에 파악.
3. **RS 순위 기반 워치리스트**: RS 90 이상 종목 중 어닝 서프라이즈가 최근 발생한 종목을 자동 워치리스트에 추가.
4. **모멘텀 크래시 감지**: 시장 급반전 시 모멘텀 전략의 구조적 취약점을 보완하기 위한 Market Regime 감지 기능 강화.
5. **Post-Earnings Drift 백테스트**: 과거 어닝 서프라이즈 데이터를 활용하여 PEAD 전략의 한국 시장 유효성 검증.
