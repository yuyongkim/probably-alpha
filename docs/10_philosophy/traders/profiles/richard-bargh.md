## 한 줄 요약

> **"최고의 트레이딩 엣지는 시장이 아니라 자기 자신에게서 나온다 — 자신의 심리를 이해하고 통제하는 것이 지속 가능한 수익의 기반이다."**

## 핵심 투자 철학 (5 Principles)

1. **프로세스가 결과보다 중요하다 (Process Over Outcome)**
   - 개별 거래의 손익이 아니라, 프로세스의 일관된 실행이 장기 성과를 결정한다.
   - 좋은 프로세스에서 나온 손실은 받아들이고, 나쁜 프로세스에서 나온 이익은 경계한다.
   - 매 거래를 프로세스 준수 여부로 평가하는 습관이 핵심이다.

2. **자기 인식이 최고의 엣지다 (Self-Awareness as Edge)**
   - 시장 분석 능력보다 자신의 감정, 편향, 패턴을 인식하는 능력이 더 중요하다.
   - 두려움, 탐욕, FOMO 등 감정이 의사결정을 왜곡하는 순간을 포착해야 한다.
   - 자기 인식은 저널링과 반복적 자기 관찰을 통해 발전한다.

3. **트레이딩 저널은 필수 도구다 (Journaling as Essential Tool)**
   - 모든 거래를 기록하되, 숫자뿐 아니라 진입 시 감정, 사고 과정, 시장 컨텍스트를 함께 기록한다.
   - 저널을 정기적으로 리뷰하여 반복되는 실수 패턴과 성공 패턴을 추출한다.
   - 저널링은 단순 기록이 아니라 자기 대화(self-dialogue)의 도구다.

4. **마인드셋이 기법을 앞선다 (Mindset Before Technique)**
   - 아무리 좋은 전략도 올바른 마인드셋 없이는 실행할 수 없다.
   - 불확실성을 수용하고, 손실을 비용으로 받아들이는 심리적 프레임이 선행해야 한다.
   - 마인드셋 훈련은 일회성이 아니라 매일의 루틴이다.

5. **일관성이 천재성을 이긴다 (Consistency Beats Brilliance)**
   - 뛰어난 한 번의 거래보다 평범하지만 일관된 수백 번의 거래가 성과를 만든다.
   - 일관성은 규칙을 만들고, 규칙을 따르고, 규칙을 개선하는 반복적 사이클에서 나온다.
   - 일관성의 적은 감정적 의사결정이다.

## 대표 전략 유형

### 심리 기반 프로세스 트레이딩 (Psychology-Driven Process Trading)

Bargh의 접근은 특정 기법보다 트레이딩 프로세스 자체의 완성도를 추구한다:

| 구성 요소 | 역할 |
|-----------|------|
| **사전 준비 루틴** | 매 세션 전 멘탈 체크리스트 실행, 감정 상태 점검 |
| **진입 체크리스트** | 감정이 아닌 사전 정의된 기준에 의한 진입 확인 |
| **실시간 감정 모니터링** | 거래 중 감정 변화를 인지하고 기록 |
| **거래 후 리뷰** | 모든 거래를 프로세스 준수 관점에서 복기 |
| **주간/월간 패턴 분석** | 저널 데이터에서 심리적/행동적 패턴 추출 |

### 저널링 프레임워크

1. 거래 전: 시장 상황 인식, 감정 상태, 확신도 기록
2. 거래 중: 의사결정 근거, 감정 변화, 프로세스 이탈 여부 기록
3. 거래 후: 결과와 무관하게 프로세스 준수도 평가
4. 주간 리뷰: 반복 패턴 식별, 개선점 도출
5. 월간 리뷰: 전체적 성과와 심리적 상태의 상관관계 분석

## 매매 기법

### 진입 (Entry)

- **체크리스트 기반 진입**: 사전 정의된 조건이 모두 충족될 때만 진입, 감정적 충동 진입 금지
- **확신도 평가**: 진입 전 1-10 스케일로 확신도를 기록, 확신도와 실제 결과의 상관관계를 추적
- **시장 컨텍스트 확인**: 개별 종목 신호뿐 아니라 시장 전체 환경을 고려
- **감정 필터**: 과도한 흥분이나 두려움 상태에서는 진입하지 않음

### 청산 (Exit)

- **사전 정의된 청산 기준**: 진입 시점에 이미 청산 조건을 설정
- **프로세스 기반 청산**: 목표가 도달, 추세 이탈, 시간 제한 등 객관적 기준
- **감정적 청산 방지**: "불안해서" 또는 "더 오를 것 같아서" 같은 감정적 청산을 경계
- **부분 청산**: 심리적 부담을 줄이기 위한 단계적 이익 실현

### 손절 (Stop-Loss)

- **사전 설정 필수**: 진입과 동시에 손절 수준 확정, 사후 변경 금지
- **손절은 비용이다**: 손실이 아닌 사업 비용으로 인식하는 프레임 전환
- **손절 후 행동 규칙**: 즉각 재진입 금지, 냉각 기간(cooling period) 부여
- **연속 손절 대응**: 3회 연속 손절 시 거래 중단 후 저널 리뷰

### 포지션 사이징 (Position Sizing)

- **확신도 연동**: 확신도에 비례하여 포지션 크기 조절 (단, 상한 존재)
- **최대 단일 리스크**: 총 자본의 1-2% 이내
- **연속 손실 시 축소**: 드로우다운 확대 시 자동으로 포지션 크기 축소
- **심리적 편안함 기준**: 포지션이 수면에 영향을 주면 너무 큰 것

### 리스크 관리 (Risk Management)

- **일일 손실 한도**: 일일 최대 손실 도달 시 거래 중단
- **주간 손실 한도**: 주간 누적 손실 한도 초과 시 주 중반이라도 거래 중단
- **멘탈 리스크 관리**: 심리적 컨디션이 나쁠 때는 거래하지 않음
- **프로세스 이탈 경보**: 프로세스를 벗어난 거래가 감지되면 즉시 자기 점검

## 심리와 의사결정

### 핵심 멘탈 모델

- **관찰자 시점**: 자신의 감정과 행동을 제3자의 시점에서 관찰하는 능력 훈련
- **불확실성 수용**: 모든 거래의 결과는 확률적이라는 사실을 완전히 내재화
- **현재 순간 집중**: 과거 손실이나 미래 불안이 아닌, 현재의 프로세스 실행에 집중
- **성장 마인드셋**: 실수를 학습 기회로, 손실을 수업료로 재정의

### Bargh의 심리 규칙

1. 거래하기 전에 자신의 감정 상태를 반드시 점검하라
2. 프로세스를 따랐다면 결과에 관계없이 좋은 거래다
3. 저널을 쓰지 않은 거래는 존재하지 않는 것과 같다
4. 연속 손실 시 멈추고 리뷰하라 — 시장이 아니라 자신을 점검하라
5. 완벽한 거래는 없다 — "충분히 좋은" 거래를 일관되게 실행하라

## 대표 사례

### 성공 사례: 저널링을 통한 패턴 발견

- **배경**: 특정 시간대에 반복적으로 충동 진입하여 손실을 기록하는 패턴을 저널 리뷰 중 발견
- **전략**: 해당 시간대에 대한 자기 관찰을 강화하고, 추가 확인 절차를 도입
- **핵심 요인**:
  - 3개월간의 상세한 저널 데이터가 패턴 식별을 가능하게 함
  - 감정 기록이 단순 숫자 기록으로는 보이지 않던 패턴을 드러냄
  - 행동 수정 후 해당 시간대 손실이 대폭 감소
- **교훈**: 자기 인식은 데이터에서 나온다 — 기록하지 않으면 인식할 수 없다

### 실패/교훈 사례: 프로세스 이탈의 대가

- **배경**: 높은 확신으로 프로세스를 무시하고 과도한 포지션으로 진입
- **원인**: 연승 후 과신(overconfidence), 프로세스보다 직관을 우선시
- **대응**: 대규모 손실 후 거래 중단, 저널 리뷰를 통한 심리적 원인 분석
- **교훈**: "프로세스는 좋을 때나 나쁠 때나 동일하게 적용되어야 한다. 승률이 높을 때 프로세스를 무시하는 것이 가장 위험한 순간이다."

### 전략군 (Strategy Family)

- **Psychology**: Bargh의 핵심 전략군. 트레이딩 심리, 프로세스 관리, 자기 인식 분야
- 모든 전략군의 기반이 되는 메타 전략으로, 시스템 운용자의 심리적 안정성을 지원

### 구현 아이디어

1. **자동 거래 저널**: 모든 거래의 진입/청산 시 시장 상태, 신호 강도, 기술적 컨텍스트를 자동 기록하는 로깅 시스템
2. **프로세스 준수도 대시보드**: 각 거래가 사전 정의된 프로세스를 얼마나 준수했는지 시각화
3. **드로우다운 자동 대응**: 연속 손실 또는 드로우다운 확대 시 자동으로 포지션 축소/거래 중단
4. **거래 패턴 분석**: 시간대별, 요일별, 시장 환경별 거래 성과 패턴 자동 분석
5. **신호 확신도 캘리브레이션**: 과거 데이터를 통해 신호 확신도와 실제 성과의 상관관계를 지속 보정
