## 한 줄 요약

> **"경제라는 기계의 작동 원리를 이해하고, 원칙(Principles)에 기반한 시스템적 의사결정으로 모든 환경에서 생존한다."**

## 핵심 투자 철학 (5 Principles)

1. **경제는 기계다 (The Economy Is a Machine)**
   - 경제는 예측 가능한 패턴으로 작동하는 기계이다. 신용 사이클, 부채 사이클이 핵심 동인.
   - "경제를 이해하는 것은 기계의 부품이 어떻게 맞물리는지 이해하는 것과 같다."

2. **원칙 기반 의사결정 (Principle-Based Decision Making)**
   - 모든 상황을 유형화하고, 각 유형에 대한 원칙(규칙)을 수립한다.
   - 감정이 아닌 원칙에 따라 의사결정하면 일관성과 품질이 향상된다.

3. **급진적 투명성 (Radical Transparency)**
   - 조직 내 모든 의견은 공개적으로 도전받아야 한다. "아이디어 능력주의(Idea Meritocracy)."
   - 틀림을 빨리 발견하기 위해 반대 의견을 적극 구한다.

4. **분산이 성배다 (Diversification Is the Holy Grail)**
   - 서로 상관관계가 낮은 15~20개의 수익원을 결합하면 리스크 대비 수익이 극대화된다.
   - "투자의 성배는 상관관계 없는 좋은 수익 흐름들을 찾는 것이다."

5. **스트레스 테스트와 시뮬레이션 (Stress Test Everything)**
   - 모든 포트폴리오는 과거 100년 이상의 데이터로 백테스트하고, 극단적 시나리오에서의 성과를 확인한다.
   - "과거에 일어나지 않았다고 미래에 일어나지 않는다는 보장은 없다."

## 대표 전략 유형

### Systematic Global Macro + Risk Parity

Dalio의 Bridgewater는 두 가지 핵심 전략을 운용한다:

| 전략 | 역할 |
|------|------|
| **Pure Alpha** | 시장 방향에 상관없이 절대 수익 추구. 매크로 트렌드 베팅 |
| **All Weather** | 어떤 경제 환경에서도 안정적 수익. 리스크 패리티 기반 자산 배분 |
| **경제 기계 모델** | 성장/인플레이션 2축 × 상승/하락으로 4가지 환경 분류 |
| **시스템화** | 모든 투자 결정을 알고리즘으로 코드화 |
| **부채 사이클 분석** | 단기(5-8년) 및 장기(75-100년) 부채 사이클 추적 |

## 매매 기법

### 진입 (Entry)

- **시스템 신호**: 알고리즘 기반 매크로 모델이 생성하는 신호에 따라 진입
- **경제 지표 선행**: GDP, 고용, 인플레이션, 신용 데이터의 변화 방향 확인
- **상관관계 분석**: 신규 포지션이 기존 포트폴리오와의 상관관계를 높이지 않는지 확인
- **점진적 구축**: 대규모 포지션을 한 번에 구축하지 않고 단계적으로 진입

### 청산 (Exit)

- **환경 변화**: 4분면 모델에서 환경이 전환되면 해당 자산군 비중 자동 조정
- **리밸런싱**: 정기적(월/분기) 리밸런싱으로 목표 리스크 배분 유지
- **손실 한도**: 각 전략별 사전 설정된 드로다운 한도 초과 시 자동 축소
- **시스템 오버라이드**: 극단적 시장 이벤트 시 시스템 외적 판단으로 긴급 청산 가능

### 손절 (Stop-Loss)

- **포트폴리오 레벨**: 개별 포지션이 아닌 포트폴리오 전체의 리스크를 관리
- **변동성 기반**: 각 자산의 변동성에 반비례하여 비중 조절 (리스크 패리티)
- **상관관계 급등**: 위기 시 자산 간 상관관계가 급등하면 전체 노출 축소
- **최대 드로다운**: Pure Alpha 전략의 목표 최대 드로다운 12-18%

### 포지션 사이징 (Position Sizing)

- **리스크 패리티**: 각 자산군이 포트폴리오 리스크에 동일한 비중으로 기여하도록 배분
- **변동성 역비례**: 변동성이 높은 자산은 작은 비중, 낮은 자산은 큰 비중
- **레버리지 활용**: 채권 등 저변동성 자산은 레버리지로 리스크 기여도를 주식과 동일하게
- **동적 조정**: 시장 변동성 변화에 따라 실시간 포지션 사이즈 재조정

### 리스크 관리 (Risk Management)

- **성배 전략**: 15~20개 비상관 수익원 결합으로 샤프비율 극대화
- **테일 리스크 헤지**: 극단적 시나리오(전쟁, 팬데믹 등)에 대비한 헤지 포지션 상시 유지
- **시나리오 분석**: 모든 포지션에 대해 "만약 내가 틀렸다면?" 시나리오를 사전 작성
- **팀 기반 의사결정**: 개인 편향을 줄이기 위해 팀의 신뢰도 가중 투표(believability-weighted voting)

## 심리와 의사결정

### 핵심 멘탈 모델

- **급진적 개방성**: 자신이 틀릴 수 있음을 항상 인정. "무엇을 모르는지 아는 것"이 핵심
- **고통 + 반성 = 진보**: 실패와 고통은 학습의 원천. 이를 회피하지 않고 체계적으로 반성
- **신뢰도 가중**: 모든 의견이 동등하지 않다. 해당 분야의 트랙레코드가 있는 사람의 의견에 더 큰 가중치
- **2차/3차 효과 사고**: 1차적 결과뿐 아니라 그로 인한 연쇄 효과까지 고려

### Dalio의 심리 규칙

1. 자신이 틀렸다는 증거를 적극적으로 찾아라
2. 가장 큰 실수에서 가장 큰 교훈이 나온다 — 실수 일지를 작성하라
3. 의사결정의 질은 과정으로 판단하라. 결과는 확률의 산물이다
4. 합의(consensus)는 편안하지만 위험하다. 독립적 사고를 유지하라
5. 원칙을 코드화하라. 머릿속에만 있는 규칙은 일관되게 실행되지 않는다

## 대표 사례

### 성공 사례: 2008년 금융위기 대응

- **배경**: Bridgewater는 서브프라임 위기를 사전에 예측하고 포트폴리오를 재구성
- **분석**: 부채 사이클 모델이 미국의 과도한 레버리지와 신용 버블을 포착. "아름다운 디레버리징"과 "추악한 디레버리징"의 프레임워크로 분석
- **결과**: 2008년 Pure Alpha 전략 약 +9.5% 수익 (S&P 500은 -37%)
- **교훈**: 시스템적 분석과 과거 부채 위기 연구가 미래 위기에 대한 면역력을 제공

### 성공 사례: All Weather 포트폴리오

- **배경**: 1996년 개인 신탁 자산 관리를 위해 개발, 이후 세계 최대 리스크 패리티 펀드로 성장
- **성과**: 수십 년간 주식 대비 1/3의 변동성으로 유사한 수익률 달성
- **교훈**: 리스크 배분의 균형이 장기 복리 수익의 핵심. 수익률 극대화보다 리스크 관리가 우선

### 실패 사례: 1982년 멕시코 디폴트 예측 오류

- **배경**: Dalio는 1982년 멕시코 부채 위기가 미국 경제를 침체로 이끌 것이라 확신, TV에서 공개 예측
- **결과**: 시장은 반대로 급등. Bridgewater는 거의 파산 위기, 직원을 모두 해고하고 혼자 남음
- **교훈**: "이 경험이 나를 겸손하게 만들었다. 그 이후로 나는 항상 '내가 틀릴 수 있다'고 생각한다." 이것이 Principles 철학의 기원

## 내 시스템과의 연결

### SEPA 공통점

Dalio의 시스템적 접근과 매크로 분석은 OmX의 Gamma 에이전트 설계에 핵심 영감을 제공한다:

| Dalio 원칙 | OmX 구현 |
|------------|----------|
| 경제 기계 모델 | **Gamma 에이전트** — 성장/인플레이션 환경 자동 분류 |
| 원칙의 코드화 | **전체 시스템** — 모든 투자 원칙을 알고리즘으로 구현 |
| 스트레스 테스트 | **백테스트 엔진** — 과거 데이터로 전략 검증 |
| 분산 효과 분석 | **Delta 에이전트** — 포트폴리오 상관관계 모니터링 |
| 시나리오 분석 | **Gamma 에이전트** — "만약 ~라면" 시나리오 자동 생성 |

### SEPA 차이점

| 항목 | Dalio 방식 | OmX 시스템 |
|------|-----------|-----------|
| 자산군 | 모든 자산군 (주식, 채권, 통화, 원자재) | 한국/미국 주식 집중 |
| 접근 | 리스크 패리티 (균등 리스크 배분) | 모멘텀/추세 기반 집중 |
| 타임프레임 | 중장기 매크로 사이클 (수개월~수년) | Stage 2 포지션 (수주~수개월) |
| 종목 선택 | 자산군/국가 단위 배분 | 개별 종목 스크리닝 |
| 레버리지 | 리스크 패리티를 위한 구조적 레버리지 | 무레버리지 현물 기반 |

### 전략군 (Strategy Family)

- **Family F (Macro/Fundamental)**: Dalio의 핵심 전략군. 거시경제 환경 분석 기반 자산 배분

### 에이전트 매핑

| 에이전트 | Dalio 요소 | 구현 내용 |
|----------|-----------|----------|
| **Gamma** | 경제 기계 모델 | 성장률, 인플레이션, 금리, 유동성 지표 통합 분석 |
| **Gamma** | 4분면 환경 분류 | 현재 경제 환경을 4분면 중 하나로 자동 분류 |
| **Delta** | 리스크 관리 | 포트폴리오 레벨 드로다운 모니터링, 상관관계 추적 |
| **Delta** | 포지션 사이징 | 변동성 기반 동적 사이즈 조절 |
| **Omega** | 통합 판단 | 매크로 환경 + 종목 기술적 분석 통합 |

### 구현 아이디어

1. **경제 4분면 대시보드**: 현재 경제 환경(성장/인플레이션 조합)을 실시간 시각화, 섹터 추천에 반영
2. **원칙 코드화 엔진**: 매매 원칙을 if-then 규칙으로 작성, 일관된 실행 보장
3. **상관관계 히트맵**: 보유 종목 간 상관관계 실시간 모니터링, 분산 효과 시각화
4. **부채 사이클 지표**: FRED 데이터로 한국/미국 부채 사이클 위치 자동 추적
5. **실수 일지 자동화**: 손절 발생 시 자동으로 원인 분류, 패턴 분석하여 시스템 개선에 반영
