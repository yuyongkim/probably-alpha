# Tom Baldwin

## 기본 정보

| 항목 | 내용 |
|------|------|
| 본명 | Tom Baldwin |
| 활동 기간 | 1980년대 ~ 1990년대 |
| 인터뷰 수록 | *Market Wizards* (Jack Schwager, 1989) |
| 주요 실적 | $25,000 시작 → 수억 달러 규모 거래, 국채 선물 피트의 최대 개인 트레이더 |
| 운용 스타일 | Scalping / Tape Reading / Day Trading |
| 핵심 시장 | 미국 국채(T-Bond) 선물 — CBOT 피트 |
| 특이사항 | 분석을 하지 않음. 순수하게 주문 흐름(order flow)만으로 거래. 하루 수천 계약 거래 |

## 한 줄 요약

> **"분석은 필요 없다. 피트에서 주문 흐름을 읽고, 빠르게 실행하고, 틀리면 즉시 빠져나와라 — 거래는 반사신경과 감각의 영역이다."**

## 핵심 투자 철학 (5 Principles)

1. **순수 주문 흐름 (Pure Order Flow)**
   - 차트 분석, 펀더멘탈 분석, 기술적 지표를 일절 사용하지 않는다.
   - 피트에서 매수/매도 주문의 흐름, 규모, 긴급성을 직접 관찰하여 방향을 판단한다.
   - "분석은 하지 않는다. 가격과 흐름이 모든 것을 말해준다."

2. **공격성과 속도 (Aggression and Speed)**
   - 기회를 포착하면 주저 없이 대규모로 진입한다.
   - 피트 거래의 핵심은 밀리초 단위의 의사결정 속도다.
   - 다른 트레이더보다 빠르게 움직이는 것이 곧 엣지다.

3. **즉각적 손절 (Instant Cut)**
   - 포지션이 역행하면 생각할 시간 없이 즉시 청산한다.
   - 손실을 "희망"으로 유지하는 것은 파멸의 지름길이다.
   - 손절에 대한 심리적 저항을 완전히 제거한 상태에서 거래한다.

4. **경험 기반 직관 (Experience-Based Intuition)**
   - 수만 시간의 피트 경험이 만들어낸 무의식적 패턴 인식 능력에 의존한다.
   - 논리적으로 설명할 수 없는 "감각"이 거래의 핵심이다.
   - 이 감각은 가르칠 수 없으며, 오직 경험을 통해서만 습득된다.

5. **일중 리스크 제한 (Intraday Risk Containment)**
   - 오버나이트 포지션을 거의 보유하지 않는다.
   - 하루의 리스크를 하루 안에 해결한다.
   - 장 마감 시 포지션을 거의 모두 청산한다.

## 대표 전략 유형

### Pit-Based Scalping and Flow Trading

Baldwin의 전략은 거래소 피트에서의 물리적 임장 거래에 기반한다:

| 구성 요소 | 역할 |
|-----------|------|
| **주문 흐름 관찰** | 대형 기관 주문의 방향, 규모, 긴급성 파악 |
| **스프레드 캡처** | 매수-매도 호가 사이에서 스프레드 수익 획득 |
| **모멘텀 스캘핑** | 단기 방향성 움직임에 빠르게 올라타기 |
| **역추세 스캘핑** | 과도한 단기 움직임 후 되돌림에 베팅 |
| **포지션 관리** | 초단기 보유, 수초~수분 내 청산 |

### 주문 흐름 읽기의 핵심 요소

1. **규모(Size)**: 대형 주문이 한쪽으로 쏠리면 그 방향으로 움직임 예상
2. **긴급성(Urgency)**: 시장가 주문의 빈도 — 긴급할수록 방향성 강함
3. **참여자 식별**: 피트에서 누가 주문하는지 관찰 (기관 vs 개인, 헤저 vs 투기자)
4. **소진(Exhaustion)**: 한쪽 주문이 소진되는 순간 반전 가능성 포착
5. **분위기(Sentiment)**: 피트의 전반적 분위기, 소리, 에너지 수준

## 매매 기법

### 진입 (Entry)

- **흐름 기반**: 대형 매수/매도 주문 감지 시 같은 방향으로 즉시 진입
- **반전 포착**: 한쪽 방향 주문이 소진된 순간 반대 방향 진입
- **레벨 거래**: 주요 심리적 가격대(라운드 넘버)에서의 주문 집중을 활용
- **규모**: 한 번에 수백~수천 계약을 거래 (피트에서의 존재감 자체가 영향력)

### 청산 (Exit)

- **목표 수익**: 틱(tick) 단위의 소규모 수익 — 대규모 거래량으로 보상
- **즉각 청산**: 방향이 맞으면 수 틱 수익 후 즉시 청산
- **모멘텀 라이딩**: 강한 방향성 움직임 시 보유 시간을 약간 연장하여 더 큰 수익 추구
- **장 마감 청산**: 일중 거래 원칙에 따라 장 마감 전 모든 포지션 정리

### 손절 (Stop-Loss)

- **즉시 손절**: 포지션이 1-2틱 역행하면 즉시 청산. 생각하지 않고 반사적으로 실행
- **시간 손절**: 진입 후 수초 내 예상 방향으로 움직이지 않으면 즉시 청산
- **금액 기준 없음**: 틱 단위로 관리하며, 고정 금액 손절선이 아닌 흐름 기반 즉시 반응
- **원칙**: "나쁜 포지션에 1초도 머무르지 않는다"

### 포지션 사이징 (Position Sizing)

- **대규모 기본**: 수백~수천 계약이 기본 단위
- **확신 비례**: 흐름이 명확할수록 더 큰 규모
- **유동성 제공**: 자신이 시장에 유동성을 제공하는 규모의 거래자
- **일중 한도**: 하루 최대 노출 한도를 설정하되, 구체적 수치는 비공개

### 리스크 관리 (Risk Management)

- **일일 손실 한도**: 하루 최대 손실 한도를 설정, 도달 시 거래 중단
- **오버나이트 금지**: 포지션을 다음 날로 가져가지 않음으로써 갭 리스크 제거
- **집중력 관리**: 피트 거래는 극도의 집중력을 요구. 컨디션이 나쁘면 거래하지 않음
- **시장 환경**: 변동성이 극도로 낮은 날은 거래량을 줄임

## 심리와 의사결정

### 핵심 멘탈 모델

- **반사신경**: 의사결정은 의식적 사고가 아닌 훈련된 반사신경으로 이루어진다.
- **감정 제거**: 공포, 탐욕, 희망을 거래에서 완전히 제거한다. 기계적으로 실행한다.
- **현재 집중**: 과거의 거래(이익이든 손실이든)는 다음 거래와 무관하다. 매 순간이 새로운 시작이다.
- **체력**: 피트 거래는 정신적으로뿐만 아니라 육체적으로도 극한의 스태미나를 요구한다.

### Baldwin의 심리 규칙

1. 생각하는 순간 이미 늦었다 — 실행이 분석보다 먼저다
2. 어제의 수익이나 손실은 오늘과 무관하다
3. 두려울 때 가장 공격적으로 움직여야 한다
4. 피트에서의 자신감은 곧 수익이다 — 위축되면 잡아먹힌다
5. 몸과 마음의 컨디션이 거래 성과를 직접적으로 결정한다

## 대표 사례

### 성공 사례: $25,000에서 시작한 피트 전설

- **배경**: Baldwin은 $25,000의 소액으로 CBOT 국채 선물 피트에서 거래를 시작
- **전략**: 순수 주문 흐름 기반 스캘핑. 하루 수천 계약 거래
- **핵심 요인**:
  - 다른 트레이더보다 빠른 반사신경과 공격성
  - 즉각적 손절로 손실을 최소화하고, 작은 이익을 대규모 거래량으로 축적
  - 피트에서의 물리적 존재감과 심리적 우위 확보
- **결과**: 국채 선물 피트에서 가장 큰 개인 트레이더 중 하나로 성장
- **교훈**: 속도, 공격성, 규율의 조합은 분석 없이도 수익을 만들 수 있다

### 성공 사례: 대규모 국채 거래

- **배경**: 경제 지표 발표 등 변동성 이벤트 시 대규모 주문 흐름 포착
- **전략**: 이벤트 직후 밀려오는 기관 주문의 방향을 순간적으로 판단, 같은 방향으로 대규모 진입
- **결과**: 단일 이벤트에서 수십만~수백만 달러의 수익

### 교훈 사례: 피트 거래의 한계

- **배경**: 전자거래의 확산으로 피트 거래의 엣지가 서서히 사라짐
- **의미**: Baldwin의 엣지는 피트라는 물리적 환경에 의존. 전자거래 시대에는 동일한 엣지를 재현하기 어려움
- **교훈**: 모든 엣지에는 유효 기간이 있다. 시장 구조 변화에 적응하지 못하면 엣지는 소멸한다.

### SEPA 공통점

| Baldwin 원본 | OmX 시스템 대응 |
|-------------|----------------|
| 빠른 실행 | 시스템의 자동화된 신호 → 알림 → 빠른 실행 파이프라인 |
| 즉각적 손절 | **Delta 에이전트** — 자동 손절 로직 |
| 시장 흐름 파악 | **Alpha 에이전트** — 거래량 분석, 매수/매도 압력 감지 |
| 일중 리스크 관리 | **Delta 에이전트** — 일일 손실 한도 관리 |

### SEPA 차이점

| 항목 | Baldwin 원본 | OmX 시스템 |
|------|-------------|-----------|
| 분석 여부 | 분석 없음, 순수 흐름 | 정량적 기술/펀더멘탈 분석 |
| 보유 기간 | 초~분 (스캘핑) | 수일~수주 (스윙/포지션) |
| 시장 | 선물 피트 | 주식 |
| 자동화 | 수동 (인간 반사신경) | 반자동~자동 |
| 지표 사용 | 없음 | Trend Template, VCP 등 다수 |
| 거래 빈도 | 하루 수십~수백 회 | 주간 수 회 |

### 전략군 (Strategy Family)

- **Family Mixed**: Baldwin의 전략은 SEPA의 표준 전략군에 직접 매핑되지 않는 독립적 스타일
- 주문 흐름 기반 초단기 거래는 현재 OmX 시스템의 범위 밖이나, 거래량 분석의 원류로서 참고 가치가 있음

### 에이전트 매핑

| 에이전트 | Baldwin 요소 | 구현 내용 |
|----------|-------------|----------|
| **Agent -** | 직접 매핑 에이전트 없음 | Baldwin의 스타일은 현재 시스템에 직접 구현되지 않음 |
| **Alpha (간접)** | 거래량/흐름 분석 | 거래량 급증, 매수/매도 불균형 감지 기능에 간접 반영 |
| **Delta (간접)** | 즉각적 손절 | 자동 손절의 속도와 규율에서 영감 |

### 구현 아이디어

1. **주문 흐름 지표**: 호가창 데이터(Level 2)를 분석하여 매수/매도 불균형을 수치화
2. **거래량 서프라이즈 알림**: 평균 대비 거래량 급증 시 실시간 알림 (Baldwin의 "큰 주문 감지"를 자동화)
3. **초단기 모멘텀 지표**: 분 단위 가격 모멘텀을 계산하여 단기 방향성 판단 보조
4. **실행 속도 최적화**: 신호 발생 → 주문 실행 사이의 지연(latency)을 최소화하는 인프라
5. **시장 미시구조 대시보드**: 스프레드, 거래량, 호가 깊이 등 시장 미시구조 데이터를 실시간 시각화
