# Scott Ramsey

## 기본 정보

| 항목 | 내용 |

| 본명 | Scott Ramsey |
| 활동 기간 | 1990년대 ~ 현재 |
| 주요 실적 | 장기간 안정적 절대수익 달성, 낮은 드로다운 유지 |

## 한 줄 요약

> **"시장의 구조적 비효율을 찾아내고, 매크로 환경이 그 비효율을 해소하는 방향으로 움직일 때 진입하라."**

## 핵심 투자 철학 (5 Principles)

1. **구조적 분석 우선 (Structural Analysis First)**
   - 시장의 가격 움직임 이면에 있는 구조적 요인(수급, 규제, 포지션 편중)을 먼저 분석한다.
   - 표면적인 가격 패턴보다 시장 참여자들의 포지션 구조가 더 중요하다.

2. **비대칭 기회 포착 (Asymmetric Opportunity)**
   - 손실은 제한되고 이익은 크게 열려있는 구조적 셋업만을 추구한다.
   - "모든 거래에서 리스크 대비 보상이 비대칭적이어야 한다."

3. **인내와 선택적 공격 (Patience and Selective Aggression)**
   - 좋은 셋업이 나올 때까지 기다리며, 기회가 왔을 때 과감하게 베팅한다.
   - 거래 빈도보다 거래 품질이 중요하다.

4. **매크로 환경 인식 (Macro Context Awareness)**
   - 개별 거래는 항상 거시경제적 맥락 안에서 해석해야 한다.
   - 금리, 통화정책, 글로벌 자금 흐름이 시장 구조를 결정한다.

5. **유연한 관점 전환 (Flexible Perspective)**
   - 시장이 자신의 가설과 다르게 움직이면 즉시 재평가한다.
   - 자존심보다 생존이 우선이다. 틀렸을 때 빨리 인정하는 것이 핵심 역량이다.

## 대표 전략 유형

### Discretionary Structural Macro

Scott Ramsey의 전략은 전통적인 매크로 트레이딩과 구조적 분석을 결합한 재량적 접근법이다.

| 구성 요소 | 역할 |
|-----------|------|
| **구조적 비효율 분석** | 시장 참여자 포지션 편중, 규제 왜곡 등 식별 |
| **매크로 방향성** | 금리/통화정책/경기 사이클 분석 |
| **이벤트 트리거** | 구조 해소를 촉발하는 촉매 식별 |
| **비대칭 구조 설계** | 옵션/스프레드를 활용한 리스크 비대칭 확보 |
| **타이밍 인내** | 조건이 갖춰질 때까지 기다림 |

## 매매 기법

### 진입 (Entry)

- **조건**: 구조적 비효율이 확인되고, 매크로 촉매가 임박했을 때
- **타이밍**: 시장 센티먼트가 극단에 도달하고 포지션 편중이 확인된 시점
- **접근법**: 옵션 또는 스프레드를 활용하여 비대칭 진입 구조 설계
- **확인**: 복수의 독립적 신호가 같은 방향을 가리킬 때 진입 확신 강화

### 청산 (Exit)

- **목표 달성**: 구조적 비효율이 해소되었다고 판단될 때 청산
- **촉매 소멸**: 예상한 촉매가 실현되지 않거나 지연될 때 재평가
- **시간 기반**: 일정 기간 내 예상 움직임이 없으면 포지션 재검토
- **부분 청산**: 수익 실현 후 잔여 포지션으로 추가 상승 포착

### 손절 (Stop-Loss)

- **구조 무효화**: 진입 근거가 된 구조적 가설이 무효화되면 즉시 손절
- **최대 손실 한도**: 사전에 정의된 최대 손실 금액 초과 시 무조건 청산
- **시간 손절**: 예상 시나리오가 작동하지 않으면 시간 기반 손절 적용
- **유연한 조정**: 시장 상황에 따라 손절 기준을 사전에 재설정

### 포지션 사이징 (Position Sizing)

- **확신도 기반**: 구조적 확신이 높을수록 포지션 사이즈 확대
- **비대칭 구조 활용**: 옵션 매수 전략으로 최대 손실을 프리미엄으로 제한
- **분산**: 상관관계가 낮은 복수의 구조적 기회에 자본 배분
- **변동성 조정**: 시장 변동성에 따라 명목 노출 조절

### 리스크 관리 (Risk Management)

- **포트폴리오 레벨 관리**: 개별 거래가 아닌 전체 포트폴리오의 리스크를 관리
- **상관관계 모니터링**: 위기 시 상관관계가 1로 수렴하는 것을 항상 경계
- **스트레스 테스트**: 극단적 시나리오에서의 손실을 사전에 산정
- **유동성 확보**: 항상 충분한 현금을 유지하여 기회 포착 여력 확보

## 심리와 의사결정

### 핵심 멘탈 모델

- **구조적 사고**: 가격 자체보다 가격을 만드는 구조에 집중한다.
- **확률적 프레이밍**: 모든 거래는 확률 분포의 한 샘플이다.
- **인내의 가치**: "아무것도 하지 않는 것"도 포지션이다.
- **객관성 유지**: 자신의 포지션이 판단을 왜곡하지 않도록 의식적으로 경계한다.

### Ramsey의 의사결정 규칙

1. 구조적 근거가 명확하지 않으면 거래하지 않는다
2. 리스크/리워드가 비대칭적이지 않으면 진입하지 않는다
3. 시장이 가설과 반대로 움직이면 가설을 의심하라 (시장을 의심하지 마라)
4. 큰 수익 후 자만하지 않는다 — 다음 거래는 처음부터 다시 시작이다
5. 불확실성이 높을 때는 포지션을 줄이는 것이 최선이다

## 대표 사례

### 성공 사례: 매크로 구조 활용 트레이드

- **배경**: 시장 참여자들의 포지션이 한 방향으로 극단적으로 편중된 상황 식별
- **전략**: 반대 방향의 비대칭 포지션(옵션 매수)을 저렴하게 구축
- **결과**: 촉매 이벤트 발생 시 포지션 편중 해소(unwind)로 급격한 가격 변동 → 큰 수익 실현
- **교훈**: 군중이 한 방향으로 몰려있을 때, 반대 방향의 비대칭 기회가 존재한다

### 실패/교훈 사례: 타이밍 리스크

- **배경**: 구조적 분석은 정확했으나 촉매 발생 시기를 잘못 예측
- **원인**: 시간 가치 소멸(time decay)로 옵션 포지션 손실
- **교훈**: "맞는 방향이어도 타이밍이 틀리면 돈을 잃는다. 인내에도 비용이 있다."

## 내 시스템과의 연결

### SEPA 공통점

| Ramsey 요소 | OmX 구현 |
|-------------|----------|
| 구조적 비효율 분석 | **Alpha 에이전트** — 수급 불균형, 포지션 편중 데이터 분석 |
| 비대칭 리스크 설계 | **Delta 에이전트** — R:R 비대칭 구조 자동 계산 |
| 매크로 환경 인식 | **Gamma 에이전트** — 거시경제 레짐 분류 |
| 촉매 모니터링 | **Omega 에이전트** — 이벤트 트리거 통합 평가 |

### SEPA 차이점

| 항목 | Ramsey 원본 | OmX 시스템 |
|------|------------|-----------|
| 분석 방식 | 재량적/직관적 구조 해석 | 정량적 데이터 기반 구조 분석 |
| 자산 범위 | 매크로 전 자산군 | 주식 중심 (확장 가능) |
| 실행 | 완전 재량 | 반자동/알고리즘 보조 |
| 옵션 활용 | 핵심 도구 | 보조적 헤지 도구 |

### 전략군 (Strategy Family)

- **Mixed**: 재량적 매크로와 구조적 분석의 결합. 특정 전략군에 고정되지 않는 유연한 접근

### 에이전트 매핑

| 에이전트 | Ramsey 요소 | 구현 내용 |
|----------|------------|----------|
| **Alpha** | 구조적 비효율 | 수급 데이터, 공매도 비율, 옵션 포지션 편중 분석 |
| **Gamma** | 매크로 환경 | 금리/환율/경기선행지수 기반 레짐 분류 |
| **Delta** | 비대칭 리스크 | 옵션 기반 R:R 구조 자동 계산 |
| **Omega** | 종합 판단 | 구조+매크로+촉매 통합 스코어링 |

### 구현 아이디어

1. **포지션 편중 지표**: 선물/옵션 미결제약정 분석으로 시장 참여자 포지션 편중도 측정
2. **구조적 비효율 스캐너**: 규제 변화, 수급 왜곡 등 구조적 이벤트 모니터링 자동화
3. **비대칭 구조 시뮬레이터**: 옵션 조합별 페이오프 구조를 시뮬레이션하여 최적 비대칭 구조 탐색
4. **매크로 레짐 대시보드**: 현재 거시경제 레짐을 실시간으로 분류하고 과거 유사 레짐의 자산별 성과 표시
5. **촉매 캘린더**: 주요 매크로 이벤트(FOMC, 고용지표 등)와 구조적 해소 가능성을 연계 표시
