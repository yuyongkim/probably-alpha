# Bruce Gelber

## 기본 정보

| 항목 | 내용 |
|------|------|
| 본명 | Bruce Gelber |
| 활동 기간 | 1970년대 ~ 현재 |
| 주요 실적 | 수십 년간의 거래 경험으로 축적된 시장 직관과 안정적 수익 |
| 인터뷰 수록 | *Market Wizards* (Jack Schwager, 1989) — 1권 |
| 운용 스타일 | Discretionary / Experience-Based |
| 분류 | Mixed (재량적 경험 기반) |

## 한 줄 요약

> **"시장에서 오래 살아남는 것 자체가 최고의 엣지다. 경험에서 나오는 직관은 어떤 시스템보다 유연하다."**

## 핵심 투자 철학 (5 Principles)

1. **경험이 엣지를 만든다 (Experience Creates Edge)**
   - 수십 년간의 시장 경험이 축적되면, 패턴을 인식하는 직관이 형성된다.
   - 이 직관은 형식화하기 어렵지만 극히 가치 있는 의사결정 도구다.

2. **시장 적응력 (Market Adaptability)**
   - 시장은 끊임없이 변한다. 과거에 통한 전략이 미래에도 통하리라는 보장은 없다.
   - 변화를 감지하고 적응하는 능력이 핵심이다.

3. **리스크 관리 우선 (Risk Management Above All)**
   - 수익보다 생존이 먼저다. 한 번의 큰 손실이 커리어를 끝낼 수 있다.
   - 포지션 크기와 레버리지를 보수적으로 관리한다.

4. **단순함의 가치 (Value of Simplicity)**
   - 복잡한 시스템보다 단순하고 검증된 원칙이 장기적으로 더 효과적이다.
   - "말할 수 없을 만큼 복잡한 것은 이해하지 못한 것이다."

5. **시장 존중 (Respect the Market)**
   - 시장은 항상 옳다. 시장과 싸우지 마라.
   - 자신의 의견과 시장이 충돌하면, 시장을 따르라.

## 대표 전략 유형

### 재량적 경험 기반 트레이딩

Bruce Gelber의 전략은 수십 년의 시장 경험에서 축적된 직관과 패턴 인식에 기반한다.

| 구성 요소 | 역할 |
|-----------|------|
| **패턴 인식** | 과거 유사 시장 상황과 현재의 비교/대조 |
| **시장 분위기 판독** | 가격 행동, 거래량, 뉴스 반응으로 시장 심리 판독 |
| **리스크 직관** | 경험에서 나오는 위험 감지 능력 |
| **유연한 실행** | 상황에 따라 즉흥적으로 전략 조정 |
| **보수적 사이징** | 생존을 최우선으로 하는 포지션 관리 |

### 핵심 접근법

- **가격 행동 분석**: 차트와 가격 움직임에서 수급의 힘을 읽는다
- **뉴스 반응 분석**: 뉴스 자체보다 시장이 뉴스에 어떻게 반응하는지가 중요
- **경험적 패턴 매칭**: 현재 상황과 과거 유사 상황을 비교하여 방향 예측
- **리스크 우선 판단**: 잠재 수익보다 잠재 손실을 먼저 평가

## 매매 기법

### 진입 (Entry)

- **직관 기반**: 시장의 "느낌"이 특정 방향을 가리킬 때
- **가격 행동**: 핵심 지지/저항 수준에서의 가격 행동 관찰 후 진입
- **뉴스 반응**: 악재에도 하락하지 않거나, 호재에도 상승하지 않는 비정상적 반응 시
- **점진적 진입**: 처음에 작게 진입하고, 시장이 확인해주면 추가 진입

### 청산 (Exit)

- **목표 도달**: 사전에 대략적으로 설정한 목표 영역 도달 시
- **직관적 불안**: "무언가 변했다"는 감각이 있을 때 청산
- **모멘텀 소멸**: 움직임의 힘이 약해지는 것을 감지하면 부분 청산
- **시간 경과**: 예상보다 오래 걸리면 재평가

### 손절 (Stop-Loss)

- **직관적 손절**: 시장이 "틀렸다"는 느낌이 강해지면 즉시 청산
- **기술적 수준**: 핵심 지지/저항 수준 이탈 시 손절
- **금액 기반**: 사전 정의된 최대 허용 손실에 도달하면 무조건 청산
- **작은 손실 원칙**: 작은 손실을 여러 번 감수하되, 큰 손실은 절대 허용하지 않음

### 포지션 사이징 (Position Sizing)

- **보수적 접근**: 항상 감당할 수 있는 수준 이내로 사이즈 제한
- **확신도 반영**: 시장 직관이 강할수록 약간 더 큰 사이즈 (그러나 보수적 한도 내)
- **변동성 고려**: 시장 변동성이 높을수록 사이즈 축소
- **레버리지 자제**: 과도한 레버리지를 경계, 생존을 위해 보수적으로 운용

### 리스크 관리 (Risk Management)

- **생존 최우선**: 모든 의사결정에서 장기 생존을 최우선 목표로 설정
- **드로다운 한도**: 일정 드로다운 도달 시 매매 중단 및 재평가
- **분산**: 단일 포지션/방향에 과도하게 노출하지 않음
- **현금 비중**: 불확실할 때는 현금이 최고의 포지션

## 심리와 의사결정

### 핵심 멘탈 모델

- **겸손**: 시장은 누구보다 크다. 겸손한 태도가 생존의 열쇠다.
- **직관 존중**: 오랜 경험에서 비롯된 직관을 신뢰하되, 맹신하지 않는다.
- **평정심**: 큰 수익에도 흥분하지 않고, 큰 손실에도 절망하지 않는다.
- **지속 학습**: 수십 년의 경험이 있어도 시장은 항상 새로운 것을 가르친다.

### Gelber의 의사결정 규칙

1. 시장이 이해되지 않으면 아무것도 하지 마라
2. 작게 시작하라 — 시장이 맞다면 나중에 추가할 수 있다
3. 큰 손실은 절대 허용하지 마라 — 작은 손실은 사업 비용이다
4. 시장과 싸우지 마라 — 시장은 항상 최종 결정자다
5. 경험을 존중하라 — 수십 년간 살아남은 것이 가장 큰 증거다

## 대표 사례

### 성공 사례: 경험적 직관에 의한 전환점 포착

- **배경**: 시장의 분위기와 가격 행동이 과거 주요 전환점과 유사한 패턴을 보임
- **판단**: 수십 년 경험에서 축적된 직관이 "이것은 전환점이다"라고 신호
- **전략**: 보수적 사이즈로 전환 방향에 진입
- **결과**: 시장이 실제로 전환하며 안정적 수익 실현
- **교훈**: 오랜 경험에서 비롯된 패턴 인식은 시스템으로 코딩하기 어렵지만 실효적이다

### 실패/교훈 사례: 과신의 위험

- **배경**: 오랜 성공이 자만을 낳고, 평소보다 큰 포지션 유지
- **결과**: 예상치 못한 외부 충격으로 큰 손실
- **교훈**: "경험은 자산이지만 자만은 부채다. 시장은 항상 겸손을 요구한다."

### 전략군 (Strategy Family)

- **Mixed**: 특정 전략군에 속하지 않는 유연한 재량적 접근. 모든 전략의 기초가 되는 "시장 경험"의 중요성을 강조

### 에이전트 매핑

| 에이전트 | Gelber 요소 | 구현 내용 |
|----------|------------|----------|
| **Beta** | 가격 행동 분석 | 가격/거래량 패턴의 자동 인식 및 분류 |
| **Delta** | 리스크 관리 | 보수적 손절, 포지션 사이징 자동화 |
| **Omega** | 시장 직관 | 복합 지표 기반 시장 레짐 판단 |
| **메타** | 적응 | 시장 레짐 변화 감지 및 파라미터 자동 조정 |

### 구현 아이디어

1. **뉴스 반응 분석기**: 뉴스 이벤트와 가격 반응의 괴리를 자동 감지 (악재에 하락 안 함 등)
2. **역사적 패턴 매칭**: 현재 시장 상황과 유사한 과거 시점을 자동 탐색하고 후속 경로 비교
3. **직관의 코드화 시도**: Gelber 유형의 재량적 판단 기준을 가능한 범위에서 규칙화
4. **시장 레짐 전환 감지**: 변동성, 상관관계, 거래량 패턴 변화로 시장 레짐 전환 조기 감지
5. **생존 모드 자동화**: 연속 손실/드로다운 시 자동으로 사이즈 축소 및 매매 빈도 감소
