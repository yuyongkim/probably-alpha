# Victor Sperandeo

## 한 줄 요약

> **"시장의 전환점(turning point)을 Dow Theory와 매크로 분석으로 포착하고, 리스크를 수학적으로 관리하라."**

## 핵심 투자 철학 (5 Principles)

1. **전환점 포착 (Catching Turning Points)**
   - 추세의 시작과 끝을 Dow Theory 기반으로 식별한다.
   - 추세의 중간이 아닌 전환 시점에서의 진입이 최고의 리스크/리워드를 제공한다.

2. **Dow Theory의 실전 적용 (Practical Dow Theory)**
   - 고전적 Dow Theory를 현대 시장에 맞게 재해석하여 적용.
   - 추세의 정의: 연속적인 고점 상승/저점 상승(상승 추세), 고점 하락/저점 하락(하락 추세).

3. **확률과 기대값 (Probability and Expected Value)**
   - 모든 거래를 확률과 기대값으로 평가한다. 감정이 아닌 수학으로 결정.
   - "나는 확률과 통계의 언어로 시장을 읽는다."

4. **매크로 우선 (Macro First)**
   - 금리, 세금, 규제 변화가 시장 방향의 근본 동인이다.
   - 정부 정책의 변화가 시장 전환점의 선행 지표.

5. **자본 보호의 수학 (Mathematics of Capital Preservation)**
   - 50% 손실을 회복하려면 100% 수익이 필요하다. 손실의 비대칭성을 항상 인식.
   - "최고의 트레이더는 가장 적게 잃는 트레이더다."

## 대표 전략 유형

### Macro Turning Point Analysis

Sperandeo의 전략은 Dow Theory 기반 추세 전환점 식별 + 매크로 분석이다:

| 구성 요소 | 역할 |
|-----------|------|
| **1-2-3 추세 전환** | Dow Theory 기반 3단계 추세 반전 패턴 확인 |
| **2B 패턴** | 가짜 돌파(false breakout) 포착으로 역추세 진입 |
| **매크로 컨텍스트** | 금리, 재정정책, 인플레이션 방향이 전환을 지지하는지 확인 |
| **확률 계산** | 각 셋업의 역사적 성공률 기반 기대값 계산 |
| **리스크 제한** | 모든 거래에서 사전 정의된 최대 손실 한도 적용 |

### 1-2-3 추세 전환 패턴

```
상승 추세 종료 (1-2-3 Top):
        1 (최고점)
       / \
      /   \
     /     2 (반등 고점 - 1보다 낮음)
    /     / \
   /     /   \
  /     /     3 (2의 저점 하향 이탈 → 매도 신호)
```

## 매매 기법

### 진입 (Entry)

- **1-2-3 패턴**: 추세 전환의 3단계 완성 확인 후 진입 (3번 포인트에서)
- **2B 패턴**: 이전 고점/저점을 살짝 넘었다가 다시 반전하는 가짜 돌파 시 진입
- **매크로 확인**: 기술적 전환 신호가 매크로 환경 변화와 일치하는지 확인
- **초기 소량**: 전환 초기에 소량 진입, 전환 확인 후 추가 매수
- **타이밍 정밀화**: 일봉/주봉 차트의 전환 패턴이 동시에 나타날 때 확신도 상승

### 청산 (Exit)

- **목표 도달**: 이전 지지/저항 수준, 피보나치 되돌림 등으로 목표가 설정
- **반대 전환 신호**: 반대 방향의 1-2-3 패턴 출현 시 청산
- **추세 재확인**: 진입한 추세가 여전히 유효한지 주기적으로 점검
- **시간 기반**: 예상 기간 내 목표에 도달하지 못하면 청산 고려

### 손절 (Stop-Loss)

- **패턴 무효화**: 1-2-3 패턴의 1번 포인트를 다시 넘으면 즉시 손절
- **최대 손실**: 한 거래당 자본의 1-2%를 초과하지 않음
- **기계적 실행**: 손절은 예외 없이 실행. "손절 없는 거래는 도박이다"
- **사전 설정**: 진입 전에 반드시 손절선을 확정. 진입 후 수정 금지

### 포지션 사이징 (Position Sizing)

- **리스크 기반**: 손절까지의 거리 × 수량 = 총 자본의 1-2%
- **확률 가중**: 역사적 성공률이 높은 셋업에 더 큰 사이즈 배분
- **기대값 최적화**: 기대값(승률 × 평균이익 - 패률 × 평균손실) 극대화를 목표
- **최대 노출**: 전체 포트폴리오 리스크가 자본의 5-6%를 초과하지 않도록 관리

### 리스크 관리 (Risk Management)

- **손실의 비대칭성 인식**: -50% 회복에는 +100% 필요. 큰 손실은 절대 불가
- **연간 손실 Zero 목표**: 18년 연속 연간 수익. 이는 보수적 사이징과 빠른 손절의 결과
- **상관관계 관리**: 같은 방향의 포지션이 동시에 실패할 리스크 관리
- **시장 환경 인식**: 추세가 불명확한 횡보장에서는 노출을 최소화

## 심리와 의사결정

### 핵심 멘탈 모델

- **확률적 사고**: 모든 거래는 확률 게임. 개별 결과에 감정적으로 반응하지 않는다
- **과정 중심**: "올바른 과정은 장기적으로 올바른 결과를 낳는다"
- **자기 인식**: 자신의 편향(확증 편향, 손실 회피 등)을 인식하고 시스템으로 통제
- **지적 정직성**: 틀렸을 때 빠르게 인정. "시장과 논쟁하지 마라"

### Sperandeo의 심리 규칙

1. 거래 전에 최대 손실을 정의하라. 정의되지 않은 리스크는 무한 리스크다
2. 군중과 같은 방향이라면 한 번 더 검증하라
3. 연속 수익은 자만을, 연속 손실은 두려움을 낳는다. 둘 다 사이즈로 관리하라
4. 정치와 정책을 무시하지 마라. 정부가 시장을 움직인다
5. 통계와 역사가 직관보다 신뢰할 수 있다

## 대표 사례

### 성공 사례: 18년 연속 연간 수익

- **배경**: Sperandeo는 18년간 단 한 해도 연간 기준 손실 없이 연평균 72% 수익률 기록
- **전략**: 매크로 전환점 포착 + 엄격한 리스크 관리. 불확실할 때는 노출을 극도로 줄임
- **핵심**: 최대 드로다운을 6% 이하로 유지. 작은 이익을 꾸준히 쌓는 복리 효과
- **교훈**: 화려한 대박이 아닌 일관된 소규모 수익의 복리가 장기적으로 경이적 성과를 만든다

### 성공 사례: 1987년 블랙 먼데이 예측

- **배경**: Sperandeo는 1987년 10월 대폭락 직전 시장 전환점을 포착
- **분석**: Dow Theory 기반 추세 전환 신호 + 금리 상승, 달러 약세 등 매크로 경고 신호 확인
- **결과**: 사전에 방어적 포지션 전환, 대폭락의 피해를 최소화
- **교훈**: Dow Theory의 기술적 신호와 매크로 분석의 결합이 극단적 이벤트에서도 보호막이 된다

### 실패 사례: 과신의 대가

- **배경**: 초기 경력에서 큰 수익 후 자만에 빠져 포지션 사이즈를 과도하게 확대
- **원인**: 연속 성공이 만든 과신. "내가 틀릴 수 없다"는 착각
- **결과**: 예상 밖의 시장 반전에 큰 손실
- **교훈**: "성공은 가장 위험한 순간이다. 성공 후에야 사이즈를 줄여야 한다는 것을 배웠다"

## 내 시스템과의 연결

### SEPA 공통점

Sperandeo의 추세 전환점 분석과 매크로 필터는 OmX의 Gamma 에이전트와 직접 연결된다:

| Sperandeo 원칙 | OmX 구현 |
|----------------|----------|
| 1-2-3 전환 패턴 | **Gamma 에이전트** — 시장 전체의 추세 전환 감지 |
| 2B 패턴 | **Beta 에이전트** — 개별 종목의 가짜 돌파 감지 |
| 매크로 전환점 | **Gamma 에이전트** — 금리, 정책 변화의 시장 영향 분석 |
| 확률적 사이징 | **Delta 에이전트** — 기대값 기반 포지션 사이징 |
| Dow Theory 추세 | **Alpha 에이전트** — 고점/저점 패턴으로 추세 판단 |

### SEPA 차이점

| 항목 | Sperandeo 방식 | OmX 시스템 |
|------|---------------|-----------|
| 진입 | 추세 전환점에서 역추세 진입 | Stage 2 추세 합류 (순추세) |
| 핵심 도구 | Dow Theory, 1-2-3 패턴 | Trend Template, VCP |
| 타임프레임 | 중기 스윙 (수주~수개월) | Stage 2 포지션 (수주~수개월) |
| 전환 vs 합류 | 추세 전환을 노림 | 확립된 추세에 합류 |
| 매크로 비중 | 매크로가 핵심 (50%+) | 매크로는 보조 필터 (Gamma) |

### 전략군 (Strategy Family)

- **Family F (Macro/Fundamental)**: Sperandeo의 핵심 전략군. 매크로 전환점 분석 기반

### 에이전트 매핑

| 에이전트 | Sperandeo 요소 | 구현 내용 |
|----------|----------------|----------|
| **Gamma** | 매크로 전환점 | 금리 사이클 전환, 정책 변화 자동 감지 |
| **Gamma** | Market Regime | 시장 전체 추세 방향 (상승/하락/횡보) 자동 판정 |
| **Alpha** | Dow Theory 추세 | 고점/저점 패턴 기반 추세 자동 판정 |
| **Beta** | 2B 패턴 | 가짜 돌파 감지 알고리즘 |
| **Delta** | 확률 기반 사이징 | 셋업별 역사적 성공률 × 기대값으로 사이즈 결정 |

### 구현 아이디어

1. **시장 전환점 알림**: Dow Theory 기반 1-2-3 패턴을 KOSPI/KOSDAQ 지수에 자동 적용, 전환 신호 발생 시 알림
2. **2B 패턴 스크리너**: 개별 종목에서 가짜 돌파(false breakout)를 감지하여 함정 매수 방지
3. **정책 변화 모니터링**: 한국은행 금리 결정, 정부 정책 발표를 자동 추적, 시장 영향 분석
4. **기대값 대시보드**: 각 셋업 유형의 역사적 승률과 평균 R:R을 실시간 표시
5. **드로다운 경보 시스템**: 포트폴리오 드로다운이 사전 설정값 초과 시 자동으로 노출 축소 제안
