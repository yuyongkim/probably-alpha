# Jason Shapiro

## 기본 정보

| 항목 | 내용 |
|------|------|
| 본명 | Jason Shapiro |
| 활동 기간 | 1990년대 ~ 현재 |
| 인터뷰 수록 | *Unknown Market Wizards* (Jack Schwager, 2020) |
| 주요 실적 | 장기간 연평균 30%+ 수익률, 낮은 변동성과 제한된 드로다운 |
| 운용 스타일 | Contrarian Macro / Positioning Analysis |
| 핵심 시장 | 선물 (주가지수, 채권, 통화, 상품 등 글로벌 매크로 자산) |
| 분석 도구 | COT(Commitments of Traders) 보고서 기반 포지셔닝 분석 |
| 특이사항 | 자신을 "가장 큰 바보(the biggest idiot)"라 부르며 겸손한 접근. 군중 반대편에 서는 역발상 전문가 |

## 한 줄 요약

> **"군중의 포지션이 극단에 달했을 때 반대편에 서라 — COT 보고서가 보여주는 포지셔닝의 극단이 곧 반전의 신호다."**

## 핵심 투자 철학 (5 Principles)

1. **역발상 (Contrarian Thinking)**
   - 다수가 한쪽 방향에 몰려 있을 때 반대편에 선다.
   - 군중이 극도로 강세일 때 매도하고, 극도로 약세일 때 매수한다.
   - "모두가 이미 매수했다면, 추가로 매수할 사람은 누가 남았는가?"

2. **포지셔닝 분석 (Positioning Analysis)**
   - COT(Commitments of Traders) 보고서를 핵심 분석 도구로 활용한다.
   - 상업적 헤저, 대형 투기자, 소형 투기자의 포지션 극단을 추적한다.
   - 포지셔닝의 극단은 시장 반전의 선행 지표다.

3. **기술적 확인을 기다려라 (Wait for Technical Confirmation)**
   - 포지셔닝이 극단이라고 해서 즉시 진입하지 않는다.
   - 가격이 실제로 반전의 기술적 신호를 보일 때까지 기다린다.
   - "올바른 방향 + 올바른 타이밍" 둘 다 필요하다.

4. **리스크 한도 엄수 (Strict Risk Limits)**
   - 단일 거래와 전체 포트폴리오 양쪽에 엄격한 리스크 한도를 설정한다.
   - 드로다운이 한도에 도달하면 무조건 포지션을 축소한다.
   - 리스크 관리가 수익보다 항상 우선한다.

5. **겸손과 유연성 (Humility and Flexibility)**
   - 자신이 틀릴 수 있음을 항상 인정한다.
   - 시장이 자신의 분석과 다르게 움직이면 빠르게 인정하고 대응한다.
   - "나는 가장 큰 바보다 — 하지만 바보임을 아는 바보는 살아남는다."

## 대표 전략 유형

### Contrarian Macro Based on COT Positioning

Shapiro의 전략은 COT 데이터 기반의 역발상 매크로 거래다:

| 구성 요소 | 역할 |
|-----------|------|
| **COT 분석** | 상업/대형투기/소형투기 포지션의 극단 수준 파악 |
| **역발상 프레임** | 포지셔닝 극단에서 반대 방향 거래 기회 포착 |
| **기술적 확인** | 가격 반전의 기술적 신호 대기 후 진입 |
| **매크로 맥락** | 글로벌 경제, 금리, 통화 정책의 큰 그림 파악 |
| **리스크 관리** | 엄격한 손절과 포지션 사이징 |

### COT 보고서 분석 프레임워크

| 참여자 유형 | 의미 | Shapiro의 활용법 |
|------------|------|-----------------|
| **Commercials (상업적 헤저)** | 실물 시장 참여자, 정보 우위 | 상업적 헤저의 극단적 포지션은 스마트 머니 신호 |
| **Large Speculators (대형 투기자)** | 추세 추종 펀드, 모멘텀 | 대형 투기자의 극단 포지션은 추세 소진 신호 |
| **Small Speculators (소형 투기자)** | 개인 투자자, 감정적 | 소형 투기자의 극단은 가장 강력한 역발상 신호 |

### 진입 조건 체크리스트

1. COT 포지셔닝이 역사적 극단 수준 (상위/하위 10%)
2. 투기자들이 한쪽으로 극단적으로 쏠려 있음
3. 상업적 헤저가 반대 방향으로 포지션 확대
4. 가격이 기술적 반전 신호를 보이기 시작
5. 매크로 환경이 반전 시나리오를 지지

## 매매 기법

### 진입 (Entry)

- **COT 극단 감지**: 주간 COT 보고서에서 포지셔닝 극단 확인
- **기술적 트리거**: 추세선 이탈, 이동평균 교차, 반전 캔들 패턴 등 기술적 신호 확인
- **점진적 구축**: 초기 소규모 포지션 → 방향 확인 후 추가 → 최대 목표 포지션
- **다자산 분산**: 관련 없는 여러 시장에서 동시에 역발상 기회 탐색

### 청산 (Exit)

- **포지셔닝 정상화**: COT 포지셔닝이 극단에서 중립으로 돌아오면 청산 시작
- **목표 수익**: 추세 반전 후 새로운 추세가 형성되기 시작하면 단계적 청산
- **반대 극단**: 포지셔닝이 반대쪽 극단에 도달하면 전량 청산
- **시간 기반**: 진입 후 합리적 기간 내 예상 방향 전개가 없으면 재평가

### 손절 (Stop-Loss)

- **기술적 손절**: 진입 근거가 되었던 기술적 수준(지지/저항) 이탈 시
- **비율 손절**: 자본의 1-2% 이내로 단일 거래 손실 제한
- **시간 손절**: 일정 기간 내 방향 전개가 없으면 포지션 축소
- **포트폴리오 손절**: 전체 포트폴리오 드로다운이 한도 도달 시 전면 축소

### 포지션 사이징 (Position Sizing)

- **리스크 기반**: 각 거래의 달러 리스크(진입-손절)를 기준으로 포지션 크기 결정
- **확신도 반영**: COT 극단 + 기술적 확인 + 매크로 지지가 모두 정렬 시 더 큰 사이즈
- **변동성 조정**: 시장 변동성이 높을수록 포지션 크기를 줄여 동일한 달러 리스크 유지
- **최대 노출**: 전체 포트폴리오의 방향성 노출 한도 설정

### 리스크 관리 (Risk Management)

- **월간 드로다운 한도**: 월간 최대 드로다운 한도를 설정, 도달 시 신규 진입 중단
- **연간 드로다운 한도**: 연간 목표 최대 드로다운을 초과하지 않도록 관리
- **상관관계 모니터링**: 포지션 간 상관관계를 주시하여 의도치 않은 방향성 집중 방지
- **뉴스 리스크**: 주요 이벤트(FOMC, 고용 등) 전후 포지션 축소 또는 헤지

## 심리와 의사결정

### 핵심 멘탈 모델

- **역발상 본능**: 군중의 열광이나 공포를 보면 반사적으로 반대를 생각한다.
- **확률적 사고**: 모든 거래는 확률 게임이다. 개별 결과보다 기대값과 시행 횟수가 중요하다.
- **겸손**: "나는 가장 큰 바보다"라는 자기 인식이 과신을 방지한다.
- **인내**: 포지셔닝이 극단에 도달하고 기술적 확인이 나올 때까지 기다린다.

### Shapiro의 심리 규칙

1. 모두가 확신할 때가 가장 위험하다 — 확신의 크기는 리스크의 크기다
2. 자신이 틀릴 확률을 항상 50% 이상으로 가정하라
3. 손절은 지혜의 표현이지 실패의 인정이 아니다
4. 군중을 따르는 것은 편하지만 수익을 주지 않는다
5. 겸손한 트레이더가 오래 살아남는 트레이더다

## 대표 사례

### 성공 사례: 포지셔닝 극단에서의 역발상 거래

- **배경**: COT 보고서에서 투기자들의 특정 자산 롱 포지션이 역사적 극단에 도달
- **전략**: 기술적 반전 신호 확인 후 숏 진입. 포지셔닝이 정상화될 때까지 보유
- **핵심 요인**:
  - COT 데이터가 명확한 극단을 보여줌
  - 기술적 확인(추세선 이탈)이 타이밍을 잡아줌
  - 엄격한 손절로 리스크를 제한하면서 추세 반전의 큰 움직임 포착
- **교훈**: 포지셔닝 극단 + 기술적 확인 = 높은 확률의 역발상 진입

### 성공 사례: 낮은 변동성의 일관된 수익

- **배경**: 장기간에 걸쳐 연평균 30%+ 수익률을 제한된 드로다운으로 달성
- **전략**: 다양한 시장에서 COT 기반 역발상 거래를 반복
- **핵심 요인**: 엄격한 리스크 관리로 큰 손실을 방지하면서 역발상의 높은 승률 활용
- **교훈**: 일관성은 큰 수익이 아닌 작은 손실에서 나온다

### 교훈 사례: 조기 역발상의 위험

- **배경**: 포지셔닝이 극단에 접근하고 있었으나 아직 극단에 도달하기 전에 역발상 진입
- **원인**: 기술적 확인을 기다리지 않고 COT 데이터만으로 조기 진입
- **결과**: 추세가 예상보다 더 오래 지속되어 손절
- **교훈**: "시장은 예상보다 더 극단적으로 갈 수 있다. 확인을 기다리는 인내가 필수다."

### 구현 아이디어

1. **COT 데이터 자동 수집/분석**: 주간 COT 보고서를 자동 파싱하여 포지셔닝 극단 감지 알림
2. **센티먼트 극단 지표**: COT 외에도 풋/콜 비율, VIX, 투자자 설문 등 다양한 센티먼트 데이터 통합
3. **역발상 진입 모듈**: 포지셔닝 극단 + 기술적 반전 확인 시 자동 알림/신호 생성
4. **시장 과열/과매도 대시보드**: 주요 자산군별 포지셔닝 극단 수준을 실시간 시각화
5. **추세 추종과 역발상의 통합**: 추세 진행 중에는 SEPA, 추세 극단에서는 Shapiro식 역발상 — 두 접근법의 동적 전환
