## 한 줄 요약

> **"외환 시장의 매크로 방향성을 판단하고, 3:1 이상의 Risk-Reward에서만 진입하며, 확신의 강도에 따라 포지션 크기를 결정한다."**

## 핵심 투자 철학 (5 Principles)

1. **Risk-Reward가 모든 것에 우선한다 (Risk-Reward Trumps Everything)**
   - 최소 3:1의 Risk-Reward 비율이 확보되지 않으면 아무리 좋은 아이디어라도 진입하지 않는다.
   - "좋은 트레이드란 리스크가 제한되고 보상이 열린 트레이드다."

2. **확신의 강도가 사이즈를 결정한다 (Conviction Determines Size)**
   - 모든 매매가 동일하지 않다. 확신이 높은 아이디어에는 더 큰 포지션을 취한다.
   - 확신이 낮으면 작은 사이즈로 시장의 반응을 확인한다.

3. **시장을 24시간 감시하라 (Monitor Markets 24/7)**
   - 외환 시장은 24시간 운영된다. 핵심 가격 수준과 이벤트를 놓치지 않는다.
   - 정보의 우위는 더 많이 아는 것이 아니라, 더 빠르게 해석하는 것에서 나온다.

4. **유동성을 존중하라 (Respect Liquidity)**
   - 포지션 구축과 해소 시 시장 유동성을 반드시 고려한다.
   - 유동성이 부족한 시점에 대규모 포지션을 보유하는 것은 자살 행위다.

5. **손실은 비즈니스 비용이다 (Losses Are a Cost of Business)**
   - 손실을 감정적으로 받아들이지 않는다. 사업의 운영 비용과 같다.
   - 중요한 것은 개별 손실이 아니라 장기적인 손익 비율이다.

## 대표 전략 유형

### Macro FX Directional Trading

Lipschutz의 핵심 전략은 거시경제 분석을 기반으로 한 외환 방향성 매매이다:

| 구성 요소 | 역할 |
|-----------|------|
| **매크로 분석** | 금리 차이, 경상수지, 재정정책, 중앙은행 정책 분석 |
| **기술적 확인** | 지지/저항선, 추세선, 모멘텀 지표로 매크로 뷰 확인 |
| **R:R 계산** | 모든 진입에서 최소 3:1 Risk-Reward 확보 |
| **포지셔닝 분석** | 시장 참여자들의 포지셔닝 편향 파악 |
| **유동성 관리** | 진입/청산 시 유동성 시간대 선택, 슬리피지 최소화 |

## 매매 기법

### 진입 (Entry)

- **매크로 뷰 수립**: 경제 데이터, 중앙은행 정책, 정치적 이벤트를 종합하여 통화 방향 판단
- **기술적 확인**: 매크로 뷰와 일치하는 기술적 신호(지지/저항 돌파, 추세 확인) 대기
- **R:R 검증**: 기술적 수준 기반으로 목표가와 손절가 설정, 3:1 이상 확인
- **점진적 구축**: 대형 포지션은 여러 가격 수준에서 분할 진입
- **시간대 선택**: 유동성이 풍부한 시간(런던/뉴욕 오버랩)에 주요 포지션 진입

### 청산 (Exit)

- **목표가 도달**: R:R 계산 시 설정한 목표가에 도달하면 최소 일부 청산
- **매크로 변화**: 자신의 매크로 뷰를 변화시키는 새로운 데이터/이벤트 발생 시 청산
- **모멘텀 반전**: 가격 모멘텀이 명확하게 반전될 때 포지션 축소
- **분할 청산**: 목표가 근처에서 분할 청산, 일부는 추가 상승/하락 여지를 위해 유지

### 손절 (Stop-Loss)

- **기술적 기반**: 주요 지지/저항선을 기준으로 손절 설정
- **R:R 유지**: 손절 폭이 R:R 3:1을 해치지 않는 수준에서 설정
- **유연한 관리**: 기계적 손절보다 시장 상황에 따른 판단적 손절 선호
- **최대 한도**: 단일 트레이드에서 자본의 2% 이상 손실 절대 회피

### 포지션 사이징 (Position Sizing)

- **확신도 기반**: 확신이 가장 높은 트레이드에 가장 큰 사이즈
- **계층적 사이징**: 최고 확신(풀사이즈) → 중간 확신(절반) → 낮은 확신(탐색적 소규모)
- **포트폴리오 관점**: 동시에 여러 통화 쌍을 거래할 때 상관관계 고려
- **유동성 제약**: 시장 유동성에 비해 과도한 사이즈 절대 금지

### 리스크 관리 (Risk Management)

- **상관관계 관리**: 서로 연관된 통화 쌍의 동시 포지션 노출 관리
- **이벤트 리스크**: 주요 경제 지표 발표, 중앙은행 회의 전후 포지션 조정
- **유동성 리스크**: 유동성이 낮은 시간대(아시아 세션 등)에는 포지션 축소
- **드로다운 관리**: 일정 수준 이상 드로다운 시 전체 포지션 축소 후 재평가
- **레버리지 제한**: FX 시장의 높은 레버리지에 주의, 실질 노출 관리

## 심리와 의사결정

### 핵심 멘탈 모델

- **확률적 사고**: 개별 트레이드는 확률 게임이다. 100회 중 60회 맞으면서 R:R 3:1이면 탁월한 성과가 된다.
- **감정 분리**: 손실에 대한 감정적 반응을 철저히 통제. 손실은 사업 비용일 뿐이다.
- **정보 해석**: 정보 자체보다 정보를 해석하는 능력이 중요하다. 같은 뉴스를 보고 다른 결론을 내릴 수 있어야 한다.
- **시장 존중**: 시장이 자신의 예상과 다르게 움직이면, 시장이 옳다고 인정한다.

### Lipschutz의 심리 규칙

1. 시장에 대한 확고한 뷰가 없으면 매매하지 마라
2. 포지션의 크기가 수면을 방해하면 너무 크다는 신호다
3. 손실 후 즉시 복수 매매를 하지 마라. 냉정하게 재평가하라
4. 뉴스 자체가 아닌, 뉴스에 대한 시장의 반응을 관찰하라
5. 최고의 트레이드는 인내의 결과다. 기회를 만들려 하지 말고 기다려라

## 대표 사례

### 성공 사례: Salomon Brothers FX 부서

- **배경**: 1982년 Salomon Brothers에 입사하여 FX 트레이딩 부서를 사실상 구축
- **성과**: 연간 $300M 이상의 수익을 지속적으로 기록, Salomon FX의 핵심 수익원
- **전략**: 매크로 분석 + 기술적 분석 + R:R 3:1 원칙의 체계적 적용
- **교훈**: 체계적이고 규율 있는 접근이 대규모 기관 환경에서도 일관된 성과를 만든다

### 성공 사례: Hathersage Capital 설립

- **배경**: Salomon 퇴사 후 1995년 Hathersage Capital Management 설립
- **전략**: G10 통화에 집중한 매크로 FX 전략
- **성과**: 장기적으로 안정적 수익 기록, 기관 투자자 자금 유치
- **교훈**: 자신의 전문 영역(FX)에 집중하는 것이 분산보다 효과적

### 실패 사례: 초기 학습 — $250,000 전액 손실

- **배경**: Lipschutz는 대학 시절 상속받은 $12,000를 $250,000까지 불린 후 하룻밤에 전액 손실
- **원인**: 과도한 레버리지, 리스크 관리 부재, 하나의 포지션에 전 재산 집중
- **전환점**: 이 경험이 리스크 관리의 중요성을 각인. R:R 3:1 원칙의 기원
- **교훈**: "돈을 잃는 경험 없이는 진정한 트레이더가 될 수 없다. 중요한 것은 그 경험에서 무엇을 배우느냐다."

## 내 시스템과의 연결

### SEPA 공통점

Lipschutz의 R:R 중심 접근과 확신 기반 사이징은 OmX 시스템의 리스크 관리 모듈과 직접 연결된다:

| Lipschutz 원칙 | OmX 구현 |
|---------------|----------|
| R:R 3:1 기준 | **Delta 에이전트** — 모든 진입 신호에 자동 R:R 계산, 3:1 미만 필터링 |
| 확신 기반 사이징 | **Delta 에이전트** — Omega 스코어 기반 동적 포지션 사이징 |
| 기술적 확인 | **Beta 에이전트** — 지지/저항, 패턴 돌파 확인 |
| 이벤트 리스크 관리 | **Gamma 에이전트** — 실적/이벤트 캘린더 연동 리스크 조정 |
| 손실 관리 | **Delta 에이전트** — 자동 손절, 드로다운 모니터링 |

### SEPA 차이점

| 항목 | Lipschutz 방식 | OmX 시스템 |

| 시장 시간 | 24시간 연속 | 정규 거래 시간 중심 |

### 전략군 (Strategy Family)

- **Family F (Macro)**: Lipschutz의 핵심 전략군. 매크로 분석 기반 방향성 매매

### 에이전트 매핑

| 에이전트 | Lipschutz 요소 | 구현 내용 |
|----------|---------------|----------|
| **Delta** | R:R 계산 | 진입가/목표가/손절가 자동 계산, R:R 3:1 필터 |
| **Delta** | 확신 기반 사이징 | 통합 스코어에 따른 포지션 크기 자동 결정 |
| **Beta** | 기술적 확인 | 지지/저항 돌파, 추세 확인 자동 분석 |
| **Gamma** | 이벤트 리스크 | 실적/경제지표 발표 전후 리스크 조정 |
| **Omega** | 통합 판단 | 매크로+기술+리스크 통합 → 최종 진입/대기 결정 |

### 구현 아이디어

1. **R:R 자동 계산기**: 모든 잠재 진입에 대해 기술적 지지/저항 기반 자동 R:R 계산, 3:1 미만 자동 제외
2. **확신도 기반 사이징 엔진**: Alpha+Beta+Gamma 스코어를 합산하여 확신도 산출, 포지션 크기 자동 결정
3. **이벤트 리스크 캘린더**: 실적 발표, 경제지표 발표 일정 자동 수집, 해당일 전후 리스크 파라미터 자동 조정
4. **상관관계 모니터**: 보유 포지션 간 상관관계를 실시간 추적, 과도한 동일 방향 노출 시 경고
5. **드로다운 기반 자동 디레버리징**: 계좌 드로다운이 임계점 초과 시 전체 포지션 비례 축소
