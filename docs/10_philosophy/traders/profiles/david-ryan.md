# David Ryan

## 기본 정보

| 항목 | 내용 |

## 한 줄 요약

> **"최고의 종목은 최고의 실적과 최고의 차트 패턴을 동시에 가진다 — CAN SLIM을 체계적으로 랭킹하고, 컵앤핸들과 VCP에서 정확히 진입하라."**

## 핵심 투자 철학 (5 Principles)

1. **O'Neil 방법론의 정량적 극대화 (Quantitative CAN SLIM)**
   - William O'Neil의 CAN SLIM 방법론을 가장 체계적으로 정량화한 제자다.
   - 주관적 판단을 최소화하고, 모든 조건을 수치 기반 랭킹 시스템으로 변환했다.
   - "O'Neil이 가르친 원칙을 기계적으로 적용하는 것이 내 강점이다."

2. **성장 + 모멘텀 랭킹 (Growth-Momentum Ranking)**
   - 모든 종목을 EPS 성장률, 매출 성장률, RS 순위, 기관 매집도 등의 팩터로 랭킹한다.
   - 랭킹 상위 종목만이 매수 후보군에 포함된다.
   - "종목 선정은 랭킹이다. 1등급 종목만 매수하고, 2등급 이하는 무시한다."

3. **패턴 인식이 타이밍이다 (Pattern Recognition = Timing)**
   - Cup-with-Handle, VCP, Flat Base, Double Bottom 등의 차트 패턴을 정밀하게 인식한다.
   - 패턴의 형태뿐 아니라 깊이, 기간, 거래량 특성까지 세밀하게 평가한다.
   - "같은 컵앤핸들이라도 품질에 차이가 있다. 최고 품질의 패턴에서만 진입한다."

4. **시장 방향을 존중하라 (Respect Market Direction)**
   - 시장 전체가 상승 추세일 때만 공격적으로 매수한다.
   - Follow-Through Day(FTD)로 시장 바닥을 확인하고, Distribution Day 누적으로 시장 천장을 감지한다.
   - "개별 종목이 아무리 좋아도 시장이 약세면 50%의 확률이 30%로 떨어진다."

5. **과거 대형 상승주 연구 (Study of Past Superperformers)**
   - 역사적으로 큰 상승을 기록한 종목(superperformers)의 공통 특성을 끊임없이 연구한다.
   - 과거 패턴의 반복을 통해 미래 대형 상승주를 사전에 인식한다.
   - "역사는 반복된다. 대형 상승주의 DNA는 시대를 초월하여 동일하다."

## 대표 전략 유형

### CAN SLIM 기반 Growth-Momentum 랭킹 전략

David Ryan의 전략은 O'Neil의 CAN SLIM을 정량적 랭킹 시스템으로 체계화한 것이다:

| CAN SLIM 요소 | Ryan의 정량화 |
|---------------|--------------|
| **C** (Current Earnings) | 최근 분기 EPS 성장률 > 25%, 가속 확인 |
| **A** (Annual Earnings) | 연간 EPS 성장률 > 25%, 3년 연속 |
| **N** (New Products/Management/Highs) | 신고가 돌파, 신제품/서비스 확인 |
| **S** (Supply/Demand) | 거래량 분석, 기관 매집 패턴 확인 |
| **L** (Leader/Laggard) | RS 순위 80 이상 (이상적으로 90+) |
| **I** (Institutional Sponsorship) | 기관 보유 비율 증가 추세 확인 |
| **M** (Market Direction) | Follow-Through Day / Distribution Day 카운트 |

### 차트 패턴 품질 평가

Ryan은 모든 베이스 패턴에 품질 점수를 부여한다:

| 평가 항목 | 고품질 | 저품질 |
|-----------|--------|--------|
| 베이스 깊이 | 12-25% 조정 | 35%+ 조정 |
| 베이스 기간 | 6-16주 | 3주 미만 또는 1년 초과 |
| 거래량 패턴 | 수축 시 감소, 돌파 시 급증 | 불규칙 |
| 핸들 위치 | 베이스 상반부 | 베이스 하반부 |
| 핸들 조정 | 8-12% | 15%+ |
| RS 유지 | 베이스 형성 중 RS 유지/상승 | RS 하락 |

## 매매 기법

### 진입 (Entry)

- **Cup-with-Handle 돌파 매수**: 클래식한 컵앤핸들 패턴의 핸들 고점(pivot point) 돌파 시 매수. 돌파 당일 거래량이 평균의 150%+ 필요.
- **VCP 돌파 매수**: Volatility Contraction Pattern의 마지막 수축 고점 돌파 시 매수. 수축 횟수 3회 이상, 각 수축의 폭이 감소해야 한다.
- **Flat Base 돌파**: 5-7주 이상의 횡보 후 상단 돌파. 조정 폭 15% 이내.
- **Pocket Pivot 매수**: O'Neil/Gil Morales 방식의 Pocket Pivot — 베이스 내에서 거래량 급증일에 선제 매수.
- **추격 매수 금지**: Pivot에서 5% 이상 벌어진 종목은 절대 매수하지 않는다.

### 청산 (Exit)

- **20-25% 수익 목표**: 매수가 대비 20-25% 수익 시 최소 부분 청산.
- **8주 보유 규칙**: 매수 후 첫 3주 이내에 20%+ 상승한 종목은 최소 8주간 보유 (O'Neil 규칙).
- **이동평균 이탈**: 50일 이동평균 아래 2일 연속 마감 시 청산 검토. 200일 이탈 시 즉시 청산.
- **Climax Top**: 급등 + 거래량 폭증 + 갭업 + 주봉 대양봉 = 매도 신호.
- **실적 감속**: EPS 성장률 둔화 또는 어닝 미스 시 청산.

### 손절 (Stop-Loss)

- **절대 규칙**: 매수가 대비 7-8% 하락 시 무조건 손절. 예외 없음.
- **Pivot 기반**: Pivot point 아래로 마감 시 7-8% 이전이라도 손절.
- **갭다운 대응**: 실적 발표 후 갭다운 시 시가에 즉시 청산 (장전 손절).
- **시장 환경 조정**: 약세장에서는 손절선을 5-6%로 축소.

### 포지션 사이징 (Position Sizing)

- **집중 투자**: 동시 보유 종목 4-8개.
- **최대 단일 포지션**: 총 자본의 20-25%.
- **균등 배분 기본**: 초기에는 균등 배분 후, 수익 포지션에 추가 배분(pyramiding).
- **계좌 규모 반영**: 소규모 계좌일수록 더 집중 (3-5개).

### 리스크 관리 (Risk Management)

- **일일 드로다운 모니터링**: 포트폴리오 5% 하락 시 모든 포지션 재평가.
- **연속 손절 규칙**: 3회 연속 손절 시 포지션 사이즈 50% 축소.
- **시장 노출 조절**: Distribution Day 4회 이상 누적 시 신규 매수 중단, 현금 비중 확대.
- **상관관계 관리**: 동일 섹터/테마 종목 편중 방지. 최대 3개 섹터 분산.

## 심리와 의사결정

### 핵심 멘탈 모델

- **규칙 기반 매매 (Rule-Based Trading)**: 감정이 아닌 규칙에 따라 매매한다. 규칙은 사전에 설정하고, 시장에서는 실행만 한다.
- **겸손함 (Humility)**: O'Neil 밑에서 배운 가장 큰 교훈은 겸손이다. "시장은 항상 나보다 크다."
- **준비의 중요성**: 주말에 차트 연구에 수시간을 투자한다. 준비된 트레이더만이 기회를 포착한다.
- **패턴 학습**: 과거 대형 상승주의 차트를 수천 개 이상 반복 학습하여 패턴 인식력을 강화했다.

### Ryan의 심리 규칙

1. 시장이 어렵다고 느끼면 매매를 멈추고 관찰하라
2. 챔피언십 경험이 가르쳐준 것 — 매 거래를 대회처럼 진지하게 임하라
3. 종목에 대한 편견을 가지지 마라. 숫자와 차트만 보라
4. 최고의 매매는 기다림에서 나온다. 완벽한 셋업이 아니면 패스하라
5. 실패한 거래를 기록하고 복기하는 것이 성공으로 가는 가장 빠른 길이다

## 대표 사례

### 성공 사례: U.S. Investing Championship 3연패 (1985-1987)

- **배경**: David Ryan은 1985년(161%), 1986년(160%), 1987년(46%)의 수익률로 U.S. Investing Championship에서 전무후무한 3연패를 달성했다.
- **전략**: O'Neil의 CAN SLIM을 체계적으로 적용하되, 자체 랭킹 시스템으로 종목을 정량 평가하여 최상위 종목에 집중.
- **핵심 요인**:
  - 1985-1986년 강세장에서 성장주의 폭발적 상승을 포착
  - 1987년 블랙먼데이 직전에 이미 현금 비중을 확대 (Distribution Day 감지)
  - 소수 정예 종목에 집중하여 수익 극대화
- **교훈**: 체계적 시스템 + 시장 방향 인식 + 집중 투자의 조합이 지속적 초과수익을 만든다.

### 성공 사례: O'Neil 밑에서의 학습과 실전 적용

- **배경**: Ryan은 20대 초반에 William O'Neil & Co.에 입사하여 O'Neil에게 직접 사사받았다.
- **전략**: O'Neil의 차트 모델북(과거 대형 상승주 차트 모음)을 수천 시간 연구하며 패턴 인식력을 극대화.
- **핵심 요인**: "O'Neil은 나에게 차트를 읽는 법이 아니라 차트를 느끼는 법을 가르쳤다"는 Ryan의 회고.
- **교훈**: 멘토 아래에서의 체계적 학습이 독학보다 수년의 시간을 단축시킨다.

### 실패 사례: 과신과 집중 리스크

- **배경**: 챔피언십 이후 자신감이 과잉된 시기에 단일 종목에 과도하게 집중하여 큰 손실을 경험.
- **원인**:
  - 과거 성공 경험에 의한 과신 (overconfidence bias)
  - 포지션 사이징 규칙 일탈
  - 손절선을 확대하여 더 큰 손실 초래
- **대응**: 기본 규칙으로 회귀. 포지션 사이징과 손절 규칙을 재확인.
- **교훈**: "규칙은 좋은 시기에 만들고, 나쁜 시기에 지키는 것이다."

## 내 시스템과의 연결

### SEPA 공통점

David Ryan의 CAN SLIM 정량화 접근법은 SEPA/OmX 시스템과 높은 일치도를 보인다:

| Ryan 원칙 | OmX/SEPA 대응 |
|-----------|----------------|
| CAN SLIM 랭킹 시스템 | **Alpha 에이전트** — Trend Template + RS 순위 기반 자동 랭킹 |
| Cup-with-Handle / VCP 패턴 | **Beta 에이전트** — 차트 패턴 자동 인식, 피봇 포인트 계산 |
| EPS 가속 + 매출 성장 | **Gamma 에이전트** — 펀더멘탈 팩터 자동 스크리닝 |
| Market Direction (M factor) | **Market Regime 모듈** — FTD/Distribution Day 자동 카운트 |
| 7-8% 손절 + 포지션 사이징 | **Delta 에이전트** — 리스크 관리 자동화 |

### SEPA 차이점

| 항목 | Ryan (CAN SLIM) | OmX 시스템 |
|------|-----------------|-----------|
| 종목 선정 | CAN SLIM 7개 요소 통합 | Trend Template 8조건 중심 |
| 패턴 범위 | Cup-with-Handle 중심, 다양한 O'Neil 패턴 | VCP 중심 (Minervini 특화) |
| 랭킹 방식 | 다요소 종합 랭킹 | 정량 조건 통과 여부 (Pass/Fail) |
| 시장 방향 | FTD / Distribution Day | Market Regime 지표 (복합) |
| 기관 매집 | I factor로 명시적 확인 | 거래량 패턴으로 간접 확인 |
| 뉴스/제품 | N factor (질적 평가 포함) | 정량 데이터 위주 |

### 전략군 (Strategy Family)

- **Family B (Growth Momentum)**: Ryan의 핵심 전략군. CAN SLIM 기반 성장 모멘텀 랭킹이 Family B의 정량 버전.
- **Family E (VCP/Chart Pattern)**: Cup-with-Handle, VCP 등 차트 패턴 기반 진입 전략이 Family E에 해당.

### 에이전트 매핑

| 에이전트 | Ryan 요소 | 구현 내용 |
|----------|-----------|----------|
| **Alpha** | L (Leader) + Trend Template | RS 순위 80/90+ 필터링, Stage 2 확인, CAN SLIM 랭킹 스코어 |
| **Beta** | Cup-with-Handle / VCP / Flat Base | 패턴 자동 인식, 품질 점수 부여, Pivot Point 계산 |
| **Gamma** | C + A (Earnings) | EPS/매출 성장률 가속 확인, 연간 성장 3년 연속 확인 |
| **Delta** | Risk Management | 7-8% 손절, R:R 계산, 연속 손절 시 사이즈 축소 |
| **Omega** | CAN SLIM 통합 판단 | Alpha+Beta+Gamma+Delta+Market Regime 통합 스코어 |

### 구현 아이디어

1. **CAN SLIM 종합 랭킹 엔진**: C, A, N, S, L, I 각 팩터에 가중치를 부여하여 종합 랭킹 점수를 산출. Alpha 에이전트의 기존 Trend Template에 CAN SLIM 팩터를 추가 레이어로 구현.
2. **Cup-with-Handle 자동 감지**: 베이스의 좌측벽-바닥-우측벽-핸들 구조를 알고리즘으로 감지. 깊이(12-35%), 기간(7-65주), 핸들 특성을 자동 평가.
3. **베이스 패턴 품질 점수**: 깊이, 기간, 거래량 패턴, RS 유지도, 핸들 특성 등을 종합하여 A/B/C 등급 부여. Beta 에이전트 확장.
4. **Distribution Day 카운터**: 시장 지수의 Distribution Day를 자동 카운트하여 M factor를 정량화. 4회 이상 누적 시 경고 알림.
5. **Superperformer 차트 DB**: 과거 대형 상승주의 차트 패턴을 데이터베이스화하여 현재 종목과 유사도 비교. 패턴 매칭 기반 추천.
