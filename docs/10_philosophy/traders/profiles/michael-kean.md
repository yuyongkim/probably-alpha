## 한 줄 요약

> **"최고의 포트폴리오는 최고의 전략들의 모음이 아니라, 서로 다른 타이밍에 작동하는 전략들의 조합이다 — 상관관계가 수익률보다 중요하다."**

## 핵심 투자 철학 (5 Principles)

1. **저상관 전략의 조합 (Low-Correlation Strategy Combination)**
   - 개별 전략의 절대 수익률보다 전략 간 상관관계가 더 중요하다.
   - 서로 다른 시장 환경에서 작동하는 전략들을 조합하여 전체 포트폴리오의 변동성을 낮춘다.
   - "A 전략이 잃을 때 B 전략이 벌면, 전체는 안정적으로 성장한다."

2. **포트폴리오 레벨 사고 (Portfolio-Level Thinking)**
   - 개별 거래나 개별 전략의 결과에 집착하지 않고, 포트폴리오 전체의 성과를 관리한다.
   - 하나의 전략이 손실을 내더라도 전체 포트폴리오가 안정적이면 성공이다.
   - 분산은 종목 분산이 아니라 "전략 분산"이 핵심이다.

3. **안정적 복리 성장 (Steady Compounding)**
   - 높은 수익률보다 낮은 변동성의 꾸준한 수익이 장기적으로 더 큰 자산을 만든다.
   - 드로우다운을 최소화하는 것이 수익을 극대화하는 것보다 중요하다.
   - Sharpe Ratio와 같은 위험 조정 수익률 지표를 핵심 관리 대상으로 삼는다.

4. **시장 환경 적응 (Market Regime Adaptation)**
   - 모든 전략에는 유리한 환경과 불리한 환경이 있다.
   - 다양한 시장 환경(추세, 횡보, 위기)에서 각각 작동하는 전략을 보유한다.
   - 시장 환경을 예측하는 것이 아니라, 모든 환경에 대비하는 것이 전략이다.

5. **리스크 예산 배분 (Risk Budget Allocation)**
   - 각 전략에 수익 기대치가 아닌 리스크 예산을 배분한다.
   - 변동성이 높은 전략에는 적은 리스크 예산을, 안정적인 전략에는 큰 리스크 예산을 할당한다.
   - 리스크 기여도(risk contribution)를 균등하게 유지하는 것이 목표다.

## 대표 전략 유형

### 멀티 전략 포트폴리오 (Multi-Strategy Portfolio)

Kean의 접근은 여러 독립적 전략을 하나의 포트폴리오로 결합하는 구조다:

| 구성 요소 | 역할 |
|-----------|------|
| **추세 추종 전략** | 명확한 추세가 있는 시장에서 수익 |
| **평균 회귀 전략** | 횡보 구간에서 과매수/과매도 활용 |
| **이벤트 드리븐 전략** | 특정 이벤트에 의한 비효율성 활용 |
| **변동성 전략** | 옵션 등을 활용한 변동성 프리미엄 수확 |
| **상관관계 관리** | 전략 간 상관관계를 모니터링하고 리밸런싱 |

### 전략 선택 기준

1. **독립성**: 다른 전략과의 상관관계가 낮을 것
2. **양의 기대값**: 장기적으로 양의 수익을 기대할 수 있을 것
3. **투명성**: 수익의 원천(edge)을 명확히 설명할 수 있을 것
4. **견고성**: 다양한 시장 환경과 파라미터 변화에 민감하지 않을 것
5. **실행 가능성**: 실제 거래에서 슬리피지, 수수료 등을 반영해도 수익이 날 것

## 매매 기법

### 진입 (Entry)

- **전략별 독립 신호**: 각 전략이 독립적으로 진입 신호를 생성
- **포트폴리오 레벨 확인**: 전체 포트폴리오의 리스크 예산 범위 내에서만 신규 진입 허용
- **상관관계 체크**: 새 포지션이 기존 포지션과 높은 상관관계를 갖지 않는지 확인
- **리스크 예산 준수**: 각 전략의 리스크 예산 한도 내에서만 진입

### 청산 (Exit)

- **전략별 독립 규칙**: 각 전략의 고유한 청산 규칙에 따라 실행
- **포트폴리오 리밸런싱**: 전체 포트폴리오의 리스크 균형이 깨지면 리밸런싱 차원의 청산
- **상관관계 변화 대응**: 전략 간 상관관계가 급격히 증가하면 노출도 축소
- **정기 재평가**: 월간/분기별로 전략 성과와 상관관계를 재평가

### 손절 (Stop-Loss)

- **전략별 손절**: 각 전략 고유의 손절 규칙 적용
- **포트폴리오 레벨 손절**: 전체 포트폴리오 드로우다운이 한도 초과 시 전면 축소
- **전략 중단**: 특정 전략이 예상 드로우다운 범위를 벗어나면 해당 전략 중단 후 재검토
- **상관관계 손절**: 전략 간 상관관계 급등 시 리스크 축소

### 포지션 사이징 (Position Sizing)

- **리스크 패리티**: 각 전략의 리스크 기여도를 균등하게 배분
- **변동성 정규화**: 전략별 변동성에 반비례하여 자본 배분
- **동적 리밸런싱**: 시장 환경 변화에 따라 전략별 배분 비중 조정
- **총 리스크 한도**: 전체 포트폴리오의 연간 변동성 목표 10-15%

### 리스크 관리 (Risk Management)

- **상관관계 실시간 모니터링**: 전략 간 상관관계의 변화를 실시간 추적
- **스트레스 테스트**: 2008 금융위기, 2020 코로나 등 극단적 환경에서의 포트폴리오 시뮬레이션
- **꼬리 리스크 관리**: 극단적 시나리오에서의 동시 손실 가능성을 사전 평가
- **유동성 관리**: 모든 전략의 포지션이 합리적 시간 내 청산 가능한지 확인

## 심리와 의사결정

### 핵심 멘탈 모델

- **전체를 보는 시야**: 개별 나무가 아니라 숲을 본다. 하나의 전략이 잃어도 전체가 안정적이면 성공이다.
- **분산의 수학적 이해**: 상관관계가 낮은 자산의 조합이 전체 리스크를 줄인다는 수학적 원리를 깊이 이해한다.
- **결과보다 프로세스**: 단기 수익보다 리스크 관리 프로세스의 일관된 실행을 중시한다.
- **겸손**: 어떤 전략도 영원하지 않다. 시장이 변하면 전략도 변해야 한다는 겸손함을 유지한다.

### Kean의 심리 규칙

1. 개별 전략의 손실에 감정적으로 반응하지 마라 — 포트폴리오 전체를 보라
2. 상관관계는 위기 시에 증가한다 — 평상시의 상관관계를 과신하지 마라
3. 가장 좋은 전략보다 가장 다양한 전략 조합이 이긴다
4. 리스크 예산을 초과하지 마라 — 탐욕은 분산의 적이다
5. 단순한 전략 여러 개가 복잡한 전략 하나보다 견고하다

## 대표 사례

### 성공 사례: 위기 시 포트폴리오 안정성

- **배경**: 시장 급락 국면에서 추세 추종 전략이 손실을 내지만, 변동성 매수 전략과 숏 전략이 수익을 발생
- **전략**: 저상관 전략 조합으로 인해 전체 포트폴리오 드로우다운이 시장 대비 크게 제한됨
- **핵심 요인**:
  - 사전에 다양한 시장 환경에서 각각 작동하는 전략을 배치
  - 위기 시 전략 간 상관관계 변화를 모니터링하고 리밸런싱 실행
  - 개별 전략 손실을 수용하며 전체 포트폴리오 관리에 집중
- **교훈**: "모든 전략이 동시에 이길 필요는 없다. 전체가 안정적이면 그것이 최선이다."

### 실패/교훈 사례: 상관관계 급등의 교훈

- **배경**: 평상시 낮은 상관관계를 보이던 전략들이 극단적 시장 환경에서 동시에 손실 발생
- **원인**: 위기 시 상관관계가 급등하는 현상(correlation breakdown)을 충분히 반영하지 못한 리스크 모델
- **대응**: 스트레스 테스트 시나리오에 극단적 상관관계 급등을 포함, 꼬리 리스크 헤지 추가
- **교훈**: "정상 시장의 상관관계 행렬은 위기 시 의미를 잃는다. 극단적 환경에서의 상관관계를 별도로 모델링해야 한다."

### 구현 아이디어

1. **실시간 상관관계 대시보드**: 에이전트(전략) 간 상관관계를 실시간으로 시각화하고 이상 변화 알림
2. **리스크 패리티 엔진**: 전략별 변동성에 기반하여 자본을 자동 배분하는 리스크 패리티 로직
3. **레짐별 전략 가중치**: 시장 환경(추세/횡보/위기)에 따라 전략별 배분 비중을 자동 조절
4. **스트레스 테스트 자동화**: 과거 위기 시나리오와 극단적 상관관계 가정하의 포트폴리오 시뮬레이션 자동 실행
5. **전략 독립성 모니터**: 새로운 전략 추가 시 기존 전략과의 상관관계를 자동 평가하여 포트폴리오 개선 효과 정량화
