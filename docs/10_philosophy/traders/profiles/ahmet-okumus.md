# Ahmet Okumus

## 기본 정보

| 항목 | 내용 |
|------|------|
| 본명 | Ahmet Okumus |
| 출생 | 1970년대, 터키 이스탄불 |
| 활동 기간 | 1990년대 ~ 현재 |
| 주요 저서 | 직접 저서 없음 |
| 인터뷰 수록 | *Stock Market Wizards* (Jack Schwager, 2001) |
| 주요 실적 | Okumus Fund Management 설립, 1990년대 연평균 40%+ 수익률, 집중 투자로 S&P 500 대비 압도적 초과수익 |
| 운용 스타일 | Deep Value + Growth Catalyst / Concentrated Portfolio |
| 특징 | 터키에서 미국으로 이민, 소규모 자본에서 시작하여 성공. 극도의 집중 투자(5-10종목) |

## 한 줄 요약

> **"PBR 0.8 이하의 저평가 종목 중 성장 촉매가 확인되는 5-10개에 집중하라 — 깊은 가치(deep value)와 성장 전환(growth catalyst)의 교차점이 최고의 투자 기회다."**

## 핵심 투자 철학 (5 Principles)

1. **Deep Value가 출발점이다 (Start with Deep Value)**
   - PBR(주가순자산비율) 0.8 이하, 또는 자산 가치 대비 현저히 저평가된 종목만 매수 대상으로 고려한다.
   - 자산 가치가 하방 리스크의 방패막이 역할을 한다 — "자산보다 싸게 사면 하방은 제한되고 상방은 열려 있다."
   - 단순히 싸다고 매수하지 않는다. 싸면서 동시에 좋아질 이유가 있어야 한다.

2. **촉매(Catalyst)가 필수 조건이다**
   - 저평가 자체만으로는 매수 이유가 되지 않는다. 가치를 실현시킬 명확한 촉매가 있어야 한다.
   - 촉매의 유형: 경영진 교체, 구조조정, 실적 턴어라운드, 산업 규제 변화, 자사주 매입, 기관 진입 등.
   - "촉매 없는 저평가 종목은 영원히 저평가로 남을 수 있다. 촉매가 시간의 문을 열어준다."

3. **극도의 집중 투자 (Extreme Concentration)**
   - 포트폴리오를 5-10개 종목으로 제한한다. 최고 확신 종목에 자본의 20-30%를 집중한다.
   - 분산 투자는 무지의 보호막이다. 깊은 리서치를 한 투자자에게 분산은 수익을 희석시킬 뿐이다.
   - "10개 미만의 종목을 깊이 이해하는 것이 100개 종목을 얕게 아는 것보다 낫다."

4. **인내심이 수익을 만든다 (Patience Creates Returns)**
   - 적합한 종목이 없으면 현금을 보유한다. 무리하게 투자 대상을 찾지 않는다.
   - 매수 후에도 촉매가 실현될 때까지 인내심 있게 보유한다 (수개월~1년 이상).
   - "시장에서 가장 돈이 많이 드는 것은 행동이 아니라 조급함이다."

5. **독립적 사고와 역발상 (Independent & Contrarian Thinking)**
   - 시장의 컨센서스를 의심하고, 다수가 외면하는 곳에서 기회를 찾는다.
   - 터키에서 미국으로 이민한 외부자 시각이 시장의 편향을 객관적으로 볼 수 있게 해준다.
   - "대중이 매도할 때 매수하고, 대중이 매수할 때 경계한다."

## 대표 전략 유형

### Deep Value + Growth Catalyst 집중 투자 전략

Okumus의 전략은 전통적 가치 투자와 성장 촉매를 결합한 독자적 접근법이다:

| 구성 요소 | 역할 |
|-----------|------|
| **자산 가치 분석** | PBR, 순자산가치(NAV), 청산 가치 계산으로 하방 리스크 평가 |
| **촉매 확인** | 가치 실현 촉매(경영 변화, 실적 턴어라운드, 구조조정 등) 존재 확인 |
| **실적 방향성** | EPS 턴어라운드 또는 성장 재개 신호 확인 |
| **기관 동향** | 기관의 신규 매수/매집 패턴 확인 |
| **집중 배분** | 최고 확신 5-10개 종목에 집중 배분 |

### 핵심 스크리닝 기준

1. PBR < 0.8 (자산 대비 저평가)
2. PER < 15 또는 실적 턴어라운드 초기 (아직 PER이 높지만 개선 추세)
3. 부채비율 < 100% (재무 안정성)
4. 명확한 촉매 존재 (경영진 교체, 구조조정, 자사주 매입 등)
5. 내부자 매수 또는 기관 신규 진입 신호
6. 산업 내 경쟁 지위 유지 (일시적 어려움이지 구조적 쇠퇴가 아님)

## 매매 기법

### 진입 (Entry)

- **가치 기반 진입**: PBR 0.8 이하에서 촉매가 확인될 때 매수 시작. 저평가의 정도가 클수록 매력적.
- **분할 매수**: 총 계획 포지션의 30-50%를 1차 진입, 추가 확인(실적 개선, 촉매 현실화) 시 나머지 매수.
- **내부자 추종**: 경영진/대주주의 자사주 매수가 시작되면 강력한 진입 신호.
- **기관 진입 확인**: 13F 공시 등을 통해 유명 기관의 신규 매수를 확인하면 확신도 증가.
- **역발상 타이밍**: 나쁜 뉴스에 과매도된 종목에서 촉매가 임박할 때 매수 (패닉 매도 후 안정화 시점).

### 청산 (Exit)

- **목표 가치 도달**: 자산 가치(NAV) 또는 적정 PBR(1.0-1.5) 수준에 도달하면 청산.
- **촉매 실현**: 기대한 촉매가 실현되어 주가에 반영되면 부분/전체 청산.
- **펀더멘탈 악화**: 촉매가 지연되거나 취소되면 재평가 후 청산 검토.
- **기회 비용**: 더 매력적인 투자 기회가 발견되면 기존 포지션 교체.
- **보유 기간**: 통상 6개월~2년. 촉매 실현에 필요한 시간을 충분히 확보.

### 손절 (Stop-Loss)

- **펀더멘탈 손절**: 기술적 가격 수준보다 펀더멘탈 변화에 기반한 손절. 투자 논거(thesis)가 무너지면 가격과 무관하게 청산.
- **가격 기반 보조 손절**: 매수가 대비 15-20% 하락 시 투자 논거를 재검토. 논거가 유효하면 유지, 불확실하면 청산.
- **시간 기반 손절**: 12개월 이상 촉매가 현실화되지 않으면 기회 비용을 고려하여 청산 검토.
- **물타기 조건부 허용**: 투자 논거가 여전히 유효하고 저평가가 심화되면 제한적 추가 매수 가능 (일반 모멘텀 전략과 차이점).

### 포지션 사이징 (Position Sizing)

- **극도의 집중**: 최고 확신 종목에 포트폴리오의 20-30%.
- **총 보유 종목**: 5-10개. 10개를 초과하지 않는다.
- **확신도 반영**: 촉매의 명확성 + 저평가 정도 + 재무 안정성의 조합으로 확신도를 정량 평가하여 사이징.
- **현금 보유**: 적합한 기회가 부족하면 현금 비중 30-50%까지 유지.

### 리스크 관리 (Risk Management)

- **하방 리스크 자산 보호**: PBR < 0.8 매수 원칙 자체가 하방 리스크 관리. 자산 가치가 손실의 하한선 역할.
- **포지션별 최대 손실 한도**: 단일 포지션에서 포트폴리오 전체의 5% 이상 손실 시 재평가.
- **부채 리스크 회피**: 과도한 부채를 진 기업은 아무리 저평가라도 매수하지 않는다.
- **집중 리스크 인식**: 5-10개 집중이므로 개별 종목 리스크가 크다. 이를 깊은 리서치로 상쇄한다.
- **시장 환경 조절**: 전체 시장이 고평가 구간이면 현금 비중을 확대하고 기다린다.

## 심리와 의사결정

### 핵심 멘탈 모델

- **역발상 사고 (Contrarian Mindset)**: 대중의 공포가 곧 기회다. 시장이 패닉에 빠질 때 냉정하게 가치를 평가할 수 있는 것이 엣지다.
- **인내심 (Patience)**: "아무것도 하지 않는 것"이 때때로 최선의 행동이다. 기회가 올 때까지 기다리는 인내심.
- **확신과 집중 (Conviction & Focus)**: 소수 종목에 집중하려면 그만큼 깊은 리서치와 강한 확신이 필요하다. 확신 없이 집중하는 것은 도박이다.
- **외부자 시각 (Outsider's Perspective)**: 터키에서 미국으로 온 이민자로서의 외부자 시각이 시장의 편향을 더 객관적으로 볼 수 있게 해준다.

### Okumus의 심리 규칙

1. 군중과 같은 방향으로 달리지 마라 — 군중은 대개 틀린다
2. 투자 논거를 글로 써라 — 써지지 않는 논거는 명확하지 않은 것이다
3. 매수 전에 "이 종목이 50% 하락하면 더 살 것인가?"를 자문하라. "예"가 아니면 매수하지 마라
4. 시장의 소음에 귀를 닫고, 재무제표에 눈을 열어라
5. 가장 좋은 투자는 가장 외로운 투자다 — 아무도 관심 없는 종목에서 기회가 온다

## 대표 사례

### 성공 사례: 1990년대 소규모 자본에서의 급성장

- **배경**: Okumus는 터키에서 미국으로 이민한 후 소규모 자본으로 펀드를 시작하여 연평균 40%+ 수익률을 기록하며 자산을 빠르게 증식시켰다.
- **전략**: PBR 0.8 이하의 저평가 소형주에서 경영 개선, 실적 턴어라운드 등의 촉매가 확인되는 종목에 집중 투자.
- **핵심 요인**:
  - 깊은 재무 분석으로 자산 가치 대비 할인율을 정밀 계산
  - 5-10개 종목에 집중하여 수익 극대화
  - 인내심 있게 촉매 실현을 기다림
- **교훈**: 소규모 자본 + 집중 투자 + 깊은 리서치 = 압도적 초과수익.

### 성공 사례: 턴어라운드 종목 투자

- **배경**: 일시적 어려움에 처한 기업이 경영진 교체와 구조조정을 통해 회복하는 과정에서 큰 수익을 실현.
- **전략**: PBR < 0.8에서 매수, 구조조정 진행 확인 후 피라미딩, 적정 가치 도달 시 청산.
- **핵심 요인**:
  - 기업의 핵심 사업가치와 일시적 문제를 구분하는 분석력
  - 내부자 매수 신호를 촉매 확인으로 활용
  - 촉매 실현까지 1년 이상 인내심 있게 보유
- **교훈**: 촉매가 명확한 턴어라운드 종목은 가치 투자에서 가장 높은 수익률을 제공한다.

### 실패 사례: 촉매 미실현 장기 보유

- **배경**: 저평가 + 촉매 기대로 매수했으나, 촉매가 지연되거나 취소되어 장기간 자본이 묶인 경험.
- **원인**:
  - 촉매의 실현 시점을 과도하게 낙관적으로 추정
  - "결국 오르겠지"라는 희망적 사고로 보유 기간 연장
  - 기회 비용 무시 — 해당 자본으로 더 나은 투자를 할 수 있었음
- **대응**: 촉매 기한(catalyst deadline)을 사전에 설정하는 규칙 도입. 12개월 내 촉매 미실현 시 재평가.
- **교훈**: "저평가는 영원할 수 있다. 촉매가 실현되지 않으면 시간은 적이 된다."

## 내 시스템과의 연결

### SEPA 공통점

Okumus의 Deep Value + Catalyst 전략은 SEPA의 펀더멘탈 분석 요소와 연결된다:

| Okumus 원칙 | OmX/SEPA 대응 |
|-------------|----------------|
| 저평가 스크리닝 (PBR < 0.8) | **Value 스크리너 (미래 구현)** — PBR, PER, NAV 기반 자동 필터링 |
| 촉매 확인 | **Gamma 에이전트** — 실적 턴어라운드, 가이던스 변화, 내부자 매수 감지 |
| 실적 방향성 | **Gamma 에이전트** — EPS 턴어라운드(적자→흑자, 감속→가속) 감지 |
| 기관 동향 | **수급 분석 모듈** — 기관 순매수, 13F 공시 분석 |
| 집중 투자 | **Delta 에이전트** — 포지션 사이징에 확신도 반영 |

### SEPA 차이점

| 항목 | Okumus (Deep Value) | OmX 시스템 (SEPA) |
|------|---------------------|-------------------|
| 출발점 | 자산 가치(PBR) 기반 저평가 | 기술적 추세(Stage 2) 기반 상승 모멘텀 |
| 진입 시점 | 하락/횡보 구간에서 매수 | 상승 추세 확인 후 매수 |
| 보유 기간 | 6개월~2년 (장기) | 수주~수개월 (중기) |
| 손절 방식 | 펀더멘탈 기반 (논거 무너짐) | 가격 기반 (7-8%) |
| 밸류에이션 | 핵심 기준 (PBR < 0.8) | 보조 지표 |
| 물타기 | 조건부 허용 | 절대 금지 |
| 시장 추세 | 상대적으로 덜 중요 | 핵심 필터 (M factor) |

### 전략군 (Strategy Family)

- **Family D (Deep Value + Catalyst)**: Okumus의 핵심 전략군. PBR 기반 저평가 + 촉매 확인의 조합은 Family D의 원형.
- OmX 시스템의 현재 주요 전략(Family B/E)과는 접근법이 다르지만, 향후 **가치 투자 모듈 확장** 시 Family D 전략으로 구현 가능.

### 에이전트 매핑

| 에이전트 | Okumus 요소 | 구현 내용 |
|----------|-------------|----------|
| **Value (미래)** | PBR < 0.8 스크리닝 | PBR, PER, NAV 자동 계산, 저평가 종목 필터링 (신규 에이전트) |
| **Gamma** | 촉매 감지 | 실적 턴어라운드, 내부자 매수, 구조조정 뉴스 자동 감지 |
| **Delta** | 집중 사이징 + 리스크 관리 | 5-10개 집중 시 포지션별 리스크 한도 관리 |
| **Omega** | 종합 판단 | Value + Gamma + Delta 통합 (Family D 모드) |

### 구현 아이디어

1. **Deep Value 스크리너 (Family D 전용)**: PBR < 0.8, 부채비율 < 100%, 영업이익 흑자 전환 등의 가치 기반 조건으로 자동 스크리닝. 현재의 모멘텀 중심 Alpha와 별개의 Value 스크리너로 구현.
2. **촉매 캘린더**: 기대되는 촉매(실적 발표일, 주주총회, 규제 변화 예정일 등)를 캘린더에 등록하고, 실현 여부를 자동 추적. 촉매 미실현 시 경고.
3. **내부자 매수 알림**: 대주주/경영진의 자사주 매수 공시를 자동 감지하여 알림. Okumus가 가장 중시하는 촉매 유형.
4. **NAV 계산기**: 기업의 순자산가치를 자동 계산하고, 현재 주가와의 할인율을 표시. Deep Value 투자의 핵심 도구.
5. **Family D 백테스트**: PBR < 0.8 + 촉매 조건의 한국 시장 유효성 검증. SEPA의 Family B/E와의 성과 비교 분석.
