# Stuart Walton

## 기본 정보

| 항목 | 내용 |
|------|------|
| 본명 | Stuart Walton |
| 출생 | 1960년대, 미국 |
| 활동 기간 | 1990년대 ~ 현재 |
| 주요 저서 | 직접 저서 없음 |
| 인터뷰 수록 | *Stock Market Wizards* (Jack Schwager, 2001) |
| 주요 실적 | 1990년대 후반 IT 성장주로 수백 퍼센트 수익률 기록, 큰 손실 후 복귀하여 재기 성공 |
| 운용 스타일 | Growth Momentum / IT Sector Specialist / Pattern Recognition Trading |
| 특징 | 손실로부터의 회복(Recovery)과 심리적 성장 과정이 특히 주목받음 |

## 한 줄 요약

> **"IT 성장주의 패턴을 읽고, 큰 손실에서 배우며, 심리적으로 성장한 뒤에야 비로소 일관된 수익이 가능하다 — 패턴 인식과 자기 인식의 결합이 핵심이다."**

## 핵심 투자 철학 (5 Principles)

1. **IT/기술 성장주에 집중하라 (Focus on Technology Growth Stocks)**
   - Walton은 자신이 가장 잘 이해하는 기술 섹터에 집중한다.
   - IT 산업의 성장 동력, 제품 사이클, 경쟁 구도에 대한 깊은 이해가 종목 선정의 기초다.
   - "내가 이해하지 못하는 산업의 주식은 매수하지 않는다. 기술주는 내 언어다."

2. **패턴 인식이 매매의 핵심이다 (Pattern Recognition is Key)**
   - 차트 패턴, 가격 행동, 거래량 패턴에서 반복되는 구조를 인식한다.
   - 수천 개의 과거 차트를 연구하여 축적한 패턴 라이브러리가 직관의 원천이다.
   - "좋은 차트는 좋은 이야기를 들려준다 — 기관의 매집, 수급의 균형, 돌파의 임박을 보여준다."

3. **큰 손실에서 배우는 것이 성장이다 (Learning from Major Losses)**
   - Walton은 초기 경력에서 큰 손실을 경험했으며, 이로부터 리스크 관리의 중요성을 체득했다.
   - 손실은 실패가 아니라 학습 기회다. 손실의 원인을 분석하고 시스템을 개선한다.
   - "나를 파산 직전까지 몰았던 경험이 나를 진정한 트레이더로 만들었다."

4. **시장 환경에 적응하라 (Adapt to Market Environment)**
   - 시장은 항상 변한다. 1년 전에 작동한 전략이 지금은 작동하지 않을 수 있다.
   - 강세장과 약세장에서 전혀 다른 접근법을 사용한다.
   - 약세장에서는 현금을 보유하는 것 자체가 최선의 전략이다.

5. **자기 인식(Self-Awareness)이 수익의 전제 조건이다**
   - 자신의 감정 상태, 편향, 약점을 인식하는 것이 일관된 매매의 기초다.
   - 흥분, 두려움, 탐욕이 의사결정을 왜곡한다는 것을 경험으로 체득했다.
   - "시장을 이기기 전에 먼저 자신을 이겨야 한다."

## 대표 전략 유형

### IT Growth Momentum + Pattern Recognition 전략

Walton의 전략은 IT 섹터 전문성과 기술적 패턴 인식을 결합한다:

| 구성 요소 | 역할 |
|-----------|------|
| **섹터 전문성** | IT/기술 산업의 트렌드, 제품 사이클, 경쟁 구도 분석 |
| **EPS 가속** | 분기 실적 성장률 가속 확인 (20%+ 성장, 가속 패턴) |
| **차트 패턴** | 베이스 형성, 돌파, 거래량 패턴의 정밀 분석 |
| **상대강도** | RS 상위 종목으로 유니버스 제한 |
| **시장 환경** | 강세/약세 판단에 따른 노출 조절 |

### 핵심 패턴 유형

Walton이 특히 주목하는 차트 패턴:

| 패턴 | 특성 | 신뢰도 |
|------|------|--------|
| **베이스 돌파** | 6주+ 횡보 후 거래량 수반 돌파 | 높음 |
| **풀백 매수** | 돌파 후 베이스 상단으로의 되돌림에서 재매수 | 높음 |
| **갭업 유지** | 실적 발표 후 갭업이 첫 주 내 유지되는 패턴 | 매우 높음 |
| **3-tight-closes** | 3일 연속 종가가 거의 동일한 패턴 (기관 매집 신호) | 중간-높음 |
| **High-tight flag** | 급등 후 좁은 범위의 횡보 (강력한 연속 신호) | 높음 |

## 매매 기법

### 진입 (Entry)

- **베이스 돌파 매수**: IT 성장주가 6주 이상의 베이스 패턴을 형성한 후 상단을 거래량 증가와 함께 돌파할 때 매수.
- **어닝 갭업 매수**: 실적 발표 후 큰 갭업이 발생하고 첫 주 내 갭이 유지될 때 매수. IT 주의 어닝 갭업은 새로운 성장 국면 진입의 신호.
- **풀백 매수**: 초기 돌파를 놓쳤을 때, 돌파 가격대로의 건설적인 풀백(거래량 감소 수반)에서 매수.
- **섹터 로테이션 확인**: IT 섹터 전체가 자금 유입 단계에 있는지 확인한 후 개별 종목 진입.
- **분할 진입**: 1차 포지션(50%) → 확인 후 2차(30%) → 추가 확인 후 3차(20%).

### 청산 (Exit)

- **모멘텀 정점 인식**: 주가 상승 속도가 극도로 빨라지고 거래량이 폭증하면 클라이맥스로 인식하고 부분 청산.
- **이동평균 이탈**: 10일 이동평균 아래 마감이 2일 이상 지속되면 단기 포지션 청산. 50일 이동평균 이탈 시 중기 포지션 청산.
- **실적 감속**: 다음 분기 실적 성장률이 둔화되면 청산 검토. IT주의 실적 감속은 주가 급락의 전조.
- **섹터 로테이션 이탈**: IT 섹터에서 자금이 빠져나가는 신호가 감지되면 전체 IT 포지션 축소.

### 손절 (Stop-Loss)

- **초기 손절선**: 매수가 대비 7-8% 하락 시 무조건 손절.
- **패턴 이탈 손절**: 돌파 후 베이스 내부로 재진입(false breakout)하면 즉시 손절.
- **갭다운 손절**: 실적 미스 등으로 갭다운 시 시가에 청산. IT주의 갭다운은 크고 빠르기 때문에 지체하면 손실이 급격히 확대된다.
- **심리적 손절**: 포지션에 대한 불안감이 지속되면 기술적 이유가 없더라도 포지션을 줄인다.

### 포지션 사이징 (Position Sizing)

- **핵심 포지션**: 최고 확신 종목에 포트폴리오의 15-20%.
- **보유 종목 수**: 5-8개. 섹터가 집중되어 있으므로 종목 수는 제한.
- **손실 후 축소**: 큰 손실 후에는 포지션 사이즈를 50-70% 축소하여 자신감 회복에 집중.
- **피라미딩**: 수익이 나는 포지션에만 추가 매수. 손실 포지션에는 절대 추가하지 않음.

### 리스크 관리 (Risk Management)

- **섹터 집중 인식**: IT에 집중하는 만큼 섹터 리스크를 명확히 인식하고, 시장 환경 악화 시 과감하게 현금으로 전환.
- **드로다운 규칙**: 포트폴리오 10% 드로다운 시 모든 신규 매수 중단. 15% 시 기존 포지션도 축소.
- **상관관계 관리**: IT 내에서도 서브섹터(반도체, 소프트웨어, 인터넷, 하드웨어) 간 분산.
- **회복 프로토콜**: 큰 손실 후에는 정해진 회복 절차를 따른다 — 소규모 매매로 시작, 3회 연속 수익 후 사이즈 확대.

## 심리와 의사결정

### 핵심 멘탈 모델

- **회복탄력성 (Resilience)**: Walton의 가장 큰 강점은 큰 손실에서 회복하는 능력이다. 파산 직전의 경험에서 돌아와 재기에 성공했다.
- **자기 인식 (Self-Awareness)**: 자신의 감정 상태를 지속적으로 모니터링한다. "지금 내가 흥분하고 있는가, 두려워하고 있는가?"를 매 거래 전 확인.
- **점진적 신뢰 구축**: 큰 손실 후 자신감을 회복하는 과정은 점진적이어야 한다. 소규모 매매로 시작하여 성공 경험을 쌓은 후 사이즈를 키운다.
- **과거 실수의 내면화**: 과거의 실수를 잊지 않되, 트라우마로 남기지 않는다. 실수를 규칙으로 변환한다.

### Walton의 심리 규칙

1. 큰 손실 후에는 반드시 쉬어라 — 복수 매매(revenge trading)는 파멸의 지름길이다
2. 자신의 감정 상태를 매매 일지에 기록하라 — 감정의 패턴을 인식하는 것이 첫걸음이다
3. 시장에 대한 겸손함을 유지하라 — 시장은 당신의 자존심에 관심이 없다
4. 성공 경험이 축적될 때까지 작은 사이즈로 매매하라
5. 멘탈이 무너진 상태에서의 매매는 어떤 전략이든 실패한다

## 대표 사례

### 성공 사례: 1990년대 후반 IT 성장주 랠리

- **배경**: Walton은 1990년대 후반 인터넷/기술 붐에서 IT 성장주에 집중하여 수백 퍼센트의 수익률을 기록했다.
- **전략**: IT 섹터에 대한 깊은 이해를 바탕으로 초기 인터넷 기업, 반도체 성장주, 소프트웨어 기업 등에 집중 투자.
- **핵심 요인**:
  - 기술 산업의 구조적 성장 트렌드를 일찍 인식
  - EPS 가속 + RS 상위 + 베이스 돌파의 조합으로 최적 종목 선정
  - 강세장의 순풍을 최대한 활용한 공격적 사이징
- **교훈**: 자신이 가장 잘 아는 섹터에 집중하는 것이 범용적 접근보다 우월하다.

### 성공 사례: 큰 손실 후 재기

- **배경**: Walton은 경력 중 큰 드로다운을 경험한 후, 체계적인 회복 프로세스를 통해 재기에 성공했다.
- **전략**: 소규모 매매로 시작하여 점진적으로 사이즈를 키우는 회복 프로토콜 적용.
- **핵심 요인**:
  - 손실의 원인을 정직하게 분석 (과도한 레버리지, 손절 지연, 시장 환경 무시)
  - 심리적 회복에 필요한 시간을 충분히 확보
  - 기본 규칙(손절, 사이징)을 더욱 엄격하게 재설정
- **교훈**: 트레이더의 진정한 능력은 수익을 내는 것이 아니라 손실에서 회복하는 것에서 드러난다.

### 실패 사례: 과도한 레버리지와 집중

- **배경**: 경력 초기에 IT 종목에 과도하게 레버리지를 사용하여 집중 투자한 결과 파산 직전까지 몰렸다.
- **원인**:
  - 마진 사용으로 인한 레버리지 과다
  - 단일 섹터(IT)에 대한 과도한 집중
  - 손절 규칙의 부재 또는 미이행
  - 시장 전체 방향(약세 전환)을 무시
- **대응**: 레버리지 사용 제한, 섹터 집중도 모니터링, 엄격한 손절 규칙 수립.
- **교훈**: "레버리지는 양날의 검이다. 수익을 증폭시키지만, 손실도 동일하게 증폭시킨다."

## 내 시스템과의 연결

### SEPA 공통점

Walton의 IT 성장 모멘텀 전략은 SEPA/OmX 시스템과 다음과 같이 연결된다:

| Walton 원칙 | OmX/SEPA 대응 |
|-------------|----------------|
| 베이스 돌파 + 거래량 | **Beta 에이전트** — 베이스 패턴 인식, 돌파 시 거래량 확인 |
| EPS 가속 확인 | **Gamma 에이전트** — 분기별 EPS 가속 자동 계산 |
| RS 상위 필터 | **Alpha 에이전트** — RS 순위 자동 랭킹, 상위 필터링 |
| Stage 2 상승 추세 | **Alpha 에이전트** — Trend Template 8조건으로 Stage 2 확인 |
| 시장 환경 인식 | **Market Regime 모듈** — 시장 방향 자동 판단 |
| 회복 프로토콜 | **Delta 에이전트** — 드로다운 시 자동 사이즈 축소 로직 |

| 항목 | Walton | OmX 시스템 |
|------|--------|-----------|
| 섹터 범위 | IT/기술 전문 | 전 섹터 커버 |

| 레버리지 | 사용 (경험 후 제한적) | 미사용 (초기) |
| 심리 관리 | 직접 경험에서 체득 | 시스템 규칙으로 대체 |
| 매매 일지 | 수기 기록 + 감정 상태 | 자동 로깅 + 정량 기록 |
| 회복 프로세스 | 개인적/직관적 | 정량적 규칙 기반 |

### 전략군 (Strategy Family)

- **Family B (Growth Momentum)**: Walton의 핵심 전략군. IT 성장주의 EPS 가속 + 기술적 돌파 조합이 Family B에 속한다.

### 에이전트 매핑

| 에이전트 | Walton 요소 | 구현 내용 |
|----------|-------------|----------|
| **Alpha** | Stage 2 + RS 필터 | Trend Template 자동 스크리닝, IT 섹터 필터 옵션 |
| **Gamma** | EPS 가속 + IT 산업 분석 | 분기별 EPS/매출 가속 확인, IT 섹터 성장률 비교 |
| **Beta** | 차트 패턴 인식 | 베이스 돌파, 갭업 유지, High-tight flag 자동 감지 |
| **Delta** | 리스크 관리 + 회복 프로토콜 | 7-8% 손절, 드로다운 시 사이즈 축소, 회복 모드 전환 |
| **Omega** | 종합 판단 | 모든 에이전트 통합 + 시장 환경 가중치 적용 |

### 구현 아이디어

1. **갭업 유지 패턴 감지**: 실적 발표 후 갭업이 발생한 종목에서, 첫 주 내 갭을 메우지 않고 유지되는 패턴을 자동 감지. Beta 에이전트에 "Earnings Gap Hold" 로직 추가.
2. **회복 모드 자동 전환**: 포트폴리오 드로다운이 설정 임계값을 초과하면 자동으로 "회복 모드"로 전환. 포지션 사이즈를 50% 축소하고, 3회 연속 수익 거래 후 점진적으로 복귀.
3. **섹터 집중도 대시보드**: 포트폴리오의 섹터별 노출도를 실시간 시각화. IT 섹터 과집중 경고 기능.
4. **감정 로깅 시스템**: 매매 시 감정 상태(자신감 수준 1-10)를 기록하는 선택적 입력 필드. 감정과 매매 성과의 상관관계를 장기적으로 분석.
5. **IT 섹터 모멘텀 지표**: IT 섹터 전체의 자금 유입/유출, RS, 신고가 종목 비율 등을 종합하여 "IT 섹터 건강도" 지표를 생성. Walton 스타일의 섹터 로테이션 판단에 활용.
