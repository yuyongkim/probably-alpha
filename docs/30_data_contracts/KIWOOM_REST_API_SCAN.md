## 핵심 결론: ka10001로 기본 재무지표 제공, 상세 재무제표는 없음

키움 REST API는 `ka10001 주식기본정보요청`을 통해 주요 재무지표를 JSON으로 제공한다.
다만 **상세 재무상태표(BS 항목별), 분기별 시계열, 현금흐름표** 등은 제공하지 않으므로
네이버 금융 API로 보완한다.

### ka10001 한계 (네이버로 보완하는 이유)

- **분기별 시계열 없음** — 최신 연간 값 1개만 (YoY 성장률 계산 불가)
- **BS 상세 항목 없음** — 자산총계, 부채총계, 자본총계, 유동자산/비유동자산 등
- **현금흐름표 없음** — 영업/투자/재무 CF
- **PER/ROE 업데이트 느림** — 외부벤더 데이터, 주 1회
- **빈값 빈도 높음** — 응답 예시에서 sale_amt=0, bus_pro=0, cup_nga=0

### 보완 전략: 네이버 금융 API

현재 프로젝트에서는 **네이버 금융 모바일 API**로 보완 중:

| 소스 | 엔드포인트 | 제공 데이터 |
|------|-----------|------------|
| 네이버 integration | `m.stock.naver.com/api/stock/{code}/integration` | 시총, PER, EPS, PBR, BPS, ROE, 외인비율, 52주 고저, 컨센서스 |
| 네이버 annual | `m.stock.naver.com/api/stock/{code}/finance/annual` | 연간 매출/영업이익/순이익/자산/부채/자본/ROE/ROA/OPM/NPM/EPS/BPS/PER/PBR/배당 (최대 5년) |
| 네이버 quarter | `m.stock.naver.com/api/stock/{code}/finance/quarter` | 분기별 동일 항목 (최대 8분기) |
| 네이버 summary | annual 응답 내 `corporationSummary` | 기업 개요 설명문 |
